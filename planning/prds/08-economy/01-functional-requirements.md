# 8.1 · 知语币 · 功能需求

## 用户故事
- US-EC-01：每日签到拿币
- US-EC-02：用币买 streak freeze 等道具
- US-EC-03：用币换皮肤 / 头像框
- US-EC-04：查看币明细（issuance / sink）

## 功能需求

### EC-FR-001：注册赠币
- 验证邮箱后 +100 ZC（一次）

### EC-FR-002：每日签到
- `/me/checkin` 或 dashboard 浮层
- 5-30 随机（按概率分布：30% 5-9, 40% 10-19, 25% 20-29, 5% 30）
- 连续 7 天 +50 额外，连续 30 +200
- 中断 streak 不影响签到（独立计数）

### EC-FR-003：学习任务奖励
| 任务 | 奖励 |
|---|---|
| 完节小测（≥60%） | 5-15 ZC（按得分） |
| 完章测（≥70%） | 30 ZC |
| 完阶段考（≥75%） | 100 ZC |
| 通关游戏 | 5 ZC + 三星 +10（日上限 50/游戏） |
| 阅读完一篇 DC | 3 ZC |
| 阅读完一章小说 | 3 ZC |
| 章末快测通过 | 5 ZC |
| 日累计学习 30min | 10 ZC（一次） |
| 报错被采纳 | 5 ZC |
| 推荐有效注册 | 10 ZC |
| 推荐人首次付费 | 30 ZC |

### EC-FR-004：消耗 sinks（v1）
- streak freeze：50 ZC / 张（最多持有 3）
- v1 默认仅 freeze；皮肤 / 头像框 v1.5 上线

### EC-FR-005：消耗 sinks（v1.5）
- 头像框：200-500 ZC
- 主题皮肤（学习页 / 仪表板）：500-2000 ZC
- 课程跳关券：1000 ZC（跳过节，不计完成）
- 限定道具（节日）：商店按期上架

### EC-FR-006：年度发行上限
- 单用户年累计 issuance ≤ 50,000
- 触达上限暂停 issuance，下年 1/1 重置

### EC-FR-007：账本明细
- `/me/coins` 显示余额 + 流水（issuance / sink）
- 每条：类型 / 金额 / 时间 / 来源
- 分页 50/页

### EC-FR-008：商店
- `/store` 显示可购道具
- 类目：消耗品 / 装饰 / 限定
- 购买二次确认

### EC-FR-009：防作弊
- 同 IP / 同设备多账号刷币：限流 + 检测
- 异常 issuance 告警（运营干预）
- 用户被冻结：余额冻结 + 所有发放暂停

### EC-FR-010：会员等价（不可购买）
- 系统赠送会员（推广活动）：可用 ZC 兑换月会员（400 ZC = 1 月）
- v1 仅作展示，实际兑换 v1.5 评估
