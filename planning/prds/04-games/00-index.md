## MVP 优先开发原则（强制）

每款游戏分 **MVP / V1 / V1.5** 三阶段，禁止瀑布式一次性写完所有功能。

### Stage MVP · 最小可玩闭环
- 1 难度 / 1 词包（HSK1）/ 5 关
- 基础得分 + 简单结算页
- **不含**：三星 / 排行榜 / SRS / IAP / 教学 / 分享
- 横屏强制 + 60fps 底线

### Stage V1 · 完整学习体验
- 完整关卡（15-30）+ 三星 + 排行榜（日/周/全时）+ SRS + 教学 + 4 语 + HMAC

### Stage V1.5 · 增强变现
- IAP（道具 / 复活 / 解锁）+ 高级关卡 ZC 解锁 + 社交分享 + 进阶玩法 + 自定义词包

## 关键技术决策
- 强制横屏（force-landscape，桌面也横屏画布）
- PixiJS v8 + Howler.js + Matter.js（物理）
- 词汇库取自课程知识点（已学单元优先）
- 知语币奖励：通关 +5~20，三星 +10
- 成绩排行榜（V1 起）：日 / 周 / 全时（每游戏独立）
- 错题入 SRS（V1 起）：游戏中答错知识点
- 不限免费玩，但每日通关币奖励上限 50 / 游戏

## 与其他模块关系

- 提供：错题数据 → LE / 排行榜 → 用户激活
