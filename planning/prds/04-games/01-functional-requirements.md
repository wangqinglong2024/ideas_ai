# 4.1 · 游戏 · 功能需求

## 一、用户故事
- US-GM-01：作为学习者，我希望边玩边学，把课程词汇变成游戏关卡
- US-GM-02：作为玩家，我希望看到排行榜激励我反复挑战
- US-GM-03：作为学习者，我希望游戏错题进入复习集
- US-GM-04：作为休闲用户，我希望免费玩，不强制付费

## 二、通用功能需求

### GM-FR-001：游戏专区入口
- 路径：`/games`
- 12 游戏卡片（v1 5 款 + 7 占位"敬请期待"）
- 每卡片显示：图标、名称、玩家数、平均评分、最高分（个人）
- 排序：推荐 / 新上线 / 热门 / 个人最高

### GM-FR-002：强制横屏
- 任何游戏页：竖屏 → 显示"请旋转设备"遮罩
- 桌面：画布固定 16:9，居中
- 退出游戏：返回 `/games` 首页解除横屏

### GM-FR-003：词汇关卡映射
- 每游戏支持"按词包"开关：
  - 默认：用户当前学习中的轨道知识点
  - 全部已学（默认开）
  - 错题集
  - HSK 1-9 自选
- 词包不足 50 词时回退 HSK 1+

### GM-FR-004：通关与奖励
- 关卡 = 1 局游戏（30s-3min）
- 通关币：基础 5 + 三星 +10 + 首次通关 +5
- 每日上限：50 / 游戏（防刷）
- 通关后展示：用时 / 得分 / 错题列表 / 三星评级 / "再玩"按钮

### GM-FR-005：错题入 SRS
- 游戏中答错的知识点记录 question_id
- 通关后批量推 SRS（LE 模块）
- 错题列表展示在通关页

### GM-FR-006：排行榜
- 三档：日 / 周 / 全时
- 单游戏独立
- 显示：rank / 头像 / 用户名 / 分数 / 用时
- 自己高亮（不在前 50 时显示自己 + 上下 2 名）
- 仅展示已注册用户

### GM-FR-007：游戏内购（v1.5）
- 道具：复活、提示、跳关
- 用知语币购买
- v1 不上

### GM-FR-008：分享
- 通关后"分享成绩"卡片（含分销码）

## 三、5 款 v1 游戏详细规格

### 游戏 1：拼音射击
- 玩法：屏幕上方汉字下落，玩家拼出拼音击落
- 输入：键盘（桌面）/ 屏幕拼音键盘（移动）
- 关卡：30s/局，至少击落 10 字
- 三星：100% / 80% / 60% 准确率
- 错题：未及时击落或拼错的字 → SRS

### 游戏 2：声调泡泡
- 玩法：泡泡含汉字，按声调（4 色）点击对应桶
- 30s/局
- 三星：100% / 80% / 60%
- 错题：错放的字 → SRS

### 游戏 3：打地鼠（汉字版）
- 玩法：5×3 地鼠洞，地鼠头戴汉字，按母语提示打对应字
- 30s/局
- 三星：≥ 20 / 15 / 10 只
- 错题：打错的字 → SRS

### 游戏 4：汉字消消乐
- 玩法：6×6 网格，三同字消除（音 / 形 / 义 三种模式）
- 60s/局
- 三星：分数阈值
- 错题：模式为"义"时配错 → SRS

### 游戏 5：翻牌记忆
- 玩法：4×4 卡牌（汉字 + 拼音 / 母语），翻牌配对
- 用时计 + 错次计
- 三星：用时阈值
- 错题：连续错 3 次的对 → SRS

## 四、UX
- 进入游戏 → 选词包 → 选难度 → 开始
- 通关页：得分 + 三星 + 错题 + 排行榜跳转 + 再玩 + 分享

## 五、性能
- 60fps 中端 Android
- 包大小 < 5MB / 游戏（首屏）
- 资源懒加载

## 六、安全
- 防作弊：成绩客户端签名（HMAC + ts + nonce），服务端校验
- 异常分数（超出统计上限）丢弃
