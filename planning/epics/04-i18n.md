# Epic E04 · 国际化与本地化（i18n & L10n）

> 阶段：M1 · 优先级：P0 · 估算：2 周
>
> 顶层约束：[planning/00-rules.md](../00-rules.md)

## 摘要
4 UI 语言（en / vi / th / id）+ 中文学习内容；多语言内容存储 Supabase；字体与数字/日期/货币本地化。

## 范围
- i18next + react-i18next（FE）
- 4 语言资源（namespace 拆分、懒加载、ICU MessageFormat）
- 内容多语言表 + API
- Intl 数字 / 日期 / 货币
- 字体按语言加载（与 E02 字体子集联动）
- 后台翻译管理（在 E17 中实现 UI；本 epic 仅落 API）

## 非范围
- 中文 UI（暂缓）
- RTL（v2+，仅做 logical properties 准备）
- 翻译外包平台对接（未来）
