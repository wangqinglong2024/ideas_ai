# Epic E10 · 游戏专区（12 款 · 一次性首发 · MVP 单阶段）

> 阶段：M3 · 优先级：P0
> 估算：每款游戏 3-5 天 MVP；通用基础设施 1 周；总周期约 4 周。
> 依据：`planning/prds/04-games/00-index.md`

## 摘要
12 款 H5 横屏游戏 **一次性首发上线**，全部按 **MVP 单阶段** 交付，**不再切 V1 / V1.5**，**不再分波次**，**不再使用 `coming_soon` 占位**。
玩游戏 **不发放任何知语币 / 道具 / 奖励**。所有"完整版 / 增强版 / 排行榜 / 三星 / IAP / 分享 / 教学"需求一律延后到 `99-post-mvp-backlog.md`。

## MVP 定义（强制约束所有 12 款游戏）

每款游戏 MVP 必须包含且仅包含：

- **玩法**：1 种难度 / 1 套词包（默认 HSK1）/ 60s 单局或 5 关
- **结算**：得分 + 用时 + "再玩"按钮 + 错题列表
- **错题**：通关后错题批量推 SRS（GM-FR-005 最小子集）
- **横屏强制**：竖屏遮罩 + 桌面 16:9 画布（GM-FR-002）
- **性能底线**：60fps 中端 Android
- **i18n**：4 语 UI 字符串（占位字符串可后置，但 key 必须就位）

**MVP 不含（明确排除 → 全部进 `99-post-mvp-backlog.md`）**：
- 任何形式的奖励 / 知语币 / 道具 / 经验
- 三星评级 / 关卡星图
- 排行榜（日 / 周 / 全时）
- 教学引导（30s tutorial）
- IAP / 复活 / 解锁
- 自定义词包 / 多模式
- 社交分享卡片
- HMAC 防作弊（仅记录 raw 分数，作弊治理后置）
- `coming_soon` 占位卡片
