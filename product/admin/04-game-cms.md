# 管理后台 — 游戏配置管理

> **模块定位**: 游戏内容配置 — 关卡词汇/题目、Boss 脚本、过场动画、迷你游戏参数
> **版本**: v3.0 | **日期**: 2026-04-16
> **关联**: [管理后台总览](00-index.md) | [世界地图](../04-world-map.md) | [关卡系统](../05-level-system.md) | [迷你游戏](../06-mini-games.md) | [过场动画](../07-cutscenes.md)

---

## 一、模块概述

### 1.1 目标

游戏配置管理用于动态配置关卡词汇/练习、Boss 战对话/题目、过场动画剧本、迷你游戏参数。游戏的视觉资源（地图、角色精灵图、纹理、音效）为代码内置，不在管理后台配置。

### 1.2 可配置 vs 代码内置

| 类型 | 内容 | 管理方式 |
|------|------|---------|
| ✅ **管理后台配置** | 关卡词汇表、关卡练习题、Boss 对话和题目、过场动画脚本和音频、迷你游戏参数 | 动态 API 加载 |
| ❌ **代码内置** | 世界地图图片、角色精灵图、游戏纹理、UI 图标、音效/BGM | 构建时打包 |

> 代码内置资源详见 [静态资源文档](../static-assets/static-assets.md)

---

## 二、信息架构

```
/admin/games
  ├── /levels                → 关卡内容管理（列表）
  ├── /levels/:id/edit       → 编辑关卡词汇/题目
  ├── /bosses                → Boss 战管理（列表）
  ├── /bosses/:id/edit       → 编辑 Boss 配置
  ├── /cutscenes             → 过场动画管理（列表）
  ├── /cutscenes/:id/edit    → 编辑动画脚本
  └── /minigames             → 迷你游戏参数
```

## 五、用户故事

| # | 角色 | 故事 | 验收标准 |
|---|------|------|---------|
| 1 | 编辑 | 为第 4 关配置词汇和练习题 | 从词汇库选择词汇 + 配置 10 道题 |
| 2 | 编辑 | 编辑 Boss 战对话脚本 | 逐条添加对话 + 穿插选择题 |
| 3 | 编辑 | 编写过场动画剧本 | 选角色/表情 + 三语文本 + 音频 |
| 4 | 编辑 | 调整声调狙击手难度曲线 | 修改参数后可在应用端验证 |
| 5 | 编辑 | 用 JSON 批量导入过场动画 | JSON 格式校验 + 导入成功 |
| 6 | 编辑 | 从课程导入练习题到关卡 | 可选择课程中已有的题目复用 |
