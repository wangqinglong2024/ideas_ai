# 用户列表页

> 关联文档：[用户详情页](02-user-detail.md) | [用户封禁/解封](03-user-ban.md) | [管理后台概述 §四](../00-admin-overview.md)
> 路由路径：`/admin/users`
> 角色权限：超级管理员、用户运营
> 情绪目标：**高效、精准、掌控** — 运营人员能在 3 秒内定位任意用户

---

## 一、搜索栏

### 1.1 布局

| 属性 | 说明 |
|------|------|
| **位置** | 页面内容区最顶部，与标题"用户管理"同行靠右 |
| **样式** | 毛玻璃输入框（`.glass-input`），药丸形圆角 `rounded-full`，宽度 360px |
| **图标** | 输入框左侧 Lucide `Search`，16px，灰色 `#a3a3a3` |
| **Placeholder** | "搜索用户昵称 / 邮箱 / 用户ID" |
| **清除按钮** | 输入内容后右侧出现 Lucide `X`，16px，点击清空输入并重置搜索结果 |

### 1.2 搜索逻辑

| 规则 | 说明 |
|------|------|
| **触发方式** | 输入后 500ms 防抖自动搜索（无需按回车） |
| **搜索字段** | 同时匹配：用户昵称（模糊匹配）、注册邮箱（精确前缀匹配）、用户 ID（精确匹配） |
| **最少字符** | 输入 ≥ 2 个字符后触发搜索，<2 个字符显示全量列表 |
| **结果高亮** | 搜索命中字段中的匹配文字高亮显示（Amber 色 `#fde68a` 背景） |
| **无结果** | 列表区域显示空状态："未找到匹配用户"+ Lucide `UserX` 48px 灰色图标 |

## 二、筛选条件

### 2.1 筛选栏布局

| 属性 | 说明 |
|------|------|
| **位置** | 搜索栏下方，间距 16px |
| **布局** | 一行排列全部筛选项，超出自动换行，各筛选项间距 12px |
| **样式** | 每个筛选项为毛玻璃下拉（`.glass-input`），圆角 `rounded-md`（12px） |
| **重置** | 筛选栏右末尾有"重置筛选"文字按钮，Sky 色 `#0284c7`，点击清除所有筛选条件并刷新列表 |

### 2.2 筛选项定义

#### 筛选项 1：用户类型

| 属性 | 说明 |
|------|------|
| **组件** | 下拉单选 |
| **标签** | "用户类型" |
| **选项** | 全部（默认）/ 游客 / 免费用户 / 付费用户 |
| **逻辑** | "游客" = 注册但从未登录课程或游戏的用户；"免费用户" = 活跃但无付费记录；"付费用户" = 有至少一笔已完成的课程购买订单 |

#### 筛选项 2：注册日期范围

| 属性 | 说明 |
|------|------|
| **组件** | 日期范围选择器（起始日期 — 结束日期） |
| **标签** | "注册日期" |
| **默认值** | 空（不限制） |
| **格式** | YYYY-MM-DD |
| **校验** | 起始日期 ≤ 结束日期，否则起始日期输入框红色边框 + 行内提示"起始日期不能晚于结束日期" |

#### 筛选项 3：最近活跃日期范围

| 属性 | 说明 |
|------|------|
| **组件** | 日期范围选择器 |
| **标签** | "最近活跃" |
| **默认值** | 空（不限制） |
| **逻辑** | 筛选用户的"最后活跃时间"落在此范围内 |

#### 筛选项 4：国家/地区

| 属性 | 说明 |
|------|------|
| **组件** | 下拉单选（带搜索） |
| **标签** | "国家/地区" |
| **选项** | 全部（默认）/ 越南 / 印度尼西亚 / 中国 / 美国 / 其他（按注册用户数量排序） |
| **数据来源** | 用户注册时的 IP 归属国家 |

#### 筛选项 5：段位范围

| 属性 | 说明 |
|------|------|
| **组件** | 下拉单选 |
| **标签** | "段位" |
| **选项** | 全部（默认）/ 无段位 / 青铜 / 白银 / 黄金 / 铂金 / 钻石 / 星耀 / 王者 |
| **逻辑** | 选择某段位后，显示当前段位为该大段的所有用户 |

#### 筛选项 6：账号状态

| 属性 | 说明 |
|------|------|
| **组件** | 下拉单选 |
| **标签** | "账号状态" |
| **选项** | 全部（默认）/ 正常 / 封禁中 |

## 三、列表表格列定义

### 3.1 表格布局

| 属性 | 说明 |
|------|------|
| **样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`（24px），内边距 0（表格撑满卡片） |
| **表头** | 背景色 `rgba(255,255,255,0.03)`，文字 Caption 级别（12px，字重 600），灰色 `#a3a3a3`，大写字母间距 `0.08em` |
| **行高** | 每行高度 56px，内容垂直居中 |
| **行 Hover** | 背景色变为 `rgba(255,255,255,0.04)`，过渡 150ms |
| **斑马纹** | 偶数行背景 `rgba(255,255,255,0.02)` |
| **分页** | 表格底部，每页 20 条，显示"共 X 条记录 · 第 X/Y 页"，支持上一页/下一页/跳转指定页 |

### 3.2 列定义

| # | 列名 | 宽度 | 内容说明 | 排序 |
|---|------|------|---------|------|

| 2 | 用户信息 | 240px | 左侧 32px 圆形头像 + 右侧昵称（Body 级别，白色）；头像缺失时显示 Lucide `User` 占位图标（灰色背景圆形） | 否 |

| 4 | 注册日期 | 120px | 格式 YYYY-MM-DD，Body S 级别 | ✅ 默认降序 |
| 5 | 最近活跃 | 120px | 格式 YYYY-MM-DD；超过 30 天未活跃文字变红色 `#ef4444` | ✅ |
| 6 | 用户类型 | 100px | 标签样式：游客 = 灰色标签；免费用户 = Sky 色标签 `#0284c7`；付费用户 = Amber 色标签 `#d97706` | 否 |
| 7 | 段位 | 100px | 段位图标（16px）+ 段位名称；无段位显示"—" | 否 |
| 8 | 国家/地区 | 100px | 国旗 emoji + 国家名称缩写（如 🇻🇳 VN） | 否 |
| 9 | 累计消费 | 100px | 格式 "$X.XX"，等宽字体 `tabular-nums`，右对齐；$0 显示 "—" | ✅ |

| 11 | 操作 | 160px | 操作按钮组，详见下方 §四 | 否 |

## 四、操作列

### 4.1 操作按钮定义

| 按钮 | 样式 | 条件 | 点击行为 |
|------|------|------|---------|
| **查看详情** | 文字按钮，Sky 色 `#0284c7`，Body S 级别 | 始终显示 | 跳转至 `/admin/users/{userId}` 用户详情页 |
| **封禁** | 文字按钮，错误色 `#ef4444`，Body S 级别 | 仅"正常"状态用户显示 | 打开封禁弹窗（详见 [03-user-ban.md](03-user-ban.md)） |

### 4.2 按钮间距

操作按钮之间间距 16px，水平排列。

## 五、排序支持

### 5.1 排序规则

| 属性 | 说明 |
|------|------|
| **可排序列** | 注册日期、最近活跃、累计消费 |
| **排序图标** | 列头右侧 Lucide `ArrowUpDown`（未排序）/ `ArrowUp`（升序）/ `ArrowDown`（降序），16px |
| **默认排序** | 注册日期降序（最新注册排在前面） |
| **操作方式** | 点击列头切换：未排序 → 降序 → 升序 → 未排序（三态循环） |
| **互斥** | 同一时间只能按一个列排序，点击新列时取消旧列的排序状态 |

## 六、导出功能

### 6.1 导出按钮

| 属性 | 说明 |
|------|------|
| **位置** | 页面标题"用户管理"右侧，搜索栏左侧，间距 12px |
| **样式** | 次要按钮（`.glass-button-secondary`），圆角 `rounded-full`，文字"导出 CSV"，左侧 Lucide `Download` 16px |
| **Hover** | 按钮阴影提升至 Level 2 |

### 6.2 导出逻辑

| 规则 | 说明 |
|------|------|
| **导出范围** | 当前筛选条件下的全部用户（非仅当前页） |
| **最大行数** | 单次导出最多 50,000 条；超出提示"数据量过大，请缩小筛选范围" |
| **文件格式** | UTF-8 with BOM 的 CSV 文件（兼容 Excel 中文显示） |
| **文件名** | `zhiyu_users_YYYYMMDD_HHmmss.csv` |
| **异步处理** | 点击"导出 CSV" → 按钮变为 Loading 态 + 文字变为"生成中…" → 生成完成后自动下载 + Toast 成功提示"导出完成" |
| **导出超时** | 超过 30 秒未完成，Toast 错误提示"导出超时，请缩小数据范围后重试" |

### 6.3 导出字段（GDPR 合规）

| 导出字段 | 是否包含 | 说明 |
|---------|---------|------|
| 用户ID | ✅ | — |
| 昵称 | ✅ | — |
| 邮箱 | ✅ | — |
| 注册日期 | ✅ | — |
| 最近活跃日期 | ✅ | — |
| 用户类型 | ✅ | — |
| 段位 | ✅ | — |
| 国家/地区 | ✅ | 仅国家级别 |
| 累计消费 | ✅ | USD |
| 账号状态 | ✅ | — |
| 密码 | ❌ | **GDPR：禁止导出** |
| 精确位置 | ❌ | **GDPR：禁止导出** |
| IP 地址 | ❌ | **GDPR：禁止导出** |
| 设备标识符 | ❌ | **GDPR：禁止导出** |

## 七、状态矩阵

| 状态 | 设计要求 |
|------|---------|
| **空状态 (Empty)** | 筛选无结果：表格区域居中显示 Lucide `UserX` 48px 灰色 + 文案"未找到匹配用户" + "重置筛选"CTA 按钮（Sky 色） |
| **加载中 (Loading)** | 表格区域显示 Skeleton 骨架屏：10 行 × 11 列矩形骨架闪烁，行高与实际一致（56px） |
| **首次加载 (First Load)** | 页面整体骨架屏：搜索栏骨架 + 筛选栏骨架 + 表格骨架 |
| **成功 (Success)** | 正常展示列表数据；表格下方显示"共 X 条记录" |
| **错误 (Error)** | 表格区域居中显示 Lucide `AlertCircle` 48px 红色 + 文案"数据加载失败" + "重试"按钮（Rose 色） |
| **部分加载 (Partial)** | 翻页时：当前页数据保留但降低透明度 40%，新页数据加载中显示 3 行骨架屏叠加在底部 |
| **离线 (Offline)** | 顶部 Banner 黄色提示"网络连接已断开，显示的数据可能不是最新的"；数据不自动刷新 |
