# 用户详情页

> 关联文档：[用户列表页](01-user-list.md) | [用户封禁/解封](03-user-ban.md) | [知语币管理](06-coin-management.md) | [订单详情](04-order-management.md)
> 路由路径：`/admin/users/{userId}`
> 角色权限：超级管理员、用户运营
> 情绪目标：**全面、清晰、高效** — 运营人员无需离开此页面即可掌握用户全貌并执行关键操作

---

## 一、页面整体布局

| 属性 | 说明 |
|------|------|
| **顶部区域** | 基本信息区（固定显示，不随 Tab 切换变化） |
| **Tab 区域** | 6 个 Tab：学习数据、游戏数据、消费数据、知语币、推荐关系、封禁历史 |
| **操作区** | 固定在基本信息区右侧的操作按钮组 |
| **面包屑** | 用户管理 > 用户列表 > {用户昵称} |
| **返回按钮** | 顶部面包屑左侧 Lucide `ArrowLeft` 24px，点击返回用户列表页（保留筛选条件） |

---

## 二、基本信息区

### 2.1 布局

| 属性 | 说明 |
|------|------|
| **位置** | 页面内容区最顶部 |
| **样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`（24px），内边距 24px |
| **结构** | 左侧头像 + 中部信息 + 右侧操作按钮组 |

### 2.2 字段定义

| # | 字段 | 展示方式 | 说明 |
|---|------|---------|------|
| 1 | **头像** | 64px 圆形图片，居左 | 无头像时显示 Lucide `User` 48px 灰色圆形占位 |
| 2 | **昵称** | H2 级别（24px，字重 700），头像右侧，间距 16px | 昵称后显示账号状态标签 |
| 3 | **账号状态标签** | 标签样式，昵称右侧，间距 8px | "正常" = 绿色标签（成功色）；"封禁中" = 红色标签（错误色），标签文字"封禁中 · 至 YYYY-MM-DD"或"封禁中 · 永久" |
| 4 | **用户ID** | Body S 级别（14px），灰色 `#a3a3a3`，昵称下方 | 格式："ID: XXXXXXXX"，点击可复制到剪贴板，复制成功 Toast 提示"已复制" |
| 5 | **邮箱** | Body S 级别，灰色，用户ID 右侧，间距 24px | Lucide `Mail` 14px 图标 + 邮箱地址 |
| 6 | **注册日期** | Body S 级别，灰色 | Lucide `Calendar` 14px 图标 + "注册于 YYYY-MM-DD"；如注册方式为 Google 显示 Google 图标，Apple 显示 Apple 图标 |
| 7 | **最近活跃** | Body S 级别，灰色 | Lucide `Clock` 14px 图标 + "最近活跃 YYYY-MM-DD HH:mm"；超过 30 天变红色 |
| 8 | **国家/地区** | Body S 级别，灰色 | 国旗 emoji + 国家名称 |
| 9 | **语言偏好** | Body S 级别，灰色 | 当前 UI 语言 + 学习语言模式（"拼音+中文"或"纯中文"） |
| 10 | **设备信息** | Body S 级别，灰色 | 最近登录设备类型（Web / Android / iOS）+ 浏览器/系统版本 |

### 2.3 操作按钮组

| 按钮 | 样式 | 条件 | 点击行为 |
|------|------|------|---------|
| **封禁用户** | 危险按钮（红色背景 `#ef4444` + 白色文字），圆角 `rounded-full`，Lucide `Ban` 16px | 仅"正常"状态用户显示 | 打开封禁弹窗（详见 [03-user-ban.md](03-user-ban.md)） |
| **解封用户** | 成功按钮（绿色背景 `#22c55e` + 白色文字），圆角 `rounded-full`，Lucide `ShieldCheck` 16px | 仅"封禁中"状态用户显示 | 打开解封确认弹窗（详见 [03-user-ban.md](03-user-ban.md)） |
| **调整知语币** | 次要按钮（`.glass-button-secondary`），圆角 `rounded-full`，Lucide `Coins` 16px | 始终显示 | 打开知语币调整弹窗（详见 [06-coin-management.md](06-coin-management.md)） |
| **重置密码** | 次要按钮（`.glass-button-secondary`），圆角 `rounded-full`，Lucide `KeyRound` 16px | 始终显示（P2） | 二次确认弹窗 → 确认后生成临时密码 → 发送至用户邮箱 → Toast"临时密码已发送至 {邮箱}" |

按钮之间间距 12px，水平排列，靠右对齐。

|------|---------|
| **空状态 (Empty)** | 各 Tab 内容为空时，独立展示该 Tab 的空状态插画和文案（详见各 Tab 定义） |
| **加载中 (Loading)** | 基本信息区：头像圆形骨架 + 3 行文字骨架；Tab 内容区：表格骨架屏 |
| **首次加载 (First Load)** | 整页骨架屏：基本信息区骨架 + Tab 栏骨架 + 默认 Tab 内容骨架 |
| **成功 (Success)** | 正常展示所有数据 |
| **错误 (Error)** | Tab 内容区居中显示"数据加载失败" + "重试"按钮；基本信息区如加载失败则显示"用户不存在或加载失败" + "返回列表"按钮 |
| **部分加载 (Partial)** | 基本信息区先加载，Tab 内容区独立加载骨架屏；各 Tab 之间独立，切换时仅加载当前 Tab |
| **离线 (Offline)** | 顶部 Banner 黄色提示"网络连接已断开" |
