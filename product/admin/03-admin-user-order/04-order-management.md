# 订单列表与订单详情

> 关联文档：[退款管理](05-refund.md) | [用户详情页](02-user-detail.md) | [商业模式](../../../plan/06-business-model.md) | [管理后台概述 §五](../00-admin-overview.md)
> 路由路径：订单列表 `/admin/orders` · 订单详情 `/admin/orders/{orderId}`
> 角色权限：超级管理员、用户运营
> 情绪目标：**掌控、透明、可追溯** — 每笔交易来龙去脉一目了然

---

## 一、订单列表页

### 1.1 搜索栏

| 属性 | 说明 |
|------|------|
| **位置** | 页面内容区最顶部，与标题"订单管理"同行靠右 |
| **样式** | 毛玻璃输入框（`.glass-input`），药丸形圆角 `rounded-full`，宽度 360px |
| **图标** | 输入框左侧 Lucide `Search`，16px，灰色 `#a3a3a3` |
| **Placeholder** | "搜索订单号 / 用户昵称 / 用户邮箱" |
| **清除按钮** | 输入内容后右侧出现 Lucide `X`，16px |

#### 搜索逻辑

| 规则 | 说明 |
|------|------|
| **触发方式** | 输入后 500ms 防抖自动搜索 |
| **搜索字段** | 同时匹配：订单号（精确前缀匹配）、用户昵称（模糊匹配）、用户邮箱（精确前缀匹配） |
| **最少字符** | ≥ 2 个字符触发搜索 |
| **无结果** | 列表区域空状态："未找到匹配订单" + Lucide `FileX` 48px 灰色 |

### 1.2 筛选条件

#### 筛选栏布局

| 属性 | 说明 |
|------|------|
| **位置** | 搜索栏下方，间距 16px |
| **布局** | 一行排列，超出换行，各筛选项间距 12px |
| **重置** | 右末尾"重置筛选"文字按钮，Sky 色 |

#### 筛选项 1：订单状态

| 属性 | 说明 |
|------|------|
| **组件** | 下拉单选 |
| **标签** | "订单状态" |
| **选项** | 全部（默认）/ 已完成 / 已退款 / 退款处理中 |

#### 筛选项 2：课程 Level

| 属性 | 说明 |
|------|------|
| **组件** | 下拉多选（带全选复选框） |
| **标签** | "课程 Level" |
| **选项** | L4 / L5 / L6 / L7 / L8 / L9 / L10 / L11 / L12 |
| **说明** | L1-L3 免费，无付费订单 |

#### 筛选项 3：日期范围

| 属性 | 说明 |
|------|------|
| **组件** | 日期范围选择器（起始日期 — 结束日期） |
| **标签** | "订单日期" |
| **默认值** | 空（不限制） |
| **快捷选项** | "今天" / "最近 7 天" / "最近 30 天" / "本月" / "自定义" |

#### 筛选项 4：金额范围

| 属性 | 说明 |
|------|------|
| **组件** | 双输入框（最小金额 — 最大金额） |
| **标签** | "金额范围" |
| **格式** | USD，保留两位小数 |
| **校验** | 最小金额 ≤ 最大金额，否则红色边框 + 提示 |

### 1.3 订单统计摘要栏

| 属性 | 说明 |
|------|------|
| **位置** | 筛选栏下方，间距 16px |
| **样式** | 一行 4 个统计卡片，毛玻璃面板（`.glass-card`），圆角 `rounded-xl`（20px），内边距 16px |
| **数据范围** | 当前筛选条件下的统计结果（随筛选条件变化实时更新） |

| 卡片 | 标题 | 主数值 | 格式 |
|------|------|--------|------|
| 总订单数 | "订单总数" | 整数 | 千分位分隔 |
| 总金额 | "订单总额" | USD | "$X,XXX.XX"，Amber 色 |
| 退款数 | "退款笔数" | 整数 | 红色 |
| 退款金额 | "退款金额" | USD | "$X,XXX.XX"，红色 |

### 1.4 列表表格列定义

#### 表格布局

| 属性 | 说明 |
|------|------|
| **样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`，内边距 0 |
| **表头** | 灰色背景，Caption 级别（12px，字重 600），大写字间距 |
| **行高** | 56px，垂直居中 |
| **行 Hover** | 背景色 `rgba(255,255,255,0.04)` |
| **分页** | 每页 20 条，"共 X 条 · 第 X/Y 页"，上一页/下一页/跳转 |

#### 列定义

| # | 列名 | 宽度 | 内容说明 | 排序 |
|---|------|------|---------|------|
| 1 | 订单号 | 160px | Paddle 订单号或系统订单号，等宽字体 `tabular-nums`，Body S 级别 | 否 |
| 2 | 用户 | 180px | 头像（24px）+ 昵称（Sky 色可点击，跳转用户详情页） | 否 |
| 3 | 课程名称 | 160px | "Level {X}: {课程名}" | 否 |
| 4 | 金额 | 100px | "$X.XX"，等宽字体，右对齐 | ✅ |
| 5 | 支付方式 | 100px | "Paddle" 标签（Sky 色）/ "Apple IAP" 标签（灰色）/ "知语币" 标签（Amber 色） | 否 |
| 6 | Paddle 交易号 | 160px | Paddle transaction ID；非 Paddle 支付显示"—"；等宽字体 | 否 |
| 7 | 订单日期 | 120px | 格式 YYYY-MM-DD | ✅ 默认降序 |
| 8 | 状态 | 100px | "已完成"绿色标签 / "已退款"红色标签 / "退款处理中" Amber 色标签 | 否 |
| 9 | 操作 | 160px | 操作按钮组 | 否 |

### 1.5 操作列

| 按钮 | 样式 | 条件 | 点击行为 |
|------|------|------|---------|
| **查看详情** | 文字按钮，Sky 色，Body S | 始终显示 | 跳转至 `/admin/orders/{orderId}` |
| **发起退款** | 文字按钮，错误色 `#ef4444`，Body S | **仅"已完成"状态订单显示**；"已退款"和"退款处理中"状态不显示此按钮 | 打开退款操作弹窗（详见 [05-refund.md](05-refund.md)） |

### 1.6 排序支持

| 属性 | 说明 |
|------|------|
| **可排序列** | 金额、订单日期 |
| **默认排序** | 订单日期降序 |
| **操作方式** | 点击列头切换：未排序 → 降序 → 升序 → 未排序 |

### 1.7 状态矩阵

| 状态 | 设计要求 |
|------|---------|
| **空状态 (Empty)** | 表格区域居中 Lucide `FileX` 48px 灰色 + "未找到匹配订单" + "重置筛选"CTA |
| **加载中 (Loading)** | 统计摘要栏 4 个卡片骨架 + 表格 10 行骨架屏 |
| **首次加载 (First Load)** | 搜索栏+筛选栏+摘要栏+表格全骨架屏 |
| **成功 (Success)** | 正常展示列表 + 统计摘要 |
| **错误 (Error)** | 表格区域居中"数据加载失败" + "重试"按钮 |
| **部分加载 (Partial)** | 翻页时当前页降低透明度 + 新页骨架 |
| **离线 (Offline)** | 顶部 Banner 黄色提示 |

## 二、订单详情页

### 2.0 页面布局

| 属性 | 说明 |
|------|------|
| **路由** | `/admin/orders/{orderId}` |
| **面包屑** | 订单管理 > 订单列表 > 订单 {订单号} |
| **返回按钮** | 面包屑左侧 Lucide `ArrowLeft` 24px |
| **布局** | 纵向 6 个信息区块卡片，间距 16px |

### 2.1 订单信息

| 属性 | 说明 |
|------|------|
| **标题** | H3 级别，"订单信息" |
| **样式** | 毛玻璃面板（`.glass-card`），圆角 `rounded-2xl`，内边距 24px |

| 字段 | 展示方式 |
|------|---------|
| **订单号** | Body 级别，白色，右侧"复制"图标按钮（Lucide `Copy`，16px），点击复制 + Toast"已复制" |
| **创建时间** | Body S 级别，灰色，格式 YYYY-MM-DD HH:mm:ss |
| **支付时间** | Body S 级别，灰色，格式 YYYY-MM-DD HH:mm:ss；未支付显示"—" |
| **订单状态** | 状态标签："已完成"绿色 / "已退款"红色 / "退款处理中" Amber 色 |

### 2.2 用户信息

| 属性 | 说明 |
|------|------|
| **标题** | H3 级别，"用户信息" |

| 字段 | 展示方式 |
|------|---------|
| **头像** | 40px 圆形 |
| **昵称** | Body 级别，Sky 色 `#0284c7`，可点击跳转 `/admin/users/{userId}` |
| **邮箱** | Body S 级别，灰色 |
| **用户ID** | Body S 级别，灰色，格式 "ID: {userId}" |

### 2.3 课程信息

| 属性 | 说明 |
|------|------|
| **标题** | H3 级别，"课程信息" |

| 字段 | 展示方式 |
|------|---------|
| **课程等级** | "Level {X}" + 课程名称 |
| **有效期起始** | 格式 YYYY-MM-DD |
| **有效期截止** | 格式 YYYY-MM-DD；已过期标红 |
| **当前学习进度** | 进度条 + 百分比（退款后此行显示灰色 + 备注"课程权限已回收"） |

### 2.4 支付信息

| 属性 | 说明 |
|------|------|
| **标题** | H3 级别，"支付信息" |

| 字段 | 展示方式 |
|------|---------|
| **支付金额** | "$X.XX"，H2 级别（24px），Amber 色 |
| **币种** | "USD" |
| **支付方式** | "Paddle" / "Apple IAP" / "知语币"，标签样式 |
| **Paddle 交易号** | Paddle transaction ID，等宽字体；非 Paddle 支付显示"—" |
| **Paddle 手续费** | "$X.XX"（约 5% + $0.50），灰色小字；仅 Paddle 支付显示 |
| **净收入** | "$X.XX"（支付金额 - Paddle 手续费），灰色小字；仅 Paddle 支付显示 |

### 2.5 推荐返利信息

| 属性 | 说明 |
|------|------|
| **标题** | H3 级别，"推荐返利" |

| 场景 | 展示内容 |
|------|---------|
| **有推荐人** | — |
| 推荐人信息 | 推荐人头像（32px）+ 昵称（Sky 色可点击跳转）+ 用户 ID |
| 返利金额 | "XX 知语币（≈ $X.XX）"，Amber 色 |
| 返利状态 | "已发放"绿色 / "冷却中（剩余 XX 天）" Amber 色 / "已扣回"红色 |
| 被推荐人自得返利 | "购买者自得 XX 知语币"，同样标注返利状态 |
| **无推荐人** | 灰色文字"该订单无关联推荐关系" |

#### 返利规则说明（页面底部灰色小字）

> 推荐返利 = 订单金额 × 20%，折算为知语币（1 币 = $0.01）。推荐人和被推荐人各得 20%。30 天冷却期通过后发放。退款后自动扣回。

### 2.6 退款信息

| 属性 | 说明 |
|------|------|
| **标题** | H3 级别，"退款信息" |

| 场景 | 展示内容 |
|------|---------|
| **未退款** | 灰色文字"该订单未发生退款"；如订单状态为"已完成"，此区域底部显示"发起退款"按钮（危险按钮） |
| **已退款或退款处理中** | — |
| 退款单号 | 退款记录 ID |
| 退款日期 | 格式 YYYY-MM-DD HH:mm:ss |
| 退款金额 | "$X.XX"，红色 |
| 退款原因 | 原因分类 + 补充说明 |
| 退款操作人 | 管理员姓名 |
| 退款状态 | "退款处理中" Amber 色 / "已退款"绿色 / "退款失败"红色 |
| 退款方式 | "原路退回（Paddle 处理）" |
| 课程回收状态 | "已回收"绿色 / "回收中" Amber 色 |
| 返利扣回状态 | "已扣回 XX 知语币"绿色 / "扣回中" Amber 色 / "无需扣回"灰色（无推荐关系时） |

### 2.7 操作区

| 属性 | 说明 |
|------|------|
| **位置** | 页面最底部，固定显示 |

| 按钮 | 样式 | 条件 | 点击行为 |
|------|------|------|---------|
| **发起退款** | 危险按钮（红色背景），圆角 `rounded-full`，Lucide `RotateCcw` 16px | 仅"已完成"状态订单显示 | 打开退款操作弹窗（详见 [05-refund.md](05-refund.md)） |
| **返回列表** | 次要按钮 | 始终显示 | 返回订单列表页 |

### 2.8 状态矩阵

| 状态 | 设计要求 |
|------|---------|
| **空状态 (Empty)** | 不适用（订单详情页必有数据） |
| **加载中 (Loading)** | 6 个信息区块全部显示骨架屏 |
| **首次加载 (First Load)** | 面包屑骨架 + 6 个区块骨架 |
| **成功 (Success)** | 正常展示所有信息区块 |
| **错误 (Error)** | 页面居中"订单不存在或加载失败" + "返回列表"按钮 |
| **部分加载 (Partial)** | 各区块独立加载，先加载完的先展示 |
| **离线 (Offline)** | 顶部 Banner 黄色提示 |
