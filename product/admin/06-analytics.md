# 管理后台 — 数据分析仪表盘

> **模块定位**: 运营驾驶舱 — 核心指标概览、用户/内容/收入/增长/技术分析
> **版本**: v3.0 | **日期**: 2026-04-16
> **关联**: [管理后台总览](00-index.md)

---

## 一、模块概述

### 1.1 目标

为运营团队提供一站式数据分析能力，涵盖全局概览、用户行为、内容效果、收入趋势、增长裂变和技术性能 6 个维度。所有图表支持时间范围切换和数据导出。

### 1.2 六维分析体系

```
                    ┌──────────────────┐
                    │  📊 全局概览       │
                    │  (Dashboard 首页) │
                    └────────┬─────────┘
          ┌──────────────────┼──────────────────┐
          ▼                  ▼                  ▼
   ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
   │ 👥 用户分析    │  │ 📝 内容分析    │  │ 💰 收入分析    │
   └──────────────┘  └──────────────┘  └──────────────┘
          ┌──────────────────┼──────────────────┐
          ▼                  ▼                  ▼
   ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
   │ 📈 增长分析    │  │ 🔧 技术指标    │  │ (预留)       │
   └──────────────┘  └──────────────┘  └──────────────┘
```

---

## 二、信息架构

```
/admin/analytics
  ├── /dashboard       → 全局概览（默认首页）
  ├── /users           → 用户分析
  ├── /content         → 内容分析
  ├── /revenue         → 收入分析
  ├── /growth          → 增长分析
  └── /tech            → 技术指标
```

## 四、通用功能

| 功能 | 说明 |
|------|------|
| 时间范围 | 今天 / 昨天 / 近7天 / 近30天 / 近90天 / 自定义日期 |
| 数据刷新 | 页面级手动刷新 + 概览页 5 分钟自动刷新 |
| 数据导出 | 每个图表右上角 📥 按钮，导出 CSV |
| 图表交互 | Hover 显示具体数值 + 点击图例切换系列 |
| 数据对比 | 支持同比/环比对比（选择对比时段） |

## 五、API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/admin/analytics/overview` | 全局概览指标 |
| GET | `/api/admin/analytics/users` | 用户分析数据 |
| GET | `/api/admin/analytics/content/:module` | 内容分析（discover/course/game） |
| GET | `/api/admin/analytics/revenue` | 收入分析数据 |
| GET | `/api/admin/analytics/growth` | 增长分析数据 |
| GET | `/api/admin/analytics/tech` | 技术指标数据 |

所有端点支持 `?from=&to=&granularity=day|week|month` 参数。
