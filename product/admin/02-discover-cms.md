# 管理后台 — 发现中国内容管理

> **模块定位**: 文化文章全生命周期管理 — 创建、编辑、预览、发布、下架
> **版本**: v3.0 | **日期**: 2026-04-16
> **关联**: [管理后台总览](00-index.md) | [发现中国 PRD](../02-discover-china.md) | [上传规格](09-config-checklist.md)

---

## 一、模块概述

### 1.1 目标

运营人员可在管理后台完成文化文章的全生命周期管理——从创建草稿、编辑富文本内容、配置词汇和测验，到预览应用端效果、发布上线和下架。发布后文章即时在应用端展示，无需更新代码。

### 1.2 核心指标

| 指标 | 目标 |
|------|------|
| 文章创建到发布耗时 | ≤ 30 分钟/篇 |
| 编辑器保存响应 | ≤ 1s |
| 图片上传响应 | ≤ 3s（含自动压缩转码） |
| SEO 页面同步延迟 | ≤ 5 分钟 |

```
/admin/discover
```

## 四、内容块类型

正文编辑器支持以下内容块，通过工具栏插入：

| 内容块 | 说明 | 上传规格 |
|--------|------|---------|
| 越南语段落 | 主体文本，支持加粗/斜体/标题 | — |
| 中文原文块 | 中文文本 + 自动拼音标注，可手动修正拼音 | — |
| 配图 | 文中插图，拖拽排序 | 800×auto WebP ≤150KB |
| 音频片段 | 中文朗读音频，行内播放器 | MP3 16kHz 单声道 ≤2MB |
| 词汇高亮 | 选中文本 → 标记为词汇 → 关联词汇表中的条目 | — |

**拼音自动生成**：输入中文文本后，系统自动调用拼音转换 API 生成拼音标注，管理员可逐字修正（处理多音字）。

## 五、发布流程

### 5.1 状态机

```
               ┌──────── 编辑 ────────┐
               ▼                      │
新建 ──→ 草稿 ──→ 预览 ──→ 发布 ──→ 已上线
                                        │
                                      下架 ──→ 草稿
```

### 5.2 各状态规则

| 状态 | 说明 | 应用端 | SEO 页 |
|------|------|--------|--------|
| 草稿 | 编辑中，可随时保存 | 不可见 | 不可见 |
| 预览 | 生成临时预览链接（24h 有效） | 仅预览链接可见 | 不可见 |
| 已发布 | 正式上线 | 可见 | 同步生成 |
| 已下架 | 从应用端和 SEO 页移除 | 不可见 | 返回 404 |

### 5.3 发布检查清单

发布前系统自动校验：

| 检查项 | 规则 |
|--------|------|
| 标题 | 中文和越南语标题均不为空 |
| 封面图 | 已上传且符合规格 |
| 正文 | ≥ 500 字（越南语部分） |
| 中文原文 | 至少包含 1 个中文原文块 |
| 词汇 | 8-15 个词汇，每个有汉字+拼音+释义+音频 |
| 测验 | 正好 5 题，每题有 4 选项和正确答案 |
| SEO | 描述和关键词已填写 |

未通过的项目显示红色提示，阻止发布操作。

## 六、预览功能

- 点击「预览」打开新浏览器窗口
- 使用应用端的文章详情页布局渲染
- URL 格式：`/preview/article/:token`（临时 token，24h 过期）
- 预览页顶部悬浮黄色条："⚠️ 预览模式 — 仅管理员可见"
- 预览中可直接修改并刷新

## 七、跨模块联动

| 联动方向 | 说明 |
|---------|------|
| 文章 → 词汇库 | 文章中的词汇自动同步至应用端词汇 SRS，来源标记 📚 |
| 文章 → 课程 | 每个词汇可关联一个课程 Lesson（下拉选择） |
| 文章 → 测验 | 测验答对的词汇标记为"已掌握" |
| 文章 → SEO | 发布后自动生成 SSR 公开页，URL: `/discover/:category/:slug` |
