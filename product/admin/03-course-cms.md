# 管理后台 — 系统课程内容管理

> **模块定位**: 课程全生命周期管理 — Level/Unit/Lesson 编辑、练习配置、测验管理
> **版本**: v3.0 | **日期**: 2026-04-16
> **关联**: [管理后台总览](00-index.md) | [系统课程 PRD](../03-course-system.md) | [上传规格](09-config-checklist.md)

---

## 一、模块概述

### 1.1 目标

运营人员可在管理后台完整管理课程体系的四级结构（Level → Unit → Lesson → Step），通过 7 步教学编辑器配置每课内容，并管理练习题、单元测验和结业证书。

### 1.2 课程结构

```
Level（4级）──→ Unit（每级6单元）──→ Lesson（每单元5课）──→ Step（每课7步）
  L0 语音入门      U1 声母韵母        Lesson 01 你好        1.导入
  L1 基础 HSK1     U2 声调            Lesson 02 谢谢        2.生词
  L2 初级 HSK2     ...                ...                   3.课文
  L3 中级 HSK3                                              4.语法
                                                            5.练习
                                                            6.文化角
                                                            7.总结
```

### 1.3 核心指标

| 指标 | 目标 |
|------|------|
| 单课编辑完成时间 | ≤ 45 分钟（含所有 Step） |
| 编辑器保存响应 | ≤ 1s |
| 音频上传/TTS 生成 | ≤ 5s |

---

## 二、信息架构

```
/admin/courses
  ├── /levels                          → Level/Unit 总览
  ├── /levels/:lid/units/:uid/lessons  → 某单元下的 Lesson 列表
  ├── /lessons/new                     → 新建 Lesson
  ├── /lessons/:id/edit                → 编辑 Lesson（7步 Tab）
  ├── /tests                           → 测验管理列表
  └── /tests/:id/edit                  → 编辑测验
```

| 🎤 跟读评测 | 目标音频 + 参考文本 | MP3 16kHz ≤200KB |

每道题通用字段：
- 题目编号（自动递增，可拖拽调整）
- 答对反馈文本（可选，默认"Tuyệt vời!"）
- 答错解析（可选）

## 五、发布流程

### 5.1 Lesson 发布检查

| 检查项 | 规则 |
|--------|------|
| Step 1 导入 | 场景图 + 双语描述 + 音频 |
| Step 2 生词 | 8-15 个，每个有汉字+拼音+释义+发音音频 |
| Step 3 课文 | 中文课文 + 越南语翻译 + 朗读音频 |
| Step 4 语法 | 双语标题 + 讲解内容 |
| Step 5 练习 | 8-12 题，题型分布合理 |
| Step 6 文化角 | 标题 + 内容 |
| Step 7 总结 | 要点列表 + 下节预告 |

全部 7 步通过后可发布，否则显示红色提示。

### 5.2 状态机

```
新建 ──→ 草稿（编辑中）──→ 发布 ──→ 已上线
                                      │
                                    下架 ──→ 草稿
```
