> **模块定位**: 用户账号查看、编辑、封禁、赠送会员、进度管理
> **版本**: v3.0 | **日期**: 2026-04-16
> **关联**: [管理后台总览](00-index.md) | [用户系统](../01-user-system.md) | [成就系统](../11-achievement-honor.md)

## 一、模块概述

### 1.1 目标

管理员可查看所有用户的注册信息、学习进度、会员状态、推广数据和成就记录，并执行封禁、赠送会员、重置进度等管理操作。

### 1.2 核心指标

| 指标 | 目标 |
|------|------|
| 用户列表加载 | ≤ 1s（万级数据） |
| 搜索响应 | ≤ 500ms |
| 用户详情加载 | ≤ 1s |

---

## 二、信息架构

```
/admin/users
  ├── /list          → 用户列表页
  └── /:id           → 用户详情页
```

## 四、用户详情 Tab 完整列表

| Tab | 展示内容 |
|-----|---------|
| 概览 | 模块偏好 + 三模块进度卡片 + 学习统计 + XP/等级/徽章 |
| 发现中国 | 已读文章数/总数 + 各板块进度条 + 收藏列表 + 测验成绩 |
| 课程 | 当前 Level/Unit/Lesson + 完成课时数 + 各级测验成绩 + 掉课记录 |
| 游戏 | 已通过关卡/总关卡 + 星级分布(1/2/3星) + 迷你游戏最高分 + Boss 通过记录 |
| 词汇 | 总词汇量 + 三模块来源分布(📚/📖/🎮) + SRS 等级分布 + 今日待复习 |
| 会员 | 当前套餐 + 到期时间 + 自动续费 + 历史订阅记录 + Paddle 订单 |
| 推广 | 推广码 + 佣金等级 + 累计收益 + 下线用户数 + 最近推荐 |
| 成就 | 等级 + 已获徽章列表 + Streak 天数 + 每日任务完成情况 |
| 日志 | 该用户相关的管理操作记录（封禁/解封/赠送/重置等） |

## 五、API 端点

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/api/admin/users` | 用户列表（分页+筛选+搜索） | 全部 |
| GET | `/api/admin/users/:id` | 用户详情（含全部模块数据） | 全部 |
| PUT | `/api/admin/users/:id` | 编辑基本信息（昵称/邮箱） | 全部 |
| PUT | `/api/admin/users/:id/ban` | 封禁/解封 | 超管 |
| PUT | `/api/admin/users/:id/membership` | 赠送会员 | 超管 |
| PUT | `/api/admin/users/:id/reset` | 重置进度 | 超管 |

## 六、用户故事

| # | 角色 | 故事 | 验收标准 |
|---|------|------|---------|
| 1 | 管理员 | 搜索一个用户查看学习进度 | 可按昵称/邮箱搜索并查看三模块进度 |
| 2 | 超管 | 封禁一个滥用推广的用户 | 填写原因后用户无法登录，记录到操作日志 |
| 3 | 超管 | 为 KOL 赠送 90 天会员 | 选择时长+填写原因后即时生效 |
| 4 | 超管 | 重置一个用户的游戏进度 | 选择模块+输入确认码后清零 |
| 5 | 管理员 | 查看某用户的课程掉课记录 | 课程 Tab 显示放弃率最高的课程 |
