## 二、通用说明

### 2.1 画面方向

- 所有游戏画面强制横屏（landscape），设计基准分辨率 1920×1080
- Phaser ScaleManager 自适应缩放，支持 Web 桌面和移动端 H5
- 进入游戏自动触发横屏切换，退出游戏恢复竖屏（详见 `05-game-common/08-hud-landscape.md`）

### 2.2 通用 HUD 元素

以下元素由通用系统提供，各游戏不再单独定义：
- **暂停按钮**：右上角，距顶部 16px，距右侧 16px
- **声音/音乐控制**：暂停菜单内切换
- **退出确认**：暂停菜单 → 退出 → 二次确认
- **断线重连**：断线检测 + 重连弹窗 + 状态恢复

### 2.3 本轮仅描述内容

以下文件仅描述每款游戏的**独特画面元素和交互**，包括：
- 游戏专属 HUD（分数、回合数、倒计时等）
- 游戏区域布局与视觉元素
- 核心操作的完整交互链路（用户动作 → 视觉反馈 → 音效反馈 → 分数变化）
- 子模式差异规则
- 难度递增机制
- PK/多人模式特殊规则
- 上瘾机制详情

### 2.4 G5-G8 知识深度说明

G5-G8 对应 L5-L8 课程，知识深度显著高于 G1-G4：

| 游戏 | 对应 Level | 知识范围 | 交互特点 |
|------|-----------|---------|---------|
| G5 成语接龙大战 | L5（深度阅读） | 成语 100 个 + 扩展成语库；褒贬义、字义理解 | 文本输入 + 联想提示 + 回合制 |
| G6 汉字华容道 | L6（综合提升） | 成语 150 个 + 缩句扩句 + 古诗句 | 滑块拖拽 + 步数优化 |
| G7 古诗飞花令 | L7（高级阅读） | 累计 76 首古诗词 + 高级修辞 | 文本输入 + 诗句有效性判定 |
| G8 阅读侦探社 | L8（文学赏析） | 2000-3000 字现代文 + 修辞赏析 + 论证手法 | 阅读 + 线索发现 + 推理答题 |
