## 五、正文内容区

### 5.1 多语言渲染规则

正文按段落为单位渲染，每个段落根据用户配置显示不同行数。

**配置组合一：拼音+中文 + 解释语言开启**（最完整展示）

```
Qínshǐhuáng shì Zhōngguó lìshǐ shàng dì yī wèi huángdì.    ← 拼音行
秦始皇是中国历史上第一位皇帝。                                   ← 中文行
Tần Thủy Hoàng là vị hoàng đế đầu tiên trong lịch sử          ← 解释语言行
Trung Quốc.
                                                               ← 段间距 24px
Tā tǒngyī le liù guó, jiànlì le qiáng dà de Qín cháo.       ← 拼音行
他统一了六国，建立了强大的秦朝。                                  ← 中文行
Ông đã thống nhất sáu nước, xây dựng triều đại Tần hùng mạnh. ← 解释语言行
```

**配置组合二：拼音+中文 + 解释语言关闭**

```
Qínshǐhuáng shì Zhōngguó lìshǐ shàng dì yī wèi huángdì.    ← 拼音行
秦始皇是中国历史上第一位皇帝。                                   ← 中文行

Tā tǒngyī le liù guó, jiànlì le qiáng dà de Qín cháo.       ← 拼音行
他统一了六国，建立了强大的秦朝。                                  ← 中文行
```

**配置组合三：纯中文 + 解释语言开启**

```
秦始皇是中国历史上第一位皇帝。                                   ← 中文行
Tần Thủy Hoàng là vị hoàng đế đầu tiên trong lịch sử          ← 解释语言行
Trung Quốc.

他统一了六国，建立了强大的秦朝。                                  ← 中文行
Ông đã thống nhất sáu nước, xây dựng triều đại Tần hùng mạnh. ← 解释语言行
```

**配置组合四：纯中文 + 解释语言关闭**

```
秦始皇是中国历史上第一位皇帝。                                   ← 中文行

他统一了六国，建立了强大的秦朝。                                  ← 中文行
```

**配置组合五：UI 语言=汉语**（解释语言自动关闭）

```
Qínshǐhuáng shì Zhōngguó lìshǐ shàng dì yī wèi huángdì.    ← 拼音行（若选了拼音+中文）
秦始皇是中国历史上第一位皇帝。                                   ← 中文行
```

## 六、音频播放条

当文章配有音频时，显示音频播放条；无音频时完全隐藏此区域。

### 6.1 播放条布局

播放条悬浮固定于页面底部（Tab Bar 上方），毛玻璃材质（`.glass-elevated`）。

```
┌─────────────────────────────────────────┐
│  [▶/⏸]  ━━━━━━━━━━━○━━━━━━  02:15/05:30  [1x ▾]  │
└─────────────────────────────────────────┘
```

| 元素 | 样式 | 行为 |
|------|------|------|
| 播放/暂停按钮 | Lucide `Play` 或 `Pause`（24px），Rose 色，点击热区 ≥ 44×44pt | 点击切换播放/暂停 |
| 进度条 | 高度 4px，背景色中性灰 20%，已播放部分 Rose 色，圆头 | 可拖拽调整播放位置；拖拽时显示时间 Tooltip |
| 当前时间/总时长 | Caption 字体（12px），次要文字色 | 格式 MM:SS / MM:SS |
| 倍速选择 | 当前倍速文字（Body S，14px）+ Lucide `ChevronDown`（16px），点击热区 ≥ 44×44pt | 点击 → 弹出倍速选项 Popover |

### 6.2 倍速选项

| 选项 | 标签 |
|------|------|
| 0.5 倍速 | 0.5x |
| 正常速度 | 1x（默认） |
| 1.5 倍速 | 1.5x |
| 2 倍速 | 2x |

倍速选项以 Popover 形式从倍速按钮上方弹出，当前选中项左侧显示 Lucide `Check` 图标（Rose 色）。

### 6.3 播放状态管理

| 场景 | 行为 |
|------|------|
| 首次进入文章详情 | 播放条显示，状态为暂停（▶ 图标），进度 00:00 |
| 点击播放 | 图标切换为 ⏸，音频开始播放，进度条随时间推进 |
| 点击暂停 | 图标切换为 ▶，音频暂停，进度保持当前位置 |
| 播放完毕 | 图标切换为 ▶，进度回到 00:00，可重新播放 |
| 音频加载失败 | 播放条显示为 disabled 态（opacity 0.45）+ Toast「音频加载失败」（红色，需手动关闭） |
| 播放中离开页面 | 音频**暂停播放**；返回后保持暂停位置，用户可手动恢复 |
| 播放中切换其他 Tab | 音频**暂停播放** |
| 播放中接听电话 | 音频**暂停播放**，通话结束后保持暂停位置 |

## 七、长按选词功能

### 7.1 触发方式

| 属性 | 说明 |
|------|------|
| 触发对象 | 正文中的**中文文字**（仅中文行，不含拼音行和解释语言行） |
| 触发手势 | 长按 400ms |
| 触发反馈 | 长按触发时 Haptic 反馈（设备支持时） |

### 7.2 词义浮层内容

```
┌───────────────────────────┐
│  皇帝  huángdì            │  ← 汉字 + 拼音
│                           │
│  Emperor / Hoàng đế       │  ← 解释语言释义（跟随用户设置）
└───────────────────────────┘
         ▼
   [选中的文字位置]
```

| 元素 | 样式 |
|------|------|
| 浮层容器 | 毛玻璃卡片（`.glass-elevated`，圆角 16px），阴影 Level 3 |
| 汉字 | H3 字体（20px，Manrope 600） |
| 拼音 | Body S 字体（14px，Inter 400），次要文字色，位于汉字右侧，基线对齐 |
| 释义 | Body 字体（16px，Inter 400），颜色 Sky（`#0284c7`） |
| 浮层 padding | 16px |
| 浮层最大宽度 | 280px |

### 7.3 浮层位置

| 规则 | 说明 |
|------|------|
| 默认位置 | 出现在选中文字**正上方**，水平居中对齐选中文字 |
| 上方空间不足 | 改为出现在选中文字**正下方** |
| 左右超出屏幕 | 自动调整水平偏移，确保浮层完整显示在屏幕内（距屏幕边缘 ≥ 8px） |
| 小三角指示 | 浮层底部（或顶部）有小三角形指向选中文字 |

### 7.4 浮层关闭方式

| 方式 | 说明 |
|------|------|
| 点击浮层外区域 | 浮层淡出关闭（200ms） |
| 滚动页面 | 浮层淡出关闭 |
| 长按其他文字 | 关闭当前浮层 → 打开新浮层 |

### 7.5 释义多语言规则

| 用户配置 | 浮层释义显示 |
|---------|------------|
| UI 语言=英语 + 解释语言开启 | 显示英文释义 |
| UI 语言=越南语 + 解释语言开启 | 显示越南语释义 |
| UI 语言=汉语（解释语言自动关闭） | 仅显示拼音，不显示释义行 |
| 解释语言关闭 | 仅显示拼音，不显示释义行 |

## 八、字体大小调节

### 8.1 调节入口

导航栏右侧字体调节按钮，点击后弹出调节浮层（Popover，从按钮下方弹出）。

### 8.2 调节浮层

```
┌─────────────────────────┐
│  [A-]   [A 默认]   [A+]  │
└─────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| A- 按钮 | 缩小字体，标签「A」（Body S，14px），点击热区 ≥ 44×44pt |
| A 默认按钮 | 恢复默认字体，标签「A」（Body，16px），点击热区 ≥ 44×44pt |
| A+ 按钮 | 放大字体，标签「A」（Body L，18px），点击热区 ≥ 44×44pt |
| 当前选中态 | 选中的档位底部显示 Rose 色下划线（2px 粗） |

### 8.3 三档字体参数

| 档位 | 中文正文字号 | 拼音字号 | 解释语言字号 | 段间距 |
|------|------------|---------|------------|-------|
| 小 | 16px | 12px | 14px | 20px |
| 默认 | 18px | 14px | 16px | 24px |
| 大 | 22px | 16px | 18px | 28px |

### 8.4 调节行为

| 属性 | 说明 |
|------|------|
| 即时生效 | 点击档位后正文字体大小立即变化（300ms 过渡动画） |
| 记住选择 | 用户选择的字体档位存储在本地，下次打开任何文章详情页自动应用 |
| 已到最大/最小档 | A+ 按钮在最大档时 disabled（opacity 0.45）；A- 按钮在最小档时 disabled |
| 浮层关闭 | 点击浮层外区域关闭；选择档位后浮层不自动关闭（允许用户反复调试） |

| 元素 | 样式 |
|------|------|

| 卡片 padding | 16px |

### 9.3 发音按钮行为

| 步骤 | 行为 |
|------|------|
| 1 | 用户点击发音按钮 🔊 |
| 2 | 按钮进入播放态：图标变为 Lucide `Volume2` + 声波动画（3 条弧线依次扩散） |
| 3 | 播放该汉字/词语的中文标准发音（TTS 或预录音频） |
| 4 | 播放完毕 → 图标恢复静态 |
| 5 | 播放失败 → Toast「发音播放失败」（红色，需手动关闭） |

### 9.4 翻译显示规则

| 配置 | 翻译行显示 |
|------|----------|
| 解释语言开启 + UI=英语 | 显示英文翻译 |
| 解释语言开启 + UI=越南语 | 显示越南语翻译 |
| 解释语言关闭 | 隐藏翻译行，卡片高度自适应缩小 |
| UI 语言=汉语 | 隐藏翻译行 |

## 十、趣味小测验

位于核心词汇表下方，每篇文章末尾 1-3 道选择题，用于强化记忆和提升参与度。

### 10.1 区域标题

| 元素 | 样式 |
|------|------|
| 标题 | 「🧠 趣味小测验」，H2 字体（24px，Manrope 700） |
| 多语言 | 汉语="趣味小测验"；英语="Fun Quiz"；越南语="Trắc nghiệm vui" |
| 副标题 | 「测测你学到了多少！」（Body S，14px，次要文字色） |
| 副标题多语言 | 英语="Test what you've learned!"；越南语="Kiểm tra xem bạn đã học được gì!" |
| 间距 | 顶部 32px（距词汇表底部），底部 16px |

### 10.2 题目展示

每道题为独立的毛玻璃卡片（`.glass-card`，圆角 24px）。

```
┌──────────────────────────────────────┐
│  第 1 题 / 共 3 题                     │  ← 题号（Caption，12px，Amber 色）
│                                      │
│  秦始皇统一六国后，建立的朝代是？       │  ← 题干（Body L，18px）
│                                      │
│  ○ A. 汉朝                           │  ← 选项 A
│  ○ B. 秦朝                           │  ← 选项 B
│  ○ C. 唐朝                           │  ← 选项 C
```

## 十二、收藏按钮行为

文章详情页有两处收藏入口：元信息区的收藏按钮、导航返回时列表页的收藏图标。行为统一如下：

### 12.1 未登录用户

| 步骤 | 行为 |
|------|------|
| 1 | 用户点击收藏按钮（心形） |
| 2 | 触发全局登录弹窗（Bottom Sheet 从底部上滑） |
| 3a | 登录成功 → 弹窗关闭 → 自动执行收藏 → 心形变实心 Rose 色 + 弹跳动画 → Toast「已收藏」（绿色，2s） |
| 3b | 登录取消 → 弹窗关闭 → 心形保持空心，不执行收藏 |

### 12.2 已登录用户 — 收藏

| 步骤 | 行为 |
|------|------|
| 1 | 用户点击空心心形 |
| 2 | 心形立即变为实心 Rose 色（乐观 UI） |
| 3 | 弹跳动画：`scale(1) → scale(1.3) → scale(1)`，Spring 缓动（`cubic-bezier(0.34, 1.56, 0.64, 1)`），300ms |
| 4 | 同时发送收藏请求至服务器 |
| 5a | 请求成功 → Toast「已收藏」（绿色，顶部居中，2s 自动消失） |
| 5b | 请求失败 → 心形回退为空心 + Toast「收藏失败，请重试」（红色，需手动关闭） |

### 12.3 已登录用户 — 取消收藏

| 步骤 | 行为 |
|------|------|
| 1 | 用户点击实心心形 |
| 2 | 心形立即变为空心（乐观 UI） |
| 3 | 缩小动画：`scale(1) → scale(0.8) → scale(1)`，100ms |
| 4 | 同时发送取消收藏请求 |
| 5a | 请求成功 → Toast「已取消收藏」（绿色，2s 自动消失） |
| 5b | 请求失败 → 心形回退为实心 + Toast「操作失败，请重试」（红色，需手动关闭） |

### 12.4 Toast 文案多语言

| 场景 | 汉语 | 英语 | 越南语 |
|------|------|------|--------|
| 收藏成功 | 已收藏 | Saved | Đã lưu |
| 取消收藏成功 | 已取消收藏 | Unsaved | Đã bỏ lưu |
| 收藏失败 | 收藏失败，请重试 | Save failed, please retry | Lưu thất bại, vui lòng thử lại |
| 取消失败 | 操作失败，请重试 | Action failed, please retry | Thao tác thất bại, vui lòng thử lại |

## 十三、分享按钮行为

文章详情页有两处分享入口：导航栏分享按钮、元信息区分享按钮。行为统一如下：

| 步骤 | 行为 |
|------|------|
| 1 | 用户点击分享按钮 |
| 2 | 按钮进入 loading 态（图标旋转，150ms） |
| 3 | 生成文章分享卡片图片（含标题+封面缩略图+二维码） |

| 4b | 生成失败 → Toast「分享图片生成失败，请稍后重试」（红色，需手动关闭）→ 按钮恢复可点击 |

分享图片的详细规格见 [04-share-system.md](04-share-system.md)。

## 十四、状态矩阵

| 状态 | 表现 | 设计要求 |
|------|------|---------|
| **空状态 (Empty)** | 文章内容为空（极端情况） | 导航栏正常显示。内容区：品牌插画 + 文案「文章内容暂不可用」+ 「返回列表」按钮。多语言：英语="Article content unavailable"；越南语="Nội dung bài viết không khả dụng" |
| **加载中 (Loading)** | 首次请求文章数据 | 导航栏即时渲染。标题区：2 行灰色条骨架屏。元信息：1 行短灰色条。正文区：5-6 行长短交替的灰色条骨架屏。音频条：隐藏（数据未返回时不显示）。词汇/测验/推荐区：隐藏 |
| **首次加载 (First Load)** | 从列表页进入 | 同加载中，Push 转场过程中显示骨架屏 |
| **成功 (Success)** | 数据加载完成 | 标题淡入 → 元信息出现 → 正文从上到下渐入 → 音频条（如有）从底部滑入 → 词汇表/测验/推荐以 Stagger 动画渐入 |
| **错误 (Error)** | 数据请求失败 | 导航栏正常（可返回）。内容区：错误插画 + 文案「网络不给力，点击重试」+ 「重试」按钮（Rose 色药丸形） |
| **部分加载 (Partial)** | 正文加载成功但音频/词汇/测验加载失败 | 成功部分正常展示，失败部分在对应区域显示「加载失败，点击重试」 |
| **离线 (Offline)** | 无网络连接 | 顶部离线 Banner（Amber 色）。若有缓存 → 展示缓存的文章内容（音频不可播放）；若无缓存 → 错误状态 + 离线提示 |
