## 一、可分享内容类型

发现中国模块支持两种内容类型的分享：

| 类型 | 触发入口 | 分享形式 | 登录要求 |
|------|---------|---------|---------|
| **每日金句** | 类目首页金句卡片的分享按钮 | 生成精美图片 | 无需登录 |
| **文章** | 文章详情页导航栏分享按钮 / 元信息区分享按钮 | 生成分享卡片图片 | 无需登录 |

## 二、金句分享图片规格

### 2.1 图片尺寸

| 属性 | 值 |
|------|-----|
| 宽度 | 1080px |
| 高度 | 1920px（竖版，9:16 比例，适配社交媒体故事/状态） |
| 格式 | PNG（保证文字清晰度） |
| 文件大小目标 | ≤ 500KB |

### 2.2 图片构成

```
┌──────────────────────────────────┐
│                                  │
│         [背景图]                  │  ← 全幅背景
│                                  │
│  ┌────────────────────────────┐  │
│  │                            │  │
│  │    "千里之行，始于足下。"    │  │  ← 中文金句（居中，大字）
│  │                            │  │
│  │    Qiān lǐ zhī xíng,     │  │  ← 拼音（居中）
│  │    shǐ yú zú xià.         │  │
│  │                            │  │
│  │    — 老子《道德经》         │  │  ← 出处
│  │                            │  │
│  │    A journey of a thousand │  │  ← 解释语言翻译
│  │    miles begins with a     │  │
│  │    single step.            │  │
│  │                            │  │
│  └────────────────────────────┘  │  ← 毛玻璃文字区域
│                                  │
│                                  │
│  ┌──────┐          ┌──────────┐  │
│  │ Logo │          │ 二维码    │  │  ← 底部信息栏
│  │知语   │          │          │  │
│  │Zhiyu │          │          │  │
│  └──────┘          └──────────┘  │
│  知语 Zhiyu — 有趣的中文学习平台  │  ← 品牌 Slogan
│                                  │
└──────────────────────────────────┘
```

### 2.3 背景图规则

| 属性 | 说明 |
|------|------|
| 常规金句背景 | 中国文化主题的精美摄影图（山水/建筑/自然风光），从运营后台配置的背景图库中随机选取 |
| 节日金句背景 | 对应节日主题的专属背景图（春节=红灯笼/烟花；中秋=满月/桂花；端午=龙舟/粽子等） |
| 背景处理 | 原图叠加暗色半透明遮罩（`rgba(0, 0, 0, 0.35)`），确保文字可读性 |
| 图片比例 | 9:16，`object-fit: cover` 裁剪适配 |

### 2.4 金句文字排版

| 元素 | 字体 | 大小 | 颜色 | 位置 |
|------|------|------|------|------|
| 中文金句 | Manrope 或 PingFang SC 700 | 48px | 白色 `#ffffff` | 垂直居中偏上（距顶部 35%），水平居中 |
| 拼音 | Inter 400 | 24px | 白色 80% opacity | 中文金句下方 16px |
| 出处 | Inter 500 | 20px | Amber `#fde68a` | 拼音下方 24px |
| 解释语言翻译 | Inter 400 | 22px | 白色 70% opacity | 出处下方 32px |
| 文字区域 padding | — | — | — | 水平 60px |

**金句文字显示规则**：分享图片**始终显示完整内容**（中文+拼音+出处+解释语言翻译），不受用户个人配置影响。解释语言版本使用英文（作为国际通用语言，便于社交传播）。

### 2.5 品牌 Logo 位置

| 属性 | 说明 |
|------|------|
| Logo 内容 | 知语 Zhiyu 品牌 Logo（图标 + 文字） |
| 位置 | 图片底部左侧，距底部 80px，距左侧 60px |
| Logo 尺寸 | 高度 48px，宽度自适应 |
| 颜色 | 白色版 Logo |

### 2.6 二维码位置

| 属性 | 说明 |
|------|------|
| 二维码内容 | 知语 Zhiyu 官网/应用下载链接 |
| 位置 | 图片底部右侧，距底部 80px，距右侧 60px |
| 二维码尺寸 | 100px × 100px |
| 二维码样式 | 白色前景 + 透明背景，圆角 8px 白色背景框（padding 8px） |

### 2.7 品牌 Slogan

| 属性 | 说明 |
|------|------|
| 文案 | 「知语 Zhiyu — 有趣的中文学习平台」 |
| 多语言 | 英语="Zhiyu — Fun Chinese Learning"；越南语="Zhiyu — Học tiếng Trung thú vị" |
| 字体 | Inter 400，14px |
| 颜色 | 白色 50% opacity |
| 位置 | Logo 和二维码下方居中，距底部 40px |

### 2.8 毛玻璃文字区域

金句文字区域采用毛玻璃效果，增加可读性和设计品质：

| 属性 | 说明 |
|------|------|
| 背景 | `rgba(0, 0, 0, 0.25)` + `backdrop-filter: blur(16px)` |
| 圆角 | 32px |
| 边框 | `1px solid rgba(255, 255, 255, 0.15)` |
| padding | 上下 48px，左右 40px |
| 位置 | 垂直居中于图片中部 |

## 四、分享操作流程

### 4.1 完整流程

| 步骤 | 用户操作 / 系统行为 | 界面表现 |
|------|-------------------|---------|
| 1 | 用户点击分享按钮 | 分享按钮进入 loading 态（图标旋转） |
| 2 | 系统生成分享图片 | 等待 1-3 秒 |
| 3a | 生成成功 | 弹出分享预览面板（Bottom Sheet，从底部上滑，带背景模糊遮罩） |
| 3b | 生成失败 | Toast「图片生成失败，请稍后重试」（红色，需手动关闭）；按钮恢复可点击 |
| 4 | 用户在预览面板中查看分享图片 | 面板上部为图片预览（缩放至面板宽度，可双指缩放查看细节），下部为操作按钮 |
| 5 | 用户选择操作 | 「保存到相册」或「分享到社交平台」 |
| 6a | 保存到相册 → 成功 | Toast「已保存到相册」（绿色，2s 自动消失）；面板保持打开 |
| 6b | 保存到相册 → 失败（无权限） | Toast「请在设置中允许知语访问相册」（红色，需手动关闭） |
| 6c | 分享到社交平台 | 调用系统分享面板（OS 原生 Share Sheet） |
| 7 | 用户关闭面板 | 点击面板外区域 / 下拉面板 → 面板下滑关闭 |

### 4.2 分享预览面板

```
┌─────────────────────────────────┐
│  ──── （拖拽条，下拉关闭）        │
│                                 │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  │    [分享图片预览]         │    │  ← 图片预览区（圆角 16px）
│  │                         │    │
│  └─────────────────────────┘    │
│                                 │
│  [💾 保存到相册]  [↗ 分享]       │  ← 操作按钮（并排）
│                                 │
└─────────────────────────────────┘
```

| 元素 | 样式 | 说明 |
|------|------|------|
| 面板容器 | 毛玻璃面板（`.glass-elevated`，圆角 32px 32px 0 0） | 从底部上滑，300ms ease-out |
| 拖拽条 | 宽 40px，高 4px，居中，圆角 2px，颜色中性灰 30% | 下拉关闭手柄 |
| 图片预览 | 宽度为面板宽度 - 32px（两侧各 16px padding），高度按图片比例自适应，圆角 16px | 可双指缩放 |
| 保存按钮 | 毛玻璃按钮（圆角 `rounded-full`），Lucide `Download`（20px）+ 文字「保存到相册」 | 点击热区 ≥ 44×44pt |
| 分享按钮 | Rose 色药丸形按钮（`.btn-primary`），Lucide `Share2`（20px）+ 文字「分享」 | 点击热区 ≥ 44×44pt |
| 按钮区 padding | 水平 16px，顶部 16px，底部 16px + 安全区 | 两个按钮等宽并排，间距 12px |

**按钮多语言**：

| 按钮 | 汉语 | 英语 | 越南语 |
|------|------|------|--------|
| 保存到相册 | 保存到相册 | Save to Photos | Lưu vào Ảnh |
| 分享 | 分享 | Share | Chia sẻ |

## 五、分享渠道

### 5.1 保存到相册

| 属性 | 说明 |
|------|------|
| 行为 | 将生成的分享图片保存到设备本地相册 |
| 权限 | 首次触发时请求相册写入权限 |
| 权限被拒 | Toast「请在设置中允许知语访问相册」（红色，需手动关闭） |
| 保存成功 | Toast「已保存到相册」（绿色，2s） |
| 保存失败 | Toast「保存失败，请重试」（红色，需手动关闭） |

### 5.2 系统分享面板

| 属性 | 说明 |
|------|------|
| 行为 | 调用操作系统原生 Share Sheet（Web Share API） |
| 分享数据 | 图片文件 + 文本「来自知语 Zhiyu — 有趣的中文学习平台」 |
| 不支持 Web Share API | 降级为仅显示「保存到相册」按钮，隐藏分享按钮 |
| 用户取消分享 | 系统面板关闭，无额外提示 |
| 分享完成 | 系统面板关闭；可选：Toast「分享成功」（绿色，2s） |
