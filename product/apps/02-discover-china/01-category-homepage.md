## 二、每日金句卡片

```

│                                     │
```

| 元素 | 样式 | 说明 |

| 卡片内 padding | 上下 20px，左右 16px | — |

### 2.2 金句文本结构

金句内容严格遵循用户多语言配置：

**配置组合一：拼音+中文 模式 + 解释语言开启**
```
千里之行，始于足下。                         ← 中文金句（始终显示）
Qiān lǐ zhī xíng, shǐ yú zú xià.          ← 拼音（学习语言模式=拼音+中文时显示）
— 老子《道德经》                             ← 出处（始终显示）
A journey of a thousand miles begins         ← 解释语言翻译（解释语言开启时显示）
with a single step.
释义：再远大的目标，都要从脚下的第一步开始。    ← 解读（2-3 句，解释语言开启时显示解释语言版本）
```

**配置组合二：拼音+中文 模式 + 解释语言关闭**
```
千里之行，始于足下。                         ← 中文金句
Qiān lǐ zhī xíng, shǐ yú zú xià.          ← 拼音
— 老子《道德经》                             ← 出处
```

**配置组合三：纯中文 模式 + 解释语言开启**
```

— 老子《道德经》                             ← 出处

with a single step.
```

**配置组合四：纯中文 模式 + 解释语言关闭**
```
千里之行，始于足下。                         ← 中文金句
— 老子《道德经》                             ← 出处
```

**配置组合五：UI 语言=汉语（解释语言自动关闭）**
```
千里之行，始于足下。                         ← 中文金句
Qiān lǐ zhī xíng, shǐ yú zú xià.          ← 拼音（若选了拼音+中文模式）
```

### 2.3 分享按钮

| 属性 | 说明 |
|------|------|
| 图标 | Lucide `Share2` 图标（24px，继承当前文字颜色） |
| 位置 | 卡片头部右侧，与「✨ 每日金句」标签同行 |
| 标签文字 | 无文字标签，仅图标 |
| 可点击条件 | 始终可点击（未登录也可分享金句） |
| 点击热区 | ≥ 44×44pt |
| 点击后行为 | ① 按钮进入 loading 态（图标旋转，150ms） → ② 生成金句分享图片 → ③ 弹出分享预览面板（Bottom Sheet） → ④ 用户选择「保存到相册」或「分享到社交平台」 |
| 生成失败 | Toast 错误提示：「图片生成失败，请稍后重试」（红色 Toast，需手动关闭） + 分享按钮恢复可点击 |

分享图片生成规则详见 [04-share-system.md](04-share-system.md)。

### 2.4 节日模式

当运营后台为当天排期了节日金句时，金句卡片自动切换为节日模式。

#### 2.4.1 节日优先级规则

同一天有多个节日重叠时，按以下优先级取最高者：

| 优先级 | 节日类型 | 示例 |
|--------|---------|------|
| 1（最高） | 中国四大传统节日 | 春节、端午节、中秋节、元宵节 |
| 2 | 中国其他传统节日 | 清明节、重阳节、七夕节、冬至、腊八节 |
| 3 | 二十四节气 | 立春、雨水、惊蛰等 |
| 4 | 国际节日 | 元旦、国际劳动节、母亲节、父亲节等 |
| 5（最低） | 常规金句 | 非节日日期的日常金句 |

#### 2.4.2 节日装饰元素（P2）

| 节日 | 装饰元素描述 | 位置 |
|------|------------|------|
| 春节 | 红色灯笼小图标 × 2 | 卡片左上角和右下角 |
| 元宵节 | 花灯小图标 | 卡片右上角 |
| 端午节 | 龙舟/粽子小图标 | 卡片左下角 |
| 中秋节 | 月亮/月饼小图标 | 卡片右上角 |
| 重阳节 | 菊花小图标 | 卡片左上角 |
| 七夕节 | 喜鹊/星星小图标 | 卡片两侧 |
| 清明节 | 柳枝小图标 | 卡片左上角 |
| 冬至 | 雪花小图标 | 卡片右上角 |

装饰元素为半透明（opacity 0.3），不遮挡金句文字内容，尺寸 ≤ 32px。

### 2.5 无金句时的降级处理

若当天未配置金句（运营排期中断或数据拉取失败）：

| 场景 | 表现 |
|------|------|
| 后台未排期当日金句 | 自动复用最近一天的已发布金句，卡片正常显示 |
| 数据拉取失败（网络异常） | 卡片区域显示骨架屏（Skeleton），保留卡片占位高度；恢复后自动加载 |
| 金句库为空（极端情况） | 隐藏整个金句卡片区域，类目网格上移填充空间 |

```
┌───────────────────┐

│    [类目专属插图]   │  ← 插图区（卡片上半部分，宽高比 4:3）
```

| 元素 | 样式说明 |

用户登录后：
- 全部 12 张类目卡片正常显示，无遮罩
- 所有卡片可正常点击进入文章列表
- 遮罩消除时无特殊动画（页面刷新后直接呈现解锁状态）

### 3.6 卡片点击行为

**正常卡片（已解锁类目）**：

| 步骤 | 行为 |
|------|------|
| 1 | 用户点击卡片 |
| 2 | 卡片 active 态：`scale(0.97)` + 100ms |
| 3 | 页面转场：Push（新页面从右侧推入） |
| 4 | 进入该类目的文章列表页（[02-article-list.md](02-article-list.md)） |

**遮罩卡片（未登录 + 04-12 类目）**：

| 步骤 | 行为 |
|------|------|
| 1 | 用户点击遮罩卡片 |

## 五、状态矩阵

| 状态 | 表现 | 设计要求 |
|------|------|---------|
| **空状态 (Empty)** | 无任何类目数据时 | 品牌插画（中国文化主题）+ 文案「内容正在准备中，敬请期待」+ 3~4 个 `.glass-decor` 浮动方块装饰。金句区域显示降级文案「每日金句即将上线」 |
| **加载中 (Loading)** | 页面首次请求数据 | 金句区域：毛玻璃卡片形状的骨架屏（Skeleton），高度约 180px。类目网格：12 个卡片形状的骨架屏（保持 2 列布局），每个骨架屏含上部图片区+下部文字区的灰色色块 |
| **首次加载 (First Load)** | App 冷启动后首次进入发现中国 | 全屏骨架屏 + 品牌 Logo 呼吸动画（居中，opacity 在 0.4-1.0 间循环） |
| **成功 (Success)** | 数据加载完成 | 金句卡片和 12 类目卡片依次渐入（Stagger 动画，每项延迟 50ms），呈现完整内容 |
| **错误 (Error)** | 数据请求失败 | 页面中央：错误插画 + 文案「网络不给力，点击重试」（跟随 UI 语言）+ 「重试」按钮（Rose 色药丸形按钮）。点击重试 → 按钮进入 loading 态 → 重新请求 |
| **部分加载 (Partial)** | 金句加载成功但类目加载失败，或反之 | 成功部分正常展示，失败部分显示对应区域的错误提示和重试按钮（互相独立） |
| **离线 (Offline)** | 无网络连接 | 顶部显示离线 Banner：「当前无网络连接」（Amber 色背景）。若有缓存数据 → 展示缓存的金句和类目；若无缓存 → 显示空状态+离线提示 |
