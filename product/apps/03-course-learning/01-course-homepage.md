## 一、未登录用户访问行为

### 1.1 触发条件

未登录用户点击底部导航栏「系统课程」Tab。

### 1.2 行为

- **直接跳转登录页**（全屏跳转，非弹窗）
- 跳转原因：课程模块涉及学习进度、SRS 复习队列、考核、证书等强状态数据，匿名访问无意义
- 登录成功后自动返回课程首页

### 1.3 登录页返回行为

- 登录页顶部展示返回按钮，点击返回上一个 Tab（发现中国）
- 若用户从其他入口（如深链、推送）进入课程 Tab，返回按钮回到发现中国首页

## 二、首次进入课程 Tab 的入学测试引导卡片

### 2.1 触发条件

- 用户已登录
- 用户从未完成过入学测试（`placement_test_completed = false`）
- 用户未手动关闭过该引导卡片（`placement_card_dismissed = false`）

### 2.2 卡片位置

- 课程首页**最顶部**，位于页面标题区下方、Level 地图上方
- 横向全宽卡片，毛玻璃材质（`.glass-card`）
- 卡片内边距 16px

### 2.3 卡片内容

```
┌─────────────────────────────────────────────────┐
│  🎯  测一测你的中文水平                           │
│                                                   │
│  完成入学测试，我们将为你推荐最合适的学习起点。      │
│  预计用时 5 分钟，完成即获 100 知语币奖励！         │
│                                                   │
│  [ 开始测试 ]                        [ × 关闭 ]   │
└─────────────────────────────────────────────────┘
```

| 元素 | 规格 | 说明 |
|------|------|------|
| 图标 | 🎯 或自定义插图 | 卡片左上方装饰 |
| 标题 | "测一测你的中文水平" | H3 字号（20px），加粗，多语言（EN: "Test Your Chinese Level" / VI: "Kiểm tra trình độ tiếng Trung của bạn"） |
| 副标题 | "完成入学测试，我们将为你推荐最合适的学习起点。" | Body S 字号（14px），多语言 |
| 奖励提示 | "预计用时 5 分钟，完成即获 100 知语币奖励！" | Body S 字号（14px），知语币数字用 Amber 色高亮，多语言 |
| CTA 按钮 | "开始测试" | 主按钮（Rose 色，药丸形），点击 → 进入入学测试说明页 |
| 关闭按钮 | "×" 图标 | 右上角，Lucide `X` 图标，24px，点击 → 卡片收起 |

### 2.4 卡片关闭逻辑

- 用户点击关闭按钮 → 卡片收起并隐藏（带 300ms 向上滑出动画）
- 关闭后标记 `placement_card_dismissed = true`
- 卡片**不再自动出现**
- 用户仍可通过「个人中心 → 入学测试」入口进行测试

### 2.5 用户已完成测试后的卡片变化

- 测试完成后，`placement_test_completed = true`
- 卡片从引导模式变为**结果展示模式**（仅展示一次，之后隐藏）：

```
┌─────────────────────────────────────────────────┐
│  ✅  你的推荐起点：Level 3 — 进阶之门              │
│                                                   │
│  你的中文已有不错的基础！从 Level 3 开始正适合你。   │
│                                                   │
│  [ 开始学习 Level 3 ]                              │
└─────────────────────────────────────────────────┘
```

- 用户点击"开始学习 Level X" → 直接进入对应 Level 详情页
- 该结果卡片展示一次后，后续进入课程首页不再展示

### 2.6 用户忽略测试的处理

- 关闭引导卡片且未做测试 → 系统默认将用户设置为从 L1 开始学习
- L1 自动解锁为"进行中"状态
- 卡片关闭后 Level 地图正常展示，L1 为当前进行中的 Level

## 三、Level 地图路径

### 3.1 布局描述

- **纵向蜿蜒地图**，从屏幕底部（L1）到屏幕顶部（L12）
- 页面可垂直滚动，默认定位到用户当前正在学习的 Level 位置（居中展示）
- Level 节点沿 S 形蜿蜒路径排列（L1 左侧 → L2 右侧 → L3 左侧 → L4 右侧 …）
- 相邻 Level 节点之间有路径连线（曲线连接，非直线）
- 路径连线颜色：已通过的段落用 Rose 渐变色（实线），未通过的段落用灰色（虚线）
- 背景：深色主题下为深色渐变底，浅色主题下为浅灰底，搭配微弱的装饰元素（云朵、山峦剪影，暗示"学习攀登"隐喻）

### 3.2 每个 Level 节点展示的信息

每个 Level 节点是一个圆形/圆角方形卡片，包含以下信息：

| 元素 | 位置 | 规格 | 说明 |
|------|------|------|------|
| Level 编号 | 节点正中 | H2 字号，加粗 | "L1"、"L2"…"L12" |
| Level 名称 | 节点下方 | Body S 字号 | 多语言显示，如 "启蒙之旅" / "First Steps" / "Hành trình khai sáng" |
| HSK/CEFR 标签 | 名称下方 | Caption 字号，标签形 | "HSK 1 · A1"，Amber 色背景标签 |
| 进度百分比 | 节点内（进行中时） | Body S 字号 | "85%"，仅「进行中」状态显示 |
| 进度环 | 节点外圈 | 环形进度条 | 仅「进行中」状态显示，Rose 色填充 |
| 状态图标 | 节点右上角 | 16px 图标 | 根据状态展示不同图标（见 §3.3） |
| 免费标签 | 节点左上角（L1-L3） | Caption 字号 | "免费" / "Free" / "Miễn phí"，绿色标签 |
| 价格标签 | 节点左上角（L4-L12 锁定时） | Caption 字号 | "$6"，灰色标签 |

### 3.3 Level 节点的 5 种状态定义及视觉表现

#### 状态 1：进行中 🟢

| 维度 | 描述 |
|------|------|
| 适用条件 | 用户当前正在学习此 Level（已开始且未完成综合考核） |
| 视觉表现 | 节点外圈为 **Rose 色环形进度条**（显示完成百分比）；节点背景为毛玻璃高亮态（`.glass-elevated`）；节点尺寸比其他节点大 20%（突出当前焦点） |
| 进度闪烁 | 进度条末端有**呼吸闪烁动画**（Rose 色光晕，1.5s 循环，`ease-in-out`） |
| 节点内容 | Level 编号 + 进度百分比（如"85%"）+ Level 名称 + HSK/CEFR 标签 |
| 点击行为 | 进入该 Level 详情页（单元列表） |
| 无障碍标签 | "Level X，进行中，已完成 XX%，点击继续学习" |

#### 状态 2：已完成 ✅

| 维度 | 描述 |
|------|------|
| 适用条件 | 用户已通过此 Level 的综合考核 |
| 视觉表现 | 节点外圈为**完整的绿色环**（`#22c55e`）；右上角 ✅ 勾选图标（Lucide `CheckCircle`，绿色填充）；节点背景为正常毛玻璃态 |
| 节点内容 | Level 编号 + "已完成" 文字 + Level 名称 + HSK/CEFR 标签 |
| 点击行为 | 进入该 Level 详情页（可回顾已完成内容和证书） |
| 无障碍标签 | "Level X，已完成，点击查看详情" |

#### 状态 3：已解锁待学 ⚪

| 维度 | 描述 |
|------|------|
| 适用条件 | 免费级别（L1-L3）用户尚未开始学习；或上一 Level 已完成但本 Level 尚未开始 |
| 视觉表现 | 节点外圈为**灰色虚线环**；节点背景为半透明毛玻璃态（opacity 0.7）；无状态图标 |
| 节点内容 | Level 编号 + Level 名称 + HSK/CEFR 标签 + 免费标签（L1-L3）或无价格标签（已购买的 L4-L12） |
| 点击行为 | 进入该 Level 详情页，开始学习第一个单元 |
| 无障碍标签 | "Level X，已解锁，点击开始学习" |

#### 状态 4：已购买待学 🔵

| 维度 | 描述 |
|------|------|
| 适用条件 | 付费级别（L4-L12）用户已购买但尚未开始学习，且前序 Level 尚未完成综合考核 |
| 视觉表现 | 节点外圈为 **Sky 色虚线环**（`#0284c7`）；右上角 🔵 蓝色圆点标记；节点背景为半透明毛玻璃态 |
| 节点内容 | Level 编号 + Level 名称 + HSK/CEFR 标签 + "已购买" 标签（Sky 色背景） |
| 点击行为 | 若前序 Level 未完成 → Toast 提示"请先完成 Level X 综合考核"；若前序已完成 → 进入该 Level 详情页 |
| 无障碍标签 | "Level X，已购买待学习，请先完成前一级别" |

#### 状态 5：锁定 🔒

| 维度 | 描述 |
|------|------|
| 适用条件 | 付费级别（L4-L12）用户尚未购买 |
| 视觉表现 | 节点外圈为**灰色实线环**；右上角 🔒 锁图标（Lucide `Lock`，灰色）；节点背景为低透明度毛玻璃态（opacity 0.4）；节点整体有灰度滤镜 |
| 节点内容 | Level 编号 + Level 名称 + HSK/CEFR 标签 + 价格标签"$6"（灰色背景） |
| 点击行为 | 弹出付费墙弹窗（详见 [03-paywall.md](03-paywall.md)） |
| 无障碍标签 | "Level X，未解锁，6 美元购买，点击查看购买选项" |

### 3.5 Level 地图动画效果（P1）

| 动画 | 描述 | 时长 | 缓动 |
|------|------|------|------|
| 路径连线绘制 | 页面首次加载时，路径从 L1 向上依次绘制至用户当前 Level | 800ms | Standard `cubic-bezier(0.2, 0, 0, 1)` |
| 当前进度闪烁 | 当前 Level 节点外圈 Rose 色光晕呼吸闪烁 | 1.5s 循环 | `ease-in-out` |
| 节点入场 | 各 Level 节点依次淡入 + 微上滑 | 每个 100ms 延迟，单个 300ms | Decelerate |
| 滚动定位 | 打开页面后自动滚动到当前 Level 位置居中 | 500ms | Standard |

| 元素 | 规格 | 说明 |
|------|------|------|
| 图标 | 🔄 或 Lucide `RotateCcw` | 左侧装饰 |
| 标题 | "今日复习" | H3 字号，多语言（EN: "Today's Review" / VI: "Ôn tập hôm nay"） |

## 四、状态矩阵

| 页面状态 | 描述 | 设计要求 |
|---------|------|---------|
| **空状态 (Empty)** | 不会出现 — 课程 Tab 始终展示 12 个 Level 节点 | — |
| **加载中 (Loading)** | 学习数据加载中 | Skeleton 骨架屏：12 个圆形占位符沿蜿蜒路径排列 + 顶部引导卡片占位矩形 |
| **首次加载 (First Load)** | 用户首次进入课程 Tab | 引导卡片 + Level 地图节点依次入场动画（见 §3.5） |
| **成功状态 (Success)** | 数据加载完成 | Level 地图正常展示，自动滚动到当前进度 Level |
| **错误状态 (Error)** | 网络请求失败 | 全屏错误提示："加载失败，请检查网络后重试"（多语言）+ 重试按钮（Rose 色） |
| **部分加载 (Partial)** | 不适用 — 课程首页无分页加载 | — |
| **离线 (Offline)** | 用户无网络连接 | 顶部 Banner "网络连接已断开"（多语言）；展示本地缓存的学习进度数据（只读模式，点击 Level 节点提示"请连接网络"） |
