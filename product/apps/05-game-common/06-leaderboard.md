## 一、排行榜入口

### 1.1 入口位置

| 入口 | 位置 | 行为 |
|------|------|------|
| 游戏大厅底部快捷入口 | 底部快捷入口栏"排行榜"按钮 | 跳转至排行榜页面，默认展示"全服总榜"Tab |
| 段位展示栏 | 顶部段位栏点击总星数区域 | 跳转至排行榜页面，默认展示"全服总榜"Tab，并滚动至用户自己的排名位置 |

### 1.2 页面跳转

- 跳转动画：横向滑入（从右侧滑入），时长 300ms
- 顶部导航栏显示标题"排行榜" + 左侧返回按钮

## 二、排行榜页面布局

### 2.1 整体结构

```
┌─────────────────────────────────────┐
│  ← 返回          排行榜             │ ← 顶部导航栏
├─────────────────────────────────────┤
│  [全服总榜]  [赛季榜]  [各游戏榜]    │ ← Tab 切换栏
├─────────────────────────────────────┤
│                                     │
│  🥇 1  头像  昵称A  👑 王者 120分   │ ← 排名列表
│  🥈 2  头像  昵称B  ⭐ 星耀I 91星   │
│  🥉 3  头像  昵称C  ⭐ 星耀II 87星  │
│  4     头像  昵称D  💠 钻石I 66星   │
│  5     头像  昵称E  💠 钻石II 63星  │
│  ...                                │
│                                     │
├─────────────────────────────────────┤
│  42  [我的头像]  我的昵称  🥇 黄金II │ ← 自己排名（固定底部）
│                          47星       │
├─────────────────────────────────────┤
│  [底部 Tab 导航栏]（全局）            │
└─────────────────────────────────────┘
```

### 2.2 Tab 切换栏

| Tab | 标签文字 | 排序依据 | 说明 |
|-----|---------|---------|------|
| **全服总榜** | "全服总榜" | 累计总星数（降序） | 所有用户的段位排名 |
| **赛季榜** | "赛季榜" | 当前赛季星数变化（降序） | 仅统计本赛季的段位进度 |
| **各游戏榜** | "各游戏榜" | 指定游戏的胜场数（降序） | 需二级选择具体游戏 |

#### Tab 视觉规范

- Tab 栏高度 44px，背景毛玻璃面板
- 选中 Tab：Rose 色文字 + 底部 2px Rose 色下划线
- 未选中 Tab：灰色文字
- Tab 切换动画：下划线滑动过渡，时长 200ms
- Tab 文字使用 **Body S**（14px），字重 600

### 2.3 各游戏榜的二级选择

当切换至"各游戏榜"Tab 时，Tab 栏下方展示游戏选择器：

```
┌─────────────────────────────────────┐
│  [各游戏榜] Tab 已选中               │
├─────────────────────────────────────┤
│  🎮 汉字切切切  ▼                    │ ← 下拉选择器
├─────────────────────────────────────┤
│  排名列表...                         │
└─────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| **下拉选择器** | 展示当前选中游戏名称 + 下拉箭头 |
| **下拉菜单** | 12 款游戏列表，每项含游戏图标（20×20px）+ 游戏名称 |
| **默认选中** | 第一款游戏（G1 汉字切切切） |
| **选中后** | 下拉菜单关闭，排行榜列表刷新为该游戏的排名 |

## 三、排行榜列表项

### 3.1 列表项布局

```
┌─────────────────────────────────────────┐
│  🥇  [头像]  昵称XXXXXX    👑 王者 120分 │
│  1                                      │
└─────────────────────────────────────────┘
```

### 3.2 元素定义

| 元素 | 说明 | 字体/尺寸 |
|------|------|----------|
| **排名序号** | 数字序号，前 3 名使用奖牌图标替代 | 排名 1-3：奖牌图标 24×24px（🥇🥈🥉）；排名 4+：Body（16px），字重 700，灰色 |
| **头像** | 用户头像，圆形 | 40×40px，`border-radius: 50%` |
| **昵称** | 用户昵称，单行截断 | Body（16px），字重 500，最大宽度 120px |
| **段位图标** | 用户当前段位的小图标 | 20×20px |
| **段位名/星数** | 全服总榜/赛季榜展示段位名+星数；各游戏榜展示胜场数 | Body S（14px），右对齐 |

### 3.3 前三名特殊样式

| 排名 | 特殊样式 |
|------|---------|
| 第 1 名 | 列表项背景使用 Amber 色 5% 不透明度；昵称文字 Amber 色 |
| 第 2 名 | 列表项背景使用灰色 5% 不透明度 |
| 第 3 名 | 列表项背景使用铜褐色 5% 不透明度 |
| 第 4+ 名 | 无特殊背景 |

### 3.4 列表项交互

- 点击列表项：无跳转操作（MVP 无用户主页）
- 长按列表项：无操作
- 列表支持下拉刷新（Pull-to-Refresh）
- 列表触底无限加载（每页 50 条，触底加载下一页）

## 四、自己的排名高亮

### 4.1 位置

- 固定在列表底部、Tab 导航栏上方
- 不随列表滚动
- 始终可见

### 4.2 样式

```
┌─────────────────────────────────────────┐
│  42  [我的头像]  我的昵称    🥇 黄金II   │ ← 毛玻璃背景 + Rose 色左边框
│                              47 ⭐      │
└─────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| **背景** | 毛玻璃面板（`.glass-card`）+ 左侧 3px Rose 色竖线 |
| **内容** | 与列表项结构一致（排名 + 头像 + 昵称 + 段位） |
| **高度** | 64px |

### 4.3 特殊情况

| 情况 | 展示 |
|------|------|
| 用户从未参加过 PK | 排名显示"—"，段位显示当前段位（青铜 III 1 星） |
| 用户排名在列表可视范围内 | 列表中对应行高亮（Rose 色左边框），底部固定栏仍显示 |

## 五、排行榜刷新机制

### 5.1 刷新策略

| 维度 | 说明 |
|------|------|
| **全服总榜** | 定时刷新，每 5 分钟更新一次（服务端缓存） |
| **赛季榜** | 定时刷新，每 5 分钟更新一次 |
| **各游戏榜** | 定时刷新，每 10 分钟更新一次 |
| **自己的排名** | 实时查询（每次进入排行榜页时查询最新排名） |

### 5.2 手动刷新

- 支持下拉刷新（Pull-to-Refresh）
- 下拉刷新后强制从服务端获取最新数据（绕过缓存）
- 刷新完成后 Toast 提示："排行榜已更新"（蓝色，2 秒消失）

### 5.3 赛季切换

- 赛季结束后，"赛季榜"自动切换为新赛季
- 旧赛季排行榜数据不可查看（MVP 阶段暂不支持历史赛季查看）
- 赛季榜 Tab 标签动态展示赛季名："S1 赛季榜"

## 六、状态矩阵

| 状态 | 触发条件 | UI 表现 |
|------|---------|---------|
| **空状态** | 当前赛季尚无人参加 PK（赛季初期可能出现） | 列表区域展示插画 + "暂无排名数据，快去 PK 抢占第一吧！" + "去 PK"按钮（跳转大厅） |
| **加载中** | 切换 Tab 或首次进入，排名数据加载中 | 列表区域展示 10 行骨架屏（序号+圆形头像+文字横条）；底部自己排名区域显示骨架屏 |
| **首次加载** | 同"加载中" | — |
| **成功状态** | 排名数据加载完成 | 正常展示排名列表 + 自己排名 |
| **错误状态** | 排名数据请求失败 | 列表区域展示"加载失败" + 重试按钮；底部自己排名显示缓存数据或"—" |
| **部分加载** | 列表触底加载更多中 | 列表底部展示加载指示器（旋转圆点），加载完成后新数据追加至列表 |
| **离线状态** | 设备无网络 | 顶部离线 Banner；展示上次缓存的排行榜数据（如有）；下拉刷新时提示"请连接网络" |
