## 一、游戏概述

| 属性 | 说明 |
|------|------|
| **一句话描述** | 交换相邻词语方块形成近义词/反义词/搭配词配对消除——把语义理解变成三消游戏的爽感 |
| **灵感来源** | Candy Crush（三消游戏） |
| **题库范围** | L3 课程累计 2200 词汇 + 近义词/反义词/搭配词关系库（M3 词汇与语义 + M4 语法与句法） |
| **单局时长** | 3-5 分钟（无尽模式视存活时间；限步模式固定 30 步） |
| **情绪目标** | 思考、满足、连锁快感 |

## 二、游戏子模式

### 2.1 子模式总览

| 子模式 | 人数 | 影响段位 | 结束条件 | 特殊规则 |
|--------|------|---------|---------|---------|
| **单人·无尽** | 1 人 | 否 | 无可消除且无洗牌次数 | 无限消除，挑战最高分 |
| **单人·限步** | 1 人 | 否 | 30 步用完 | 限制 30 步内追求最高分 |
| **1v1 PK** | 2 人 | 是 | 90 秒倒计时归零 | 同棋盘，比同时间内谁分高 |
| **多人对战** | 4 人 | 是 | 90 秒倒计时归零 | 4 人同棋盘限时竞速 |

### 2.2 单人·无尽模式

- 棋盘持续自动补充新词
- 无法消除时可使用"洗牌"道具（每局 3 次免费洗牌）
- 3 次洗牌用完后仍无可消除 → Game Over
- 挑战目标：总分最高

### 2.3 单人·限步模式

- 30 步操作限制（每次交换 = 1 步，无效交换不计步数）
- 步数用完 → 进入结算
- 无洗牌道具
- 挑战目标：30 步内最高分

### 2.4 1v1 PK 模式

- 双方看到**完全相同的初始棋盘**
- 各自独立操作，互不干扰
- 90 秒倒计时结束时比较分数
- 分数高者胜；相同则消除次数多者胜；仍相同则平局

### 2.5 多人对战模式

- 4 人看到**相同初始棋盘**
- 各自独立操作
- 90 秒倒计时结束按分数排名
- 第 1 名 +1 星，其他不扣星

## 四、核心交互

### 4.1 语义匹配规则（核心差异化设计）

> **G3 的消除不是颜色匹配，而是语义匹配。** 这是区别于普通三消游戏的核心差异。

**匹配判定规则**：

| 匹配类型 | 说明 | 示例 |
|---------|------|------|
| **近义词配对** | 两个词语意思相近 | "美丽" ↔ "漂亮"、"高兴" ↔ "快乐" |
| **反义词配对** | 两个词语意思相反 | "大" ↔ "小"、"高" ↔ "矮"、"快" ↔ "慢" |
| **搭配词配对** | 两个词语构成常见搭配 | "阳光" ↔ "明媚"、"春风" ↔ "拂面" |

**消除条件**：交换两个**相邻方块**后，如果其中一个方块与交换后新位置的**相邻方块**形成上述任一类型的配对 → 消除配对的所有方块。

**特殊规则**：
- 3 消：将 3 个存在配对关系的方块排成一行/列（如"大"—"小"—"矮"，其中"大↔小"反义 + "小↔矮"近义，三者均消除）
- 实际判定由服务端预计算每个方块的可配对对象列表，客户端查表判定
- 棋盘生成时保证至少有 3 组可消除配对

### 4.2 方块交换操作

**用户动作**：点击一个方块 → 点击相邻方块（或从一个方块滑向相邻方块）

**选中态**：
1. 点击第一个方块 → 方块放大 105% + 白色光圈边框（2px）+ 轻微上浮（y-4px）+ 底部提示条显示该词信息
2. 可选中态的相邻方块（构成有效配对）显示微弱呼吸光效（提示可交换）

**有效交换**（构成配对）：
1. 两个方块以 200ms 平滑交换位置
2. 配对消除（见 §4.3）

**无效交换**（不构成任何配对）：
1. 两个方块尝试交换（移动到一半 100ms）
2. 自动回弹到原位（100ms）
3. 两个方块同时左右微晃 2 次（位移 ±4px，持续 200ms）

**音效**：
- 选中：轻微 "嗒" 点击音
- 有效交换：无（交由消除音效处理）
- 无效交换：沉闷 "嗡" 提示音

### 4.3 消除动画

**触发**：交换后形成有效配对

**视觉反馈**（按时间顺序）：

1. **配对高亮**（0ms）：参与配对的方块发出白色脉冲光（扩散 110%，持续 150ms）
2. **类型标签弹出**（100ms）：配对方块之间弹出关系标签（如 `近义 ✓` / `反义 ✓` / `搭配 ✓`），对应颜色，H3 20px，上浮 40px 后淡出（持续 800ms）
3. **方块消除**（200ms）：方块缩小至 0% + 同色粒子爆散（8 个粒子），持续 300ms
4. **得分飘字**（250ms）：消除位置飘出分数（Amber 色，H3 20px），上移 60px 后淡出
5. **方块下落**（500ms）：上方方块受重力下落填补空位（带弹跳效果：落到位后回弹 4px，200ms）
6. **新方块补入**（600ms）：顶部生成新方块从上方滑入（200ms），带轻微弹跳
7. **连锁检测**（800ms）：检测新棋盘是否产生新的配对 → 如有则继续消除（cascade）

**音效**：
- 消除：清脆 "叮铃" 消除音，音调随 Combo 递增
- 方块下落：轻微 "哒哒" 落地音
- 连锁触发：递增音阶 "叮-铃-当"

### 4.4 特殊方块系统

| 特殊方块 | 生成条件 | 视觉 | 效果 |
|---------|---------|------|------|
| **炸弹方块** 💣 | 4 消（一次消除 4 个方块） | 方块内叠加💣图标 + 方块边缘发光脉冲 | 激活时消除以自身为中心 3×3 范围内所有方块 |
| **彩虹方块** 🌈 | 5 消（一次消除 5 个方块） | 彩虹渐变色 + ✨ 闪烁 | 激活时消除棋盘上**所有同匹配类型**的方块（如激活时配对的是近义词，则消除所有近义词底色方块） |
| **横扫方块** ⚡ | L 型消除（两条线交叉消除） | 方块内叠加⚡图标 + 水平/垂直线条动画 | 激活时消除整行或整列（取决于交换方向：水平交换消整行，垂直交换消整列） |

**特殊方块激活方式**：与特殊方块交换并形成有效配对时激活

**特殊方块连锁**：两个特殊方块互相交换时效果叠加：
- 炸弹 + 炸弹 = 5×5 范围消除
- 炸弹 + 横扫 = 消除十字形 3 行 + 3 列
- 彩虹 + 任意 = 消除棋盘上**两种匹配类型**的所有方块
- 彩虹 + 彩虹 = 全屏清除（所有方块消除）

### 4.5 连锁消除（Cascade）

**触发**：方块消除后，新方块下落/补入后自动形成新配对

**连锁规则**：
1. 每次方块下落/补入后，扫描全棋盘
2. 发现新配对 → 自动消除（玩家无需操作）
3. 继续扫描，直到无更多配对
4. 每层连锁之间间隔 300ms（让玩家看清发生了什么）

**连锁视觉**：
- 连锁计数显示在棋盘上方：`Chain ×2` → `Chain ×3`（金色，H2 24px），每层 +1
- 连锁层数越高，消除粒子越华丽（颜色越鲜艳、粒子数越多）
- 连锁 ≥ 3 层时，屏幕微震（位移 ±2px，100ms）
- 连锁 ≥ 5 层时，背景出现短暂金色光芒扩散

**连锁分数加成**：

| 连锁层数 | 该层消除基础分倍率 |
|---------|-----------------|
| 第 1 层（首次消除） | ×1 |
| 第 2 层 | ×1.5 |
| 第 3 层 | ×2 |
| 第 4 层 | ×3 |
| 第 5 层+ | ×5 |

## 五、难度递增

### 5.1 无尽模式难度曲线

| 累计消除数 | 棋盘变化 | 词汇难度 | 备注 |
|-----------|---------|---------|------|
| 0-30 次 | 初始 7×7，3 种匹配类型均匀分布 | 常用词（Top 500） | 新手友好 |
| 30-60 次 | 同上 | 增加近义词辨析难度（如 "美丽/漂亮/好看" 三者关系） | 需更仔细观察 |
| 60-100 次 | 棋盘中出现 2-3 个「石头方块」（不可移动，不可消除） | 增加多义词 | 策略性增加 |
| 100-150 次 | 石头方块增至 4-5 个 | 增加成语（四字词占 1 格） | 空间受限 |
| 150 次+ | 石头方块 5-6 个 + 每 30 秒底部上推 1 行 | 全词库 | 高压环境 |

### 5.2 限步模式

- 30 步固定，难度不递增
- 棋盘固定无石头方块
- 重点在于策略规划（每步最优消除路径）

### 5.3 PK/多人模式

- 初始棋盘固定（所有玩家相同）
- 90 秒内不递增难度
- 无石头方块（保证公平）

## 六、生命值系统

G3 不使用传统生命值系统：

| 模式 | 结束条件 |
|------|---------|
| **无尽模式** | 无可消除 + 洗牌次数用完 → Game Over |
| **限步模式** | 30 步用完 → 结算（非 Game Over，所有人都会走到这一步） |
| **PK/多人** | 90 秒倒计时归零 → 结算 |

**无尽模式 Game Over 前置流程**：
1. 棋盘检测无可消除配对
2. 如果洗牌次数 > 0 → 屏幕中央弹出 `无法消除！使用洗牌？`（洗牌按钮高亮闪烁）
3. 点击洗牌 → 所有方块旋转飞散后重新落入（1 秒动画）→ 继续游戏
4. 洗牌次数 = 0 且无可消除 → Game Over

**Game Over 动画**：
1. 所有方块依次从外圈到内圈翻转变灰（每圈间隔 150ms）
2. 屏幕中央 `Game Over`（Display 48px，白色）
3. 1.5 秒后过渡结算页

## 七、PK 模式特殊规则

### 7.1 同棋盘机制

- 匹配成功后，服务端生成一个初始棋盘（7×7，49 个词语方块）
- 双方客户端接收相同的初始棋盘
- 各自独立操作 → 由于操作不同，两人棋盘迅速分叉
- 服务端不实时同步棋盘状态（仅同步分数）

### 7.2 对方状态反馈

| 事件 | 己方屏幕反馈 |
|------|------------|
| **对方消除** | 对手分数跳动 + 右上角对手头像旁短暂出现消除类型图标（近义🟢/反义🔴/搭配🔵） |
| **对方触发特殊方块** | 对手头像旁闪烁对应特殊方块图标（💣/🌈/⚡） |
| **对方触发高连锁**（≥ 3 层） | 己方屏幕顶部短暂显示 `对手连锁 ×3！` 提示条（800ms） |
| **对方分数超越己方** | 己方分数变 Rose 红色闪烁一次 + 提示 `追上去！` |

### 7.3 PK 结束条件

1. **90 秒倒计时归零** → 分数高者胜
2. **分数相同** → 消除次数多者胜
3. **仍相同** → 平局（不影响段位）

### 7.4 多人对战排名

- 4 人按分数排名
- 第 1 名 +1 星，第 2-4 名不扣星
- 排名相同时按消除次数排

## 八、上瘾机制详情

### 8.1 特殊方块特效

| 特殊方块 | 激活特效 |
|---------|---------|
| **炸弹方块** 💣 | 3×3 范围爆炸：中心白色闪光 → 方块碎裂飞散 → 爆炸冲击波涟漪扩散。音效：低沉爆炸音 + 碎裂音 |
| **彩虹方块** 🌈 | 彩虹光线从方块射出 → 命中所有同类型方块 → 方块依次闪光消除（每个间隔 50ms，形成扫射效果）。音效：升调扫射音 + 叮叮叮连续消除音 |
| **横扫方块** ⚡ | 闪电从方块向行/列两端同时射出 → 经过的方块依次电击消除（间隔 30ms）。音效：电击"滋滋"音 |
| **全屏清除**（彩虹×2） | 全屏白色闪光 → 所有方块同时爆裂 → 彩色碎片漫天飞舞 → 新方块雨点般落入。音效：盛大管弦乐爆发（2 秒） |

## 九、状态矩阵

### 9.1 游戏进行中状态

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **正常游戏** | 棋盘可交互，等待玩家操作 | 所有 HUD 正常显示 |
| **方块交换中** | 两方块正在交换位置 | 禁止其他操作，交换动画 200ms |
| **消除中** | 配对正在消除 + 方块下落 + 新方块补入 | 禁止操作，播放完整消除动画 |
| **连锁中** | cascade 多层连续消除 | 禁止操作，每层间隔 300ms |
| **特殊方块激活** | 炸弹/彩虹/横扫方块效果展开 | 禁止操作，播放完特效后恢复 |
| **无可消除** | 棋盘检测无有效配对 | 弹出洗牌提示（如有次数）或 Game Over |
| **暂停** | 玩家暂停 | 参见 `08-hud-landscape.md` |
| **Game Over** | 洗牌用完且无可消除 | 方块翻转变灰 → Game Over → 结算 |

### 9.2 PK 模式状态

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **双方对战中** | 各自独立操作棋盘 | 己方完整交互 + 对手分数实时更新 |
| **对手领先** | 对方分数高于己方 | 己方分数区闪红 + 提示文字 |
| **时间即将结束** | 最后 10 秒 | 倒计时变红闪烁 + 紧迫音效 |
| **对手断线** | 对手网络断开 | 等待 15 秒重连，超时判己方获胜 |

### 9.3 空状态 / 异常状态

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **棋盘生成中** | 等待服务端下发棋盘 | 7×7 Skeleton 方块闪烁 + "准备中..." |
| **加载失败** | 网络错误 | 居中错误提示 + "重试" + "返回大厅" |
