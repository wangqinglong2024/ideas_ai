## 一、证书生成条件

### 1.1 级别通过证书

| 条件 | 说明 |
|------|------|
| 触发条件 | 通过级别综合考核（总分 ≥85 且单项 ≥60） |
| 生成时机 | 综合考核结果页展示"通过"后**自动生成** |
| 是否需用户操作 | 不需要，系统自动生成并保存到个人中心"我的证书" |
| 每 Level 一张 | 每个 Level 通过后生成对应证书（L1 一张、L2 一张…最多 12 张） |
| 重考不覆盖 | 已通过并生成证书后，重考不会覆盖原有证书（保留首次通过的证书，日期以首次通过为准） |

### 1.2 阶段证书（P2 — 后续迭代）

| 条件 | 说明 |
|------|------|
| 触发条件 | 完成一个阶段内所有 Level 的综合考核（如启蒙段 L1+L2+L3 全部通过） |
| 阶段划分 | 启蒙段（L1-L3）、基础段（L4-L6）、核心段（L7-L9）、高阶段（L10-L12） |
| 生成时机 | 阶段内最后一个 Level 综合考核通过后自动生成 |
| P2 优先级 | 第二迭代实现 |

## 二、证书内容

### 2.1 证书包含的信息要素

| 要素 | 说明 | 位置 |
|------|------|------|
| **知语品牌标识** | 知语 Zhiyu Logo | 证书顶部居中 |
| **证书标题** | "学习成就证书" / "Certificate of Achievement" | Logo 下方居中 |
| **用户昵称** | 用户在个人中心设置的昵称 | 证书主体区域居中 |
| **Level 名称** | 如 "Level 3：进阶之门"（中英双语） | 昵称下方 |
| **HSK 等级** | 如 "HSK 3" | Level 名称下方左侧 |
| **CEFR 等级** | 如 "CEFR B1" | Level 名称下方右侧 |
| **考核分数** | 如 "综合成绩：92 分" | HSK/CEFR 下方居中 |
| **通过日期** | 如 "2026 年 4 月 17 日" | 分数下方 |
| **证书编号** | 唯一编号，格式："ZY-L03-20260417-XXXX" | 底部左侧小字 |
| **知语品牌 Slogan** | "知其语，通其意" | 底部居中小字 |

### 2.2 证书不包含的信息

- 不包含用户真实姓名（仅使用昵称，保护隐私）
- 不包含用户头像
- 不包含各模块详细分数（仅总分）
- 不包含二维码或验证链接（MVP 阶段不做证书在线验证）

### 3.2 证书卡片视觉描述

**整体风格**：
- 证书卡片为独立的视觉组件，模拟纸质证书的高级质感
- 卡片背景：深色模式使用渐变暗色（`#1a1a2e` → `#16213e`），浅色模式使用米白渐变（`#fefce8` → `#fffbeb`）
- 卡片边框：Amber 色渐变边框（2px），模拟金色镶边
- 卡片圆角：`--radius-2xl`（24px）
- 卡片阴影：Level 3 阴影 + 额外的 Amber 色微弱辉光（`0 0 24px rgba(217, 119, 6, 0.1)`）
- 卡片内边距：32px

**Logo 区域**：
- 知语 Zhiyu Logo 居中展示，宽度 80px
- Logo 下方 8px 间距

**标题区域**：
- "学习成就证书" — 中文标题，H2（24px），Amber 色（`#d97706`），600 字重
- "Certificate of Achievement" — 英文副标题，Body S（14px），中性灰色
- 标题两侧各有一条短装饰线（Amber 色，宽度 40px，高度 1px）

**用户昵称**：
- 用户昵称居中，H1（32px），700 字重，主文字色
- 昵称过长时自动缩小字号（最小 20px），确保单行显示

**Level 和等级信息**：
- Level 名称中英双语展示，H3（20px），600 字重
- HSK 和 CEFR 等级以两个并排的标签卡片展示
  - 标签样式：毛玻璃卡片，圆角 `--radius-sm`（8px），内边距 8px 16px
  - 标签文字：Body（16px），700 字重
  - HSK 标签边框：Sky 色；CEFR 标签边框：Amber 色

**分数和日期**：
- 综合成绩：Body L（18px），中性灰色
- 日期：Body（16px），中性灰色

**底部区域**：
- 分隔线：1px，中性灰 20% 透明度
- 证书编号：Caption（12px），中性灰色，左对齐
- Slogan：Caption（12px），中性灰色，居中

### 3.3 放大查看

- 证书卡片支持双指缩放（Pinch to Zoom）查看细节
- 也可双击卡片进入全屏查看模式（横屏展示，背景暗色）
- 全屏模式下顶部有"×"关闭按钮

## 四、证书保存

### 4.1 自动保存到个人中心

- 证书生成后自动保存到「个人中心 → 我的证书」列表
- 无需用户手动操作
- 证书数据存储在服务端（Supabase），换设备登录同一账号可看到所有证书

### 4.2 "保存图片"功能

**操作流程**：
1. 用户在证书详情页点击"保存图片"按钮（底部左侧按钮）
2. 系统将证书卡片渲染为高清图片
3. 弹出系统保存对话框（或直接下载到设备相册/下载文件夹）
4. 保存成功后显示 Toast："证书图片已保存"（成功色，2s 自动消失）
5. 保存失败显示 Toast："保存失败，请重试"（错误色，需手动关闭）

**图片规格**：

| 属性 | 值 |
|------|-----|
| 格式 | PNG（透明背景不适用于证书，使用实色背景） |
| 分辨率 | 1080 × 1440 px（3:4 竖版） |
| DPI | 300（适合打印） |
| 文件大小 | 预计 500KB-1MB |
| 背景 | 与证书卡片背景一致（深色/浅色跟随当前主题） |

## 五、证书分享

### 5.1 分享操作流程

1. 用户在证书详情页点击"分享证书"按钮（底部右侧按钮）
2. 系统生成证书分享图片（与保存图片相同规格，额外叠加分享元素）
3. 调用系统分享面板（Web Share API / 系统分享 Sheet）
4. 用户选择分享渠道（社交媒体、邮件、消息等）
5. 分享成功后显示 Toast："分享成功"

**如果系统不支持 Web Share API**（部分浏览器）：
- 显示自定义分享弹窗：
  - "复制链接"（复制证书分享页 URL）
  - "保存图片后分享"（先保存图片，提示用户手动分享）

### 5.2 分享图片内容

分享图片在证书基础上增加以下元素：

| 元素 | 位置 | 说明 |
|------|------|------|
| 知语 App 品牌水印 | 图片底部 | "知语 Zhiyu — 你的中文学习伙伴" |
| 下载引导 | 底部水印下方 | "扫码开始你的中文学习之旅" 或 App 下载链接文字 |

**分享图片规格**：
- 与保存图片相同（1080 × 1440 px，PNG，300 DPI）
- 增加品牌水印区域（底部额外 120px 高度）
- 总尺寸：1080 × 1560 px

### 5.3 顶部导航栏的分享入口

- 证书详情页右上角也有"分享 ↗"图标按钮（Lucide `share-2` 图标）
- 点击效果与底部"分享证书"按钮一致
