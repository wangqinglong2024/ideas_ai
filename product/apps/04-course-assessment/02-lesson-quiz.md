## 一、触发条件

### 1.1 自动弹出规则

- 每个课时的学习内容（导入 → 新知 → 练习 → 小结）全部完成后，**自动弹出**课时小测验
- 弹出方式：底部向上滑入的全屏面板（Bottom Sheet 形式），动画 400ms，Decelerate 缓动
- 用户无法跳过小测验直接进入下一课时（必须完成全部题目）
- 测验开始前不显示单独的"开始"按钮，直接展示第一道题

### 1.2 复习模式触发

- 用户在课时列表中点击已完成的课时 → 重新学习 → 学习完毕后可再做小测验
- 复习模式的小测验题目随机重新抽取（与首次做的题目不同）
- 复习模式的结果不影响课程进度（仅用于巩固复习）

│─────────────────────────────────────────────│
│                                               │
│  [题目区域]                                   │
│                                               │

### 2.2 顶部信息栏

| 元素 | 位置 | 说明 |
|------|------|------|
| "课时小测验" 标题 | 左上 | 固定文案，Body S（14px），中性灰色 |
| 题目进度 | 右上 | "第 X/N 题"，Body S（14px），中性灰色 |
| 进度条 | 标题下方全宽 | 线性进度条，高度 4px，已完成部分 Rose 色，未完成部分中性灰 10% 透明度 |

- 进度条按题目完成比例推进（如 5 题完成 2 题 = 40%），每完成一题进度条动画前进（300ms，Standard 缓动）

### 2.3 题目区域

- 题目区域样式遵循 [01-question-types.md](./01-question-types.md) 中定义的各题型规范
- 题目区域居中展示，左右留白 24px
- 题目区域距顶部信息栏间距 32px

## 三、题目数量和来源

| 属性 | 规则 |
|------|------|
| 题目数量 | 3-5 题（由后端根据课时知识点数量决定） |
| 出题范围 | 仅从**本课时**的知识点中抽取 |
| 出题逻辑 | 覆盖本课时的核心知识点（生字、词汇、语法点），确保不重复考同一知识点 |
| 题目随机 | 每次做测验时从题库中随机抽取（复习时与首次做的题不同） |
| 难度分布 | 简单 60% + 中等 30% + 较难 10%（鼓励为主，不打击自信） |

---

## 四、题型范围

课时小测验仅使用以下 3 种题型（保持轻量，每题 15-30 秒可完成）：

| 题型 | 使用频率 | 说明 |
|------|---------|------|
| **单选题** | 高（每次测验必有） | 词义选择、拼音选择、语法选择等 |
| **拼音标注题** | 中（L1-L4 常见） | 为 1-3 个汉字标注拼音（含声调） |
| **排序组句题** | 中（L2+ 出现） | 3-5 个词语排列成正确句子 |

- 不使用听力题（课时小测验定位为快速巩固，不需要音频设备）
- 不使用填空题和阅读理解（留给单元测评和综合考核）

## 五、答对表现

### 5.1 视觉反馈

- 正确选项/答案：绿色高亮（背景 `rgba(34, 197, 94, 0.1)` + 边框 `#22c55e`）+ ✅ 图标
- 反馈动画：正确选项弹性放大（`scale(1.02)` → `scale(1)`，300ms Spring 缓动）

### 5.2 鼓励文案

答对后底部弹出鼓励文案卡片（毛玻璃面板），文案从以下列表中随机选取：

| 中文 | English | Tiếng Việt |
|------|---------|-----------|
| 太棒了！ | Awesome! | Tuyệt vời! |
| 答对啦！ | Correct! | Đúng rồi! |
| 继续加油！ | Keep going! | Tiếp tục nào! |
| 真厉害！ | Amazing! | Giỏi quá! |
| 完美！ | Perfect! | Hoàn hảo! |
| 学得真好！ | Great job! | Học giỏi quá! |

- 文案展示语言跟随 UI 语言设置
- 文案卡片以 Decelerate 缓动从底部滑入（200ms），停留 1 秒后淡出
- 文案卡片消失后自动出现"下一题 →"按钮

### 5.3 答对后操作

- 其他选项变为 disabled 灰色态，不可再点击
- 1 秒后底部出现"下一题 →"按钮（Rose 色主按钮）
- 点击"下一题"进入下一道题（向左滑出/向右滑入动画，300ms）
- 如果是最后一题答对，按钮文案变为"查看结果"

## 六、答错表现

### 6.1 视觉反馈

- 用户选中的错误选项/答案：红色高亮（背景 `rgba(239, 68, 68, 0.1)` + 边框 `#ef4444`）+ ❌ 图标
- 正确选项自动高亮：绿色高亮 + ✅ 图标
- 反馈动画：错误选项轻微抖动（左右 shake 动画，200ms）

### 6.2 解析展示

答错后底部弹出解析卡片（毛玻璃面板），包含：

```
┌───────────────────────────────────────┐
│ ❌ 正确答案是 A                          │ ← 标题行（错误色）
│                                         │
│ 解析：                                   │
│ "高兴"的拼音是 gāo xìng（一声+四声）。   │ ← 解析文字
│ "兴"在"高兴"中读四声(xìng)，             │
│ 不要和"兴奋"的一声(xīng)混淆。           │
│                                         │
│ 📋 已加入复习队列                         │ ← SRS 提示
└───────────────────────────────────────┘
```

- 解析文字按用户语言配置渲染（中文始终显示，拼音行和解释语言行按配置显示）
- "📋 已加入复习队列" 提示以 Sky 色文字展示，表示该知识点已自动加入 SRS 间隔重复系统
- 解析卡片可上下滚动（当解析文字较长时）

### 6.3 SRS 加入规则

- 答错的题目对应的知识点（生字/词汇/语法点）自动加入 SRS 复习队列
- SRS 复习间隔遵循艾宾浩斯遗忘曲线：1天 → 2天 → 4天 → 7天 → 15天 → 30天
- 用户无需手动操作，系统自动处理

### 6.4 答错后操作

- 用户阅读解析后，底部出现"下一题 →"按钮
- 不强制用户必须阅读完解析才能继续（按钮在解析卡片弹出 1 秒后出现）
- 如果是最后一题答错，按钮文案变为"查看结果"

### 7.2 正确率展示

- 页面顶部大标题："课时小测验完成！"（H2，24px，700 字重）
- 环形进度图：
  - 圆环直径 120px，线宽 8px
  - 正确部分为成功色（`#22c55e`），错误部分为错误色（`#ef4444`）
  - 圆环中心显示正确率百分比（H1，32px，700 字重）
  - 百分比下方显示"X/N 题"（Body S，14px）
- 全对时（100%）：圆环为成功色 + 中心显示 🎉 + "全部答对！"

### 7.3 各题正误列表

- 列表展示每道题的结果：题号 + 题型 + 答对/答错
- 答对的题：✅ 绿色标记
- 答错的题：❌ 红色标记 + "已加入复习" 灰色标签
- 点击答错的题 → 展开显示：题目内容 + 用户答案 + 正确答案 + 解析

### 7.4 "继续学习"按钮

| 场景 | 按钮文案 | 点击后跳转 |
|------|---------|----------|
| 当前单元还有下一课时 | "继续学习" | 进入下一课时的学习内容 |
| 当前单元全部课时已完成 | "返回单元" | 返回单元列表页，此时"开始测评"按钮已解锁 |
| 复习模式 | "返回课时" | 返回当前课时详情页 |

### 7.5 无通过门槛说明

- 课时小测验**没有通过门槛**，无论正确率多少都可以继续学习
- 不显示"通过"或"未通过"的判定文字
- 正确率 100% 时额外显示鼓励文案："太厉害了，全部答对！"
- 正确率 < 60% 时底部额外显示温馨提示："不用担心，答错的题已加入复习队列，系统会帮你安排复习。"（Sky 色文字）

## 八、重复做测验的行为

| 场景 | 行为 |
|------|------|
| 首次做 | 课时学完后自动弹出，必须完成 |
| 复习再做 | 从课时列表点击已完成课时 → 重新学习 → 测验自动弹出 |
| 题目变化 | 每次做测验时从题库中随机重新抽取，题目可能与前次不同 |
| 结果记录 | 仅记录最近一次的测验结果（覆盖前次） |
| SRS 影响 | 每次答错均会刷新该知识点在 SRS 中的复习间隔（重置为 1 天后复习） |

## 九、状态矩阵

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **空状态 (Empty)** | 不适用 | 课时小测验不存在空状态，始终有 3-5 题 |
| **加载中 (Loading)** | 测验题目加载中 | Skeleton 骨架屏：题干区域 + 4 个选项区域的骨架占位，禁止纯 Spinner |
| **首次加载 (First Load)** | 课时学完后弹出测验 | 底部滑入动画（400ms） + 直接展示第一题（无额外等待页） |
| **成功 (Success)** | 答对反馈 | 绿色高亮 + ✅ + 鼓励文案弹出 + 进度条前进动画 |
| **错误 (Error)** | 答错反馈 | 红色高亮 + ❌ + 正确答案高亮 + 解析卡片弹出 + "已加入复习队列"提示 |
| **部分加载 (Partial)** | 不适用 | 题目数量少，一次性加载全部题目 |
| **离线 (Offline)** | 网络断开时 | 题目已在课时学习时预加载到本地，可离线作答；测验结果在恢复网络后自动上传；顶部显示"离线模式"灰色 Banner |
| **意外退出** | 杀进程/切后台 | 不保存答题进度，重新进入课时学习后需重新完成小测验（题量少，2-3 分钟可重做） |
| **网络中断提交** | 提交结果时断网 | 结果缓存到本地，恢复网络后自动上传；显示"结果已保存，将在联网后同步"Toast |
