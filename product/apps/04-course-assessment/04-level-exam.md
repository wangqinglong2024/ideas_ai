## 一、触发条件

### 1.1 解锁规则

- 该 Level 的**全部单元阶段测评**通过（≥70 分）后，级别综合考核入口解锁
- 解锁时系统推送通知："Level X 全部单元测评已通过！综合考核已解锁，来挑战吧！"
- 未全部通过时，考核入口显示为锁定态 + 提示"请先通过全部单元测评"

### 1.2 重考间隔

- 首次考核：无等待间隔，解锁后即可开始
- 重考（未通过后）：需等待 **24 小时**后才能再次考核
- 24 小时倒计时从上次提交时间开始计算

### 1.3 前置条件检查

| 检查项 | 条件 | 未满足时行为 |
|--------|------|------------|
| 单元测评全部通过 | 本 Level 所有单元测评 ≥70 分 | 考核入口锁定 + 提示未通过的单元 |
| 重考冷却期 | 距上次未通过 ≥24 小时 | 考核入口显示倒计时（详见第九节） |
| 登录状态 | 用户已登录 | 跳转登录弹窗 |
| 课程权限 | L1-L3 免费 / L4+ 已购买 | 触发付费墙 |

### 4.2 模块切换 Tab

**Tab 栏位置**：固定在顶部，水平排列 4 个模块标签

**Tab 样式**：
- 当前选中的 Tab：文字 Rose 色 + 下方 2px Rose 色下划线
- 未选中的 Tab：文字中性灰色
- Tab 间距均匀分布，占满屏幕宽度

**Tab 上的进度指示**：
- 每个 Tab 右上角显示该模块的完成状态微标：
  - 全部作答：✅ 绿色圆点
  - 部分作答：数字角标（如 "5/8" 表示已答 5 题共 8 题）
  - 未作答：无标记

**模块间切换规则**：
- 用户可**自由切换**任意模块（不强制按顺序完成）
- 切换模块时，当前模块的作答进度自动保存
- 切换动画：左右滑动（300ms，Standard 缓动）

### 4.3 每个模块内的题目进度

- 模块 Tab 下方显示当前模块的题目进度："第 X/N 题"
- 模块内进度条：仅反映当前模块的答题进度（非全局进度）
- 模块内可通过"上一题"/"下一题"翻页

### 4.4 顶部总计时器

- 计时器显示整体考核用时（从点击"开始考核"开始计时）
- 综合考核**不强制限时**，计时器仅供参考
- 计时器样式：⏱️ 图标 + MM:SS，中性灰色
- 中途退出恢复后，计时器从上次暂停处继续

### 4.5 "提交全部"按钮

- 当所有模块的所有题目全部作答后，底部栏出现"提交全部"Rose 色主按钮
- 如果仍有未作答题目，底部栏显示"还有 X 题未作答"灰色提示文字 + disabled 态的"提交全部"按钮
- 用户也可在有未作答题目时强制提交：长按"提交全部"按钮 2 秒 → 触发强制提交确认弹窗

### 4.6 防误操作设计

| 场景 | 处理方式 |
|------|---------|
| 意外退出（杀进程/切后台） | 每答一题自动保存进度（本地 + 服务端），重新进入后可从上次位置继续；保存进度有效期 24 小时 |
| 网络中断 | 题目已预加载，可离线继续答题；提交时断网则缓存结果，恢复后自动重试 |
| 误触返回 | 弹出确认弹窗："确定退出考核？您的答题进度已保存。" → "继续考核"/"保存并退出" |
| 重复点击提交 | 首次点击后按钮 Loading + disabled |
| 长时间无操作 | 超过 30 分钟无操作 → 弹出提示"您已离开一段时间，是否继续考核？" → 继续/保存退出；计时器暂停 |

## 五、考核模块详情

### 5.1 听力模块

| 属性 | 说明 |
|------|------|
| 题目数 | 6-10 题（按 Level 递增） |
| 题型 | 听力选择题（单选） |
| 音频播放规则 | 每道题的音频最多播放 **2 次** |
| 音频内容 | 短对话、短句、段落听力（按 Level 递增难度） |
| 分值 | 每题 3-5 分，模块总分根据题目数量计算后折合为 100 分制 |

**听力模块特殊规则**：
- 进入听力模块前，页面顶部显示提示："请确保设备音量已开启 🔊"
- 音频自动播放第一次（用户点击"开始"后），无需手动点击播放
- 第二次播放需用户手动点击播放按钮
- 音频播放期间选项可操作（不必等播放完才能选）

### 5.2 阅读模块

| 属性 | 说明 |
|------|------|
| 题目数 | 6-10 题（分 2-3 篇阅读材料） |
| 题型 | 阅读理解选择题（单选）、判断正误 |
| 文章长度 | L1-L3：100-400 字；L4-L6：400-1000 字；L7-L9：1000-2000 字；L10-L12：2000-3000 字 |
| 分值 | 每题 3-5 分 |

**阅读模块特殊规则**：
- 文章展示区和题目区上下分屏，可独立滚动（详见 [01-question-types.md](./01-question-types.md) 第七节）
- 文章区域支持字体大小调节（A+/A-）
- 多篇文章时，每篇文章及其子题作为一组，组间有明确分隔线

### 5.3 词汇语法模块

| 属性 | 说明 |
|------|------|
| 题目数 | 10-15 题 |
| 题型 | 单选题、多选题、填空题、排序组句题、拼音标注题 |
| 分值 | 每题 2-5 分（填空和排序分值较高） |

**词汇语法模块特殊规则**：
- 题型混合出现，每道题页面顶部标注题型名称（如"选择题"、"填空题"）
- 不同题型的交互遵循 [01-question-types.md](./01-question-types.md) 中的通用规范

### 5.4 书写模块

| 属性 | L1-L6 | L7-L12 |
|------|-------|--------|
| 题目数 | 2-3 题 | 2 题 |
| 题型 | 拼音标注 + 排序组句 + 填空 | 看图说话/简答题（AI 批改） |
| 评分方式 | 系统自动评分 | AI 批改评分（需等待 10-30 秒） |
| 分值 | 每题 10-15 分 | 每题 20-30 分 |

**书写模块（L7+ AI 批改）特殊规则**：
- 提交后需等待 AI 批改，页面跳转到 AI 批改等待页（详见第六节）
- AI 批改完成前，其他模块的结果可先行展示
- 如果 AI 批改超时（>60 秒），显示"评分超时，请稍后查看结果"提示

```
│            └─────────────┘                   │
│                                               │
│        AI 正在评阅您的书写答卷                   │
│        预计等待 10-30 秒                        │
│                                               │
│        ━━━━━━━━━━━━━●━━━━━━━━━               │ ← 模拟进度条
│                                               │
│        💡 小知识                                │
│        "书"字最早的写法像一只手拿着              │
│        毛笔在竹简上写字。                        │
│                                               │
│                                               │
└─────────────────────────────────────────────┘
```

### 6.2 等待页元素

| 元素 | 说明 |
|------|------|
| AI 图标 | 圆形图标，内含 AI/脑 图案，带脉冲呼吸动画（`scale(1) → scale(1.1) → scale(1)`，2s 循环） |
| "评分中..." 文字 | 文字后带三点加载动画（...），H3 20px |
| 说明文字 | "AI 正在评阅您的书写答卷" + "预计等待 10-30 秒"，Body 16px，中性灰 |
| 模拟进度条 | 非真实进度，以匀速缓慢前进的动画模拟评分过程（30 秒内从 0% 到 90%，完成后跳到 100%） |
| 趣味小知识 | 💡 在等待期间展示一条随机的中文/汉字趣味知识，每 10 秒切换一条 |

### 6.3 等待完成后

- AI 评分完成 → 进度条跳到 100% → 0.5 秒后自动跳转到结果页
- 跳转动画：等待页向上淡出 + 结果页从下方滑入（400ms，Decelerate 缓动）

### 6.4 超时处理

- 如果 AI 评分超过 60 秒未返回：
  - 显示"评分时间较长，您可以先查看其他模块的成绩"
  - 显示"查看已出成绩"按钮 → 跳转到结果页，书写模块显示"评分中"
  - 评分完成后推送通知，结果页自动刷新

## 八、重考规则

| 规则项 | 说明 |
|--------|------|
| 等待间隔 | **24 小时**（从上次提交时间开始计算） |
| 题目变化 | 重考时题目从题库中**随机重新抽取**，与上次不完全相同 |
| 重考次数 | **不限次数** |
| 成绩记录 | 记录所有考核成绩，以**最高分**为最终成绩 |
| 通过后重考 | 已通过的综合考核可重新考核（用于刷分），不影响已颁发的证书 |
| 模块构成 | 每次重考的模块构成不变（听力/阅读/词汇语法/书写），但题目内容不同 |
| AI 批改 | 重考的书写部分同样需要 AI 批改 |

## 十、状态矩阵

| 状态 | 描述 | 设计要求 |
|------|------|---------|
| **空状态 (Empty)** | 不适用 | 综合考核始终有完整题目 |
| **加载中 (Loading)** | 考核题目加载中 | 点击"开始考核"后全屏骨架屏（模块 Tab + 题目区域占位），2-3 秒内加载完毕 |
| **首次加载 (First Load)** | 首次进入考核 | 展示考核说明页 → 加载题目 → 展示第一个模块第一题 |
| **成功 (Success)** | 提交成功 + 通过 | AI 批改等待页（如有书写模块）→ 庆祝粒子动画 → 通过结果页 → 证书预览 |
| **错误 (Error)** | 提交成功 + 未通过 | AI 批改等待页（如有）→ 未通过结果页 → 未通过原因 + 薄弱知识点 + 24h 倒计时 |
| **部分加载 (Partial)** | 不适用 | 题目一次性预加载 |
| **离线 (Offline)** | 网络断开时 | 题目已预加载可离线答题；提交时断网缓存结果等待联网；AI 批改需联网（断网时提示等待联网） |
| **意外退出** | 杀进程/切后台 | 每答一题自动保存；重新进入显示"继续考核"；保存进度 24 小时有效 |
| **AI 批改等待** | 书写模块 AI 评分中 | 等待页动画 + 趣味小知识 + 模拟进度条；超时 60 秒提供"先查看其他成绩"选项 |
| **AI 批改超时** | AI 评分超 60 秒 | "评分时间较长"提示 + "查看已出成绩"按钮；评分完成后推送通知 |
| **重考冷却中** | 24 小时冷却期内 | 考核入口显示倒计时 + "去复习"按钮；考核按钮 disabled |
| **误触返回** | 点击返回键 | 确认弹窗 → "继续考核"/"保存并退出" |
| **长时间无操作** | >30 分钟无操作 | 弹出提示"您已离开一段时间" → 继续/退出；计时器暂停 |
