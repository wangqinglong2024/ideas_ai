# PlayLingo — MVP 产品需求文档总览

> **版本**: v1.0 | **日期**: 2026-04-15
> **产品形态**: Web 端（PWA）| 移动端优先 + PC 端响应式适配
> **首发市场**: 越南 🇻🇳
> **MVP 目标**: 1 个月完成开发，提供拼音→汉字→词汇→语法的完整 HSK1 学习链路

---

## 一、产品定位

**一句话描述**: 东南亚第一款以精致 2D 游戏为载体的中文学习 Web 应用——边玩游戏，边学中文。

**核心差异化**:
- 不是"课程 + 练习"，而是"策略游戏 + 自然习得"
- 月费 $2.99 vs Duolingo $12.99，极致性价比
- 完整越南语本地化，精准服务越南中文学习者
- 分享赚佣金机制，用户即渠道

## 四、核心用户流程（Happy Path）

### 4.1 新用户首次体验流程

```
访问网站 → 落地页（品牌介绍 + CTA）
  → 点击「开始学习」
  → 注册页（Google / Facebook / 邮箱）
  → 注册成功
  → 选择学习目标（工作/旅游/考试/兴趣，单选）
  → 角色创建（选头像 + 输入昵称）
  → 进入世界地图（拼音群岛自动解锁）
  → 新手引导气泡（指引点击第 1 关）
  → 进入关卡 1-1（声母 b/p/m/f 教学）
  → 完成关卡 → 结算画面（星级 + XP + 金币）
  → 返回世界地图 → 第 2 关解锁
  → 引导尝试小游戏入口
```

### 4.2 日常学习流程（老用户）

```
打开 App/网页 → 世界地图
  → 查看待复习词汇提醒（红点）
  → 选择：
    A) 继续推进关卡 → 进入下一关 → 完成 → 结算
    B) 进入 SRS 复习 → 复习一组词汇 → 完成
    C) 玩小游戏 → 选择一款 → 开始 → 结束排行
  → 完成每日 1 关 → 打卡成功 → streak +1
  → 返回或退出
```

### 4.3 付费转化流程

```
用户推进到汉字谷地（第 2 地图区域）
  → 完成前 50% 免费关卡
  → 尝试进入付费关卡 → 弹出会员引导
  → 展示：免费版 vs 会员版对比表
  → 突出「首月 $0.99」优惠
  → 用户选择套餐（月付/季付/年付/终身）
  → 跳转 Stripe 支付页面
  → 支付成功 → 返回 App → 会员标识激活
  → 全部关卡解锁 + 去广告 + 离线可用
```

### 4.4 推广佣金流程

```
用户进入「推广中心」（个人主页入口）
  → 查看当前等级与佣金比例
  → 点击「复制推广链接」或「生成推广海报」
  → 分享到 Facebook / Zalo / TikTok
  → 好友点击链接 → 跳转注册页（?ref=uid 自动绑定）
  → 好友注册成功 → 推荐关系建立
  → 好友付费 → 推荐人获得佣金（30 天冷却后可提现）
  → 推荐人在推广中心查看收益明细
  → 累计 ≥ $10 → 申请提现
```

## 五、系统状态全局约定

### 5.1 每个页面必须覆盖的 7 种状态

| 状态 | 要求 |
|------|------|
| **空状态** | 插画 + 引导文案 + CTA 按钮 |
| **加载中** | Skeleton 骨架屏，禁止纯 Spinner |
| **首次加载** | 全屏骨架屏 + Logo 呼吸动画 |
| **成功** | 轻量 Toast（顶部滑入，2s 自动消失） |
| **错误** | 说人话的错误描述 + 重试按钮 |
| **部分加载** | 底部加载指示器 |
| **离线** | 顶部 Banner 提示 + 缓存数据可浏览 |

### 5.2 全局交互约定

- 所有操作 100ms 内必须有视觉反馈
- 破坏性操作（退出登录、删除账号）需二次确认弹窗
- 表单即时校验（blur 后），提交失败不清空已填内容
- Toast 通知从顶部滑入，成功绿色 2s/失败红色需手动关闭/信息蓝色 3s
- 页面转场：前进从右滑入，后退向右滑出，Tab 切换淡入淡出

### 5.3 广告规则（免费用户）

- 每完成 3 个关卡，关卡结算画面后插入 1 个插页广告（15 秒可跳过）
- 非游戏页面底部固定 Banner 广告（高度 50px）
- 激励视频广告：用户主动触发（看广告获得金币 / Streak 修复）
- 会员用户：无任何广告

## 六、北极星指标与度量

| 指标类型 | 指标名 | MVP 目标（上线 1 个月） |
|---------|--------|----------------------|
| **北极星** | 周活跃学习用户（WAU） | 500 |
| 获客 | 注册转化率（访问→注册） | ≥ 50% |
| 激活 | 首日完成 ≥1 关的用户比例 | ≥ 60% |
| 留存 | 次日留存率 | ≥ 40% |
| 留存 | 7 日留存率 | ≥ 20% |
| 变现 | 免费→付费转化率 | ≥ 3% |
| 推荐 | 推广链接点击→注册转化率 | ≥ 30% |
| 学习 | 平均每用户每日完成关卡数 | ≥ 2 |
