## 一、系统概述

### 1.1 用户故事

> 作为 PlayLingo 用户，我想邀请朋友注册和付费，从中获得佣金奖励，以便获得收入并有传播动力。

### 1.2 核心机制

- **单层推荐（1 级）**：仅推荐人获得被推荐人付费的佣金
- **佣金来源**：被推荐人的**每次续费**都产生佣金（非仅首次）
- **佣金阶梯**：根据推荐人的累计推荐收入升级佣金比例
- **结算周期**：实时计入，30 天后可提现
- **所有用户均可参与**：免费用户和会员均可生成推荐链接

## 二、推荐链接

### 2.1 链接格式

```
https://playlingo.app/?ref={用户唯一推荐码}
```

- 推荐码：6-8 位字母数字组合，自动生成，不可自定义（MVP）
- 每个用户有且仅有一个推荐码

### 2.2 归属绑定规则

| 规则 | 说明 |
|------|------|
| 绑定时机 | 新用户通过推荐链接打开页面时，ref 参数存入 localStorage |
| 注册绑定 | 注册时检查 localStorage 中的 ref 参数，绑定推荐关系 |
| 有效期 | 点击推荐链接后 **7 天内** 注册有效 |
| 唯一性 | 每个用户只能被一个推荐人绑定（首次归属原则） |
| 不可改 | 绑定后推荐关系永久不变 |
| 自推禁止 | 用户不能推荐自己（后端校验） |

### 2.3 分享功能

**分享入口**：推广中心页面 + 个人主页「邀请好友」按钮

**分享方式**：
- 复制推荐链接
- 分享海报图片（带二维码 + 推荐码）
- 直接分享到社交媒体（Facebook、Zalo、Messenger）

## 三、佣金阶梯

### 3.1 四级阶梯

| 等级 | 累计推荐收入 | 佣金比例 | 等级名称 |
|------|------------|---------|---------|
| Lv.1 | $0 — $9.99 | 5% | Người mới（新手） |
| Lv.2 | $10 — $99.99 | 10% | Đại sứ（大使） |
| Lv.3 | $100 — $999.99 | 15% | Đối tác（合作伙伴） |
| Lv.4 | $1,000+ | 20% | Đối tác vàng（金牌合作伙伴） |

**说明**：
- 累计推荐收入 = 历史所有已结算佣金总和
- 升级后**不降级**（即使某月没有新推荐收入）
- 佣金比例基于**推荐人当前等级**计算

### 3.2 佣金计算示例

```
张三推荐李四注册
李四购买月费 $2.99
张三当前等级 Lv.1（5%）

佣金 = $2.99 × 5% = $0.1495 ≈ $0.15（保留 2 位小数，四舍五入）

下个月李四续费 $2.99
张三等级不变
佣金 = $2.99 × 5% = $0.15

假设张三累计佣金达到 $10.00 → 自动升级到 Lv.2（10%）
之后李四续费 $2.99
佣金 = $2.99 × 10% = $0.30
```

## 六、提现系统

### 6.1 提现规则

| 规则 | 说明 |
|------|------|
| 最低提现金额 | $10.00 |
| 提现方式 | PayPal（MVP 仅支持 PayPal） |
| 冷却期 | 佣金产生后 30 天可提现 |
| 处理时间 | 申请后 3-5 个工作日到账 |
| 手续费 | 平台不收手续费；PayPal 自身手续费由用户承担 |
| 退款扣回 | 被推荐人退款 → 对应佣金扣回 |

### 6.2 提现流程

```
推广中心 → 点击可提现金额
     ↓
```

## 七、反作弊规则

### 7.1 作弊检测

| 风险行为 | 检测方式 | 处理 |
|---------|---------|------|
| 自推自买 | 推荐人和被推荐人同 IP/设备指纹 | 不发放佣金 |
| 批量假注册 | 同一推荐码短时间大量注册 | 冻结推荐功能，人工审核 |
| 刷退款 | 被推荐人频繁购买后退款 | 扣回佣金，标记风险 |
| 多账号刷 | 同一设备/IP 多个账号推荐 | 合并计算，超额部分不发放 |

### 7.2 处罚机制

- 轻度违规：佣金冻结 + 警告通知
- 重度违规：永久取消推荐资格，扣回所有未提现佣金
- 申诉渠道：客服邮箱（MVP 不做在线申诉）

## 八、被推荐人视角

### 8.1 注册体验

被推荐人通过推荐链接访问 → Landing Page 与正常一致：
- 不显示任何"您被 XXX 邀请"类文案
- ref 参数静默存入 localStorage
- 注册流程与普通用户完全一致

### 8.2 被推荐人福利（可选，待确认）

> MVP 暂不设置被推荐人额外福利。V2 可考虑：
> - 注册即送 3 天免费会员
> - 首月额外折扣

## 九、验收标准

| 项目 | 标准 |
|------|------|
| 推荐链接 | 正确生成、可分享、可追踪 |
| 归属绑定 | 7 天内注册正确归属，过期不归属 |
| 佣金计算 | 金额准确，阶梯升级正确 |
| 冷却期 | 30 天冷却期逻辑正确 |
| 提现 | $10 以上可提现，记录可追溯 |
| 海报 | 二维码有效，推荐码正确 |
| 反作弊 | 自推和批量注册被正确拦截 |
| 退款扣回 | 被推荐人退款后佣金正确扣回 |
