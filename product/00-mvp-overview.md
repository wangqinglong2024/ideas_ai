## 一、项目背景与市场机会

### 1.1 市场痛点

东南亚中文学习市场存在三大痛点：

| 痛点 | 现状 | PlayLingo 的解法 |
|------|------|-----------------|
| **价格过高** | Duolingo $12.99/月、Speak $20/月，超出东南亚用户承受力 | **$1.99-2.99/月**，仅为竞品 1/6 |
| **缺乏本地化** | 竞品均未深度适配越南语界面和本地文化 | **越南语优先**，UI/剧情/支付全面本地化 |
| **学习枯燥** | 所有竞品本质是"课程+练习"，用户始终在"上课" | **2D 策略游戏**，用户主观体验是"玩游戏" |

### 1.2 目标市场

**首发市场：越南 🇻🇳**

| 指标 | 数据 |
|------|------|
| 人口 | ~1 亿，平均年龄 30 岁 |
| 电子学习市场 | 2026 年预计 **$46 亿**，增速东南亚最快 |
| 中文需求 | 中国工厂/台商大量设厂，学中文 = 涨薪 **50%+** |
| 移动渗透率 | >84% 智能手机，99.8% 4G 覆盖 |
| 竞争状况 | **蓝海**——无竞品同时满足"越南语界面 + 低价 + 游戏化 + 中文专精" |

**第二市场（月 7+）：印度尼西亚 🇮🇩** — 人口 ~2.8 亿，1120 万华裔（全球最大海外华人社群）

### 1.3 目标用户画像

| 用户群 | 年龄 | 学习动机 | 月付费意愿 |
|--------|------|---------|-----------|
| **年轻职场人**（核心） | 18-30 | 在中企/工厂升职加薪，刚需 | $2-5/月 |
| **大学生** | 18-24 | HSK 认证、留学申请 | $1-3/月 |
| **兴趣爱好者** | 15-35 | 追中国影视/音乐/网文 | $2-5/月 |
| **华裔家庭** | 家长 30-50 | 子女中文启蒙 | $3-10/月 |

## 二、竞争格局与差异化战略

### 2.1 竞品矩阵

| 维度 | Duolingo | HelloChinese | Speak | **PlayLingo** |
|------|----------|-------------|-------|--------------|
| **产品形态** | 课程闯关 App | 课程闯关 App | AI 对话 App | **2D 策略游戏** |
| **学习动力** | 打卡/排行榜 | 课程进度 | 口语需求 | **游戏荣誉感 + 领地征服** |
| **视觉品质** | 卡通风 | 简洁风 | 极简风 | **精致插画/独立游戏级** |
| **内容载体** | 选择题/翻译 | 选择题/手写 | AI 对话 | **策略关卡/过场动画/Boss 战** |
| **AI 角色** | 核心付费功能 | 辅助功能 | 核心产品 | **构建工具 + 按量增值** |
| **月价格** | $12.99-29.99 | $11.99 | $20 | **$1.99-2.99** |
| **越南本地化** | 部分 | 无 | 无 | **完整** |
| **被大模型替代风险** | 高 | 中 | 极高 | **低** |

### 2.2 核心差异化——"不是 AI 套壳"

| 维度 | AI 套壳模式（竞品） | PlayLingo |
|------|-------------------|-----------|
| **核心卖点** | "AI 帮你练口语" | "在游戏中自然学会中文" |
| **成本结构** | 用户越多 → AI 调用越多 → 成本线性增长 | 游戏内容一次制作 → 无限分发 → 边际成本趋零 |
| **被替代风险** | 大模型公司随时可做同样的事 | 游戏内容 + 荣誉体系 + 社交关系 = 不可替代 |
| **护城河** | 无（API 调用无壁垒） | 内容库 + 游戏体验 + 用户社区 |

> **核心逻辑**：AI 是我们的"工厂设备"，我们卖的是工厂生产出来的"游戏产品"。竞品把 AI 当产品卖，大模型升级一次就可能被替代；我们用 AI 生产内容，内容资产不可替代。

### 2.3 价格战优势

- **6.5 倍价差**：$1.99/月 vs Duolingo $12.99/月
- 在越南月均收入 $305 的市场，**$2/月 = 一杯奶茶**，$13/月 = 奢侈品
- 成本结构天然支持低价：AI 一次生成无限分发、1 人 + AI 研发效率抗衡 20 人团队、无人力包袱

## 六、运营与营销策略

### 6.1 四阶段运营节奏

| 阶段 | 时间 | 核心目标 | 关键动作 |
|------|------|---------|---------|
| **MVP 内测** | 月 1 | 200 种子用户，验证核心链路 | 功能上线、种子用户招募 |
| **冷启动** | 月 2-3 | 5,000 注册 / 500 付费 | TikTok 矩阵 + KOC 合作 + 佣金推广员 |
| **增长期** | 月 4-6 | 30,000 注册 / 3,000 付费 | 付费投放 + PvP 裂变（v1.1） + B2B |
| **爆发期** | 月 7-12 | 100,000 注册 / 10,000 付费 | 品牌化 + 印尼扩张 + 大型活动 |

### 6.2 三大营销引擎

**① TikTok 矩阵（CPI 目标 < $0.20）**
- 3 个账号矩阵：官方主号（1 条/天）+ 教学号（2 条/天）+ 娱乐号（1 条/天）
- 内容策略：挑战赛、趣味科普、用户成长故事、对比视频

**② KOL/KOC 合作网络**
- 冷启动期：20 个 KOC（粉丝 1K-50K），零付费 + 佣金分成
- 增长期：5-10 个 KOL（粉丝 50K-500K），$50-200/条 + 佣金分成
- 选择标准：内容相关（中国/学习/游戏）、互动率 >3%

**③ 佣金驱动增长飞轮**
```
用户玩游戏学中文 → 产生成就感（升级/解锁/排名）
→ 分享成就 + 推荐赚佣金（双重动力）
→ 新用户注册（首月折扣）
→ 新用户付费 → 推荐人赚佣金
→ 推广力度加大（佣金正循环）
→ 更多新用户 → 飞轮加速
```

### 6.3 病毒增长目标

| 指标 | 目标值 |
|------|--------|
| K-factor（病毒系数） | **>0.8**（加佣金激励后） |
| 邀请点击率 | >40% |
| 注册转化率 | >50% |
| 佣金用户占 DAU | >5% |
| 推广员月均推荐 | >5 人 |

### 6.4 投放漏斗预期

```
广告曝光 → 落地页  → 注册  → 完成新手引导 → 7日留存 → 付费
 100%   →  30%   → 50%  →    70%       →  25%   → 5%
```

### 9.3 支付合规

- **MoR 模式（Merchant of Record）**：通过 Stripe / Paddle / LemonSqueezy 平台处理全球税务、退款、发票
- 越南税务：增值税 5% + 企业所得税 5%（MoR 平台代扣代缴）

### 9.4 佣金合规

- **单层推荐**，符合越南竞争法（禁止非法多层分销）
- 佣金基于实际交易、KYC 验证、AML 月度提现上限 $500
- 法律文件（MVP 必需）：服务条款、隐私政策、订阅条款、佣金推广协议（均需越南语+英语双语版）

### 9.5 安全架构

| 措施 | 说明 |
|------|------|
| 认证 | JWT 7 天 + Refresh Token 30 天（滑动窗口） |
| 传输 | 全站 HTTPS |
| 数据库 | Supabase RLS 零信任 |
| 速率限制 | 登录 5 次/分、注册 3 次/分、API 100 次/分 |
| 反作弊 | 同 IP/设备指纹检测、服务端分数校验、低质量推荐过滤 |
