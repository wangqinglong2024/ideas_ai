# 🏭 工厂轨道 · 索引

> 服务对象：东南亚（越/泰/印尼/菲）在中资工厂工作的蓝领、技术员、品检、翻译、行政。
> 核心场景：车间 / QC / 仓储 / 食堂 / 班组会议 / 行政办公 / 工伤医务 / 培训。
> **岗位晋升制**：进阶以"能晋升到什么岗位"衡量，不强制 HSK 等级。

---

## 一、用户画像

| 维度 | 描述 |
|---|---|
| 年龄 | 18-45 |
| 国籍 | 越南（北部最多）、泰国、印尼、菲律宾 |
| 角色 | 普工、机台操作、QC、IPQC、OQC、班长、组长、技术员、翻译、人事 |
| 中文起点 | 多数 0 基础（厂里招聘门槛低，但晋升必须会中文）|
| 学习动机 | **保住工作 / 升职 / 加薪 / 跳槽到管理岗** |
| 学习时间 | 早会前 30min + 班后 1h，碎片化 |
| 付费能力 | 偏低（19-49 USD 包年是甜区） |

## 四、特色考核（工厂轨独有）

| 阶段 | 闯关任务 |
|:---:|---|
| 2 | 听 1 段早会录音，挑出 5 个工序词 |
| 4 | 看 1 张工艺单截图，标出 3 处关键参数 |
| 6 | 写 1 段早会发言稿（≥ 5 句），按规则评分 |
| 8 | 模拟客户验厂场景，写 3 句应对中文 |
| 10 | 翻译 1 段中越/中泰会议片段（10 句） |

通过 → 解锁岗位徽章 + 简历可挂牌（如"工厂中文 · QC 助理认证"）。
