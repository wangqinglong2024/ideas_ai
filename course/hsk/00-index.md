# 📘 HSK 轨道 · 索引

> 服务对象：参加 HSK 1-9 考试 / 留学 / 求职加分人群。

## 三、与其他 3 轨的关系

- HSK 轨是"通用语言能力"主干，词频曲线最规范
- 电商 / 工厂 / 日常 三轨的知识点会内嵌"基础字词重合度"标签（如"本节 12 句中 8 句词来自 HSK 2"），方便用户感知
- 但其他 3 轨**不**强制要求达到任何 HSK 等级才能进阶
- 需要 HSK 证书的用户考前 2 个月可切到 HSK 轨密集冲刺

---

## 四、特色考核（HSK 轨独有）

每个阶段综合考核之外：

- **HSK 真题模拟**：按 HSK 1-9 各级官方题型 1:1 还原
- **计时模拟**：按真实考试时长（HSK 1=40min ... HSK 6=140min）
- **模拟成绩 → 真考报名建议**：≥ 80% 推荐立即报名 / ≥ 70% 推荐复习 2 周 / < 70% 不建议
- **听力专项**：每阶段 100 段真题级听力（无 ASR，纯听辨）
