## 阶段考核

- **综合考核 90 题**：听 27 / 读 27 / 写 22 / 用 14；通过线 80%
- **特色闯关**：模拟询价 5 轮对话，写出正确的 5 句中文（按规则匹配评分）
- **解锁**：业务里程碑徽章「能完成询价 → 砍价 3 句话」
- **不通过**：24h 后无限重考；错题入 SRS（1d → 2d → 4d → 7d → 15d → 30d）
