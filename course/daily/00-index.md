# 🍜 日常轨道 · 索引

> 服务对象：18-35 泛人群（旅游、追剧、网购、社交、恋爱、文化爱好）。
> 定位：**破圈引流主力 + 留存粘性主力**（不是付费转化主力）。
> **生活场景制**：进阶以"能玩转什么生活场景"衡量，不强制 HSK 等级。

---

## 一、用户画像

| 维度 | 描述 |
|---|---|
| 年龄 | 18-35 |
| 国籍 | 越南 / 泰国 / 印尼 / 菲 |
| 角色 | 学生、年轻白领、自由职业、宝妈、追剧党、旅游党、网购党 |
| 中文起点 | 0 - 略懂 |
| 学习动机 | **兴趣**（追剧 / C-pop / 恋爱 / 旅游 / 文化好奇）|
| 学习时间 | 通勤、睡前、刷手机间隙 |
| 付费能力 | 较低（量大；适合广告 + 会员订阅模式）|

## 三、与其他 3 轨的关系

- 日常轨提供"文化背景层"
- 学电商 / 工厂 / HSK 时遇到文化点 → 跳转日常轨对应章节
- 反向：日常轨学到一定程度的用户可能产生"我也想做生意/找中企工作"动机 → 引导到电商/工厂轨
- 4 轨基础字词共用（拼音、HSK 1-3 高频字词），用户跨轨切换时已掌握部分自动跳过
