## 需求摘要

实现 G10 辩论擂台的 Phaser 3 前端游戏场景。包括辩论赛场主场景（正反方对阵辩论桌 + 主持人 + 观众席）、手牌展示与出牌交互（5 张手牌扇形排列、拖拽出牌到出牌区、卡牌翻转动画、攻防数值显示）、3 回合对战可视化（出牌结算动画 + 攻击力/防御力碰撞特效 + 说服力条动态增减 + 回合结果弹窗）、反驳卡机制视觉表现（反驳卡发光 + 指向对方卡牌的连线 + 碎裂抵消效果）、卡牌收藏画廊（按类型/稀有度分组展示 + 卡牌详情翻转查看）、卡组编辑器（15 张卡组构筑 + 拖入/移出交互 + 策略标签）、4 人淘汰赛对阵图。场景横屏 1920×1080，中国风法庭/书院辩论台风格。

### 核心交互流程

#### 出牌对战流程

```
1. DebateScene 加载 → 辩论赛场渲染 + 辩题显示 + 正反方标识
2. 服务端下发 5 张手牌 → HandDisplay 展示扇形手牌（卡牌翻转入场动画）
3. 玩家触摸/拖拽某张卡牌 → 卡牌放大预览（攻/防数值 + 特殊效果说明）
4. 拖拽到出牌区 → PlayZone 高亮确认区域 → 松手确认出牌
5. 出牌动画：手牌飞到出牌区正面朝下 → 等待对手出牌（"等待对手中..." 提示）
6. 双方出牌完毕 → 同时翻牌动画 → ClashAnimation：双方卡牌飞向中央碰撞
7. 数值结算特效：攻击力数字碰撞 → 胜方数字变大发光 → 说服力条增减动画
8. 反驳卡/特殊效果触发 → 相应特效叠加
9. RoundResultOverlay 弹出本回合结果
10. 3 回合结束 → 最终结算动画 → 跳转结算页面
```

#### 反驳卡交互

```
1. 玩家持有反驳卡（回合 2-3 才可用）→ 卡牌边缘闪烁紫色光效
2. 拖出反驳卡 → 屏幕显示对方上回合已出的牌（半透明回放）
3. 点击/拖到对方上回合卡牌上 → 连线 + 锁定确认
4. 结算时：反驳目标卡牌碎裂特效 → 对方说服力回退动画
```

### 关键动画

| 动画 | 实现 | 时长 |
|------|------|------|
| 手牌入场 | 5 张卡牌依次从底部弹入 + 扇形排列 | 800ms |
| 卡牌悬浮预览 | 卡牌放大 1.5× + 详情浮层 | 即时 |
| 出牌 | 卡牌从手牌飞到出牌区 + 翻转朝下 | 500ms |
| 双方翻牌 | 同时翻转 + 放大展示 | 600ms |
| 碰撞结算 | 双方卡牌飞向中央 + 火花特效 + 数值碰撞 | 1200ms |
| 说服力条变化 | 条形图 Tween 增减 + 数字跳动 | 800ms |
| 反驳碎裂 | 目标卡牌碎裂成碎片飞散 + 分数回退 | 1000ms |
| 逻辑陷阱 | 屏幕震动 + 对方卡牌变暗 + 数值减半特效 | 900ms |
| 情感牌 | 全屏粉色光晕 + 持续加成图标出现 | 700ms |
| 卡牌升级 | 卡牌发光 → 数值变化 → 新边框闪现 | 1200ms |
