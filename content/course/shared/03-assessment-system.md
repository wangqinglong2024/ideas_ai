# shared · 三级考核体系

> 4 轨道共用考核流程。**题型库与自动出题算法** 见 [`05-question-bank.md`](./05-question-bank.md)。
> 本文件只规定**何时考、考多少、过线多少**。

---

## 1. 三级总览

```
┌─ 第一级：节小测（12 题 / 节）       ── 即时反馈，错题入 SRS
├─ 第二级：章单元测（36 题 / 章）      ── 80% 通过
└─ 第三级：阶段综合考核（80-150 题 / 阶段） ── 80-85% 通过 + 颁发证书
```

所有题型从 [`05-question-bank.md`](./05-question-bank.md) 第 1 节的 10 类（Q1-Q10）+ 阶段 1 专用 3 类（P1-P3）中**随机抽取**，按知识点 `unit_type` 自动适配。

---

## 2. 第一级 · 节小测（12 题）

- 每节 12 个知识点 = 12 道题，1:1 对应（同一知识点不出 2 题）
- 抽题算法：[`05-question-bank.md`](./05-question-bank.md) 第 3 节
- **通过**：≥ 9/12（75%）
- 错题自动加入 SRS 复习队列；不限重考次数；立即可重考
- 单题作答即出对错 + 1 行 `key_point` 解释 + 正确答案

---

## 3. 第二级 · 章单元测（36 题）

- 抽题范围：本章 144 个知识点
- 题型分布：[`05-question-bank.md`](./05-question-bank.md) 第 4 节（听 30% / 读 30% / 写 25% / 用 15%）
- SRS 优先：节小测错过的知识点权重 ×3
- **通过**：总分 ≥ 80%；4 维度（听/读/写/用）单项 ≥ 60%
- 不通过 → 推送本章关键节点重学清单 → 24h 间隔后再考

---

## 4. 第三级 · 阶段综合考核

- 题数与四维度分布：[`05-question-bank.md`](./05-question-bank.md) 第 5 节
- 抽题范围：本阶段全 12 章 × 12 节 = 1728 知识点
- SRS 优先：章单元测错题权重 ×3
- **通过**：阶段 1-10 = 80%；阶段 11-12 = 85%；4 维度单项均 ≥ 60%
- 通过 → 数字证书（CEFR 等级 + 该轨里程碑徽章 + 阶段编号；HSK 轨另含 HSK 等级标识）
- 不通过 → 24h 间隔后无限重考

---

## 5. 各轨独有的特色考核

特色考核以**情景任务**形式追加，不引入新题型，仍由 Q1-Q10 拼装。每阶段综合考核之外另出 1 套（10-20 题），通过即解锁专属徽章。

| 轨道 | 特色考核 | 题型组合 | 解锁徽章 |
|---|---|---|---|
| 📘 HSK | HSK 真题模拟（按各级官方题型）| Q1+Q5+Q6+Q7+Q8 计时 | HSK X 模拟达标 |
| 🛒 电商 | 业务场景闯关（1688 砍价 / 直播听辨 / OEM 报价单）| Q1+Q5+Q6+Q7 多步串联 | 业务里程碑徽章 |
| 🏭 工厂 | 岗位胜任评估（早会发言 / 工艺单识读 / 改善提案）| Q5+Q6+Q7+Q8 任务串 | 岗位徽章（普工 → 管理）|
| 🍜 日常 | 生活场景挑战（自由行规划 / 点餐对白 / 影视片段听辨）| Q1+Q5+Q6+Q8 情景串 | 生活徽章（旅游通 / 餐厅通 等）|

---

## 6. 跨轨认证机制

- 4 轨道使用同一阶段编号；通过标准独立
- 通过 A 轨阶段 N → B 轨自动跳过共享的基础字词知识点；B 轨独有的场景章节仍需独立完成
- HSK 轨成绩单独标注 HSK 等级（阶段 3=HSK1 ... 阶段 12=HSK9）
- 其他 3 轨证书不含 HSK 等级，含「业务里程碑徽章」

---

## 7. 反作弊与公平

- 同 IP / 同设备短时多账号刷题 → 风控锁定
- 同一阶段综合考异常通过率（< 10 分钟交卷）→ 人工抽查
- 证书生成时记录设备指纹 + 答题轨迹哈希 → 可追溯

---

## 8. 关于「不做语音识别」的说明

本平台**不为用户跟读做 AI 评分**：
- 东南亚口音差异大，开源 ASR 准确率低
- 商用 ASR API 成本高，不适配低 ARPU 市场
- 第三方 ASR 不可控（封号 / 涨价 / 数据合规风险）

替代方案：
- **视频示范**：每个知识点 1 条 30s 视频，用户对照视频自跟
- **静默评估**：口语相关能力用 Q1（听辨）+ Q8（排序）+ Q7（填空）间接评估
- **录音回放**（可选）：App 内可录自己跟读 → 与原音并排播放，用户自评（无 AI 打分）
