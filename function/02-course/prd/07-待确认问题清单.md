## A · 范围与上线

| 编号 | 决策 | 说明 |
|---|---|---|
| A1 | 首发支持 `zh / en / vi / th / id` 内容字段 | 管理后台界面固定中文；多语言只用于学习内容与应用端 UI。 |
| A2 | 主题结构采用共享预备阶段 + 电商 / 工厂 / HSK / 日常四主题 | 共享内容在各主题复用，不重复维护。 |

## B · 内容生产与发布

| 编号 | 决策 | 说明 |
|---|---|---|
| B1 | 内容由运营线下生成和整理后导入 | 系统内不提供生成工作台、任务编排或提示词库。 |
| B2 | 内容状态只按发布可见性表达 | `is_published=true` 为已发布；`false` 为待发布，包含未发布和下架。 |
| B3 | 学员举报只进入 P-A-6 处理 | 采纳或忽略不自动改变内容发布状态，内容修改仍走对应编辑页。 |
