# 管理端生成工作台 · 已废弃说明

> 2025-11 起，本文件不再对应任何 F4 原型页面。
> F4 管理端页面固定为 P-A-1..P-A-9，其中 P-A-6 是学员举报处理，P-A-7 是媒资库，P-A-8 是考试中心管理，P-A-9 是全局搜索。

---

## 当前口径

- 系统内不再提供生成工作台、提示词管理、任务编排、试跑或重试页面。
- KP 与题目由运营在线下生成、整理后，通过 P-A-4 / P-A-5 的批量导入能力进入系统。
- 内容点检、编辑、发布分别在 P-A-4、P-A-5、P-A-3 完成。
- 学员举报统一进入 P-A-6，由运营查看详情、进入编辑抽屉调整、再采纳或忽略。
- 媒资管理在 P-A-7；考试中心分层配置在 P-A-8。

## 历史迁移

| 旧能力 | 当前落点 |
|---|---|
| 生成任务创建 / 监控 | 系统外部运营流程，不进入 F4 原型 |
| 提示词模板维护 | 系统外部运营资料，不进入管理端页面 |
| 生成结果入库 | P-A-4 / P-A-5 批量导入 |
| 内容修正 | P-A-4 / P-A-5 编辑抽屉 |
| 学员反馈修正闭环 | P-A-6 学员举报处理 |
