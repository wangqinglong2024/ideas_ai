## P-A-7 · 媒资库页

**路由**：`/admin/course/media`
**权限**：editor+

### 关键约定

- 媒资缓存只在**学员端**触发（[F2-Q11 封板](../F2-AI-接口规范/12-待确认问题清单.md)）；管理端只能查看 / 软删 / 替换，不能"主动生成"。
- 删除走"DB 软删 + 7 天 cron 真删 CDN 文件"（[F2-Q9 封板](../F2-AI-接口规范/12-待确认问题清单.md)）。
- voice_profile 是抽象标识，由后端 `MediaProvider` adapter 映射供应商（[F1-Q13 封板](../F1-AI-数据模型规范/12-待确认问题清单.md)）。

### 交互

| # | 操作 | 实现 |
|---|------|------|
| A7-01 | 进入 | 调 A56；默认按命中倒序 |

| A7-04 | 「删除」 | D-1 二次确认；[A20 DELETE /media/:id](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)；DB 软删 + 入 `course_media_assets_pending_purge` 队列；7 天后 cron 真删 CDN |

### 条件显示

| 条件 | 行为 |
|------|------|
| 媒资 `hit_count = 0` 超 30 天 | 标灰 + Tooltip「无人使用，建议清理」 |
| 已入待清理队列 | 状态标「⏳ 7 天后清理」+ 可撤销 |
| 关联 KP / Q 已删除 | 标「🔗 孤立」+ 提示可清理 |

## P-A-8 · 考试中心管理页

**路由**：`/admin/course/exams`
**权限**：editor+

### 关键约定

- 考试题目共用题库，不维护独立考试题池；题目通过可抽层级标记供节测 / 章测 / 阶段考 / HSK 模考抽取。
- 页面不展示旧式「模板表格」。入口先选主题，再进入阶段列表、章列表、节列表。
- 阶段节点配置阶段考；章节点配置章测；节节点配置节测。
- HSK 模考配置仅在 HSK 主题下显示。
- 试抽仅预览，不创建学员 attempt，不写入考试快照。
- 阶段考 / 章测 / 节测保存后仍为待发布配置，必须点击发布后应用端才可见；已发布配置可单独下架。

### D-9 · 配置章测 Drawer（720px）

```
┌─ 配置章测 ─────────────────────────────────[×]┐
│ 章测名称（内容多语言）                         │
│ 中文 / 英文 / 越南语 / 泰语 / 印尼语             │
│ 题量 20   时长 25   及格分 60%   解锁条件         │
│ 抽取范围：本章下属节关联题目 / SH 共享题           │
│ 题型分布：听音选词、选声调、完形填空、配对等        │
│ ───────────────────────────────────────       │
│ 试抽预览 [重抽一份]                             │
│ ✓ 已生成 20 题预览，下面展示前 8 题               │
│ 第 1 题 · 听音选词 · 请选择听到的词 → 老板         │
│ 第 2 题 · 选声调 · 「谢」的声调？ → 第 4 声        │
│ ...                                             │
│ [取消] [保存] [下架] [发布]                      │
└────────────────────────────────────────────────┘
```

| 字段 | 校验 |
|------|------|
| `title_i18n` | 5 key；zh / en 必填 |
| `total_questions` | 与题型分布换算后的题量一致，保存时服务端兜底校验 |
| `duration_min` | ≥ 1，≤ 240 |
| `pass_score` | 0–100 |
| `blueprint.scope` | 阶段考至少覆盖 1 个阶段；章测至少覆盖 1 个章 |
| `blueprint.q_type_ratio` | 之和 = 100% |

| 操作 | 实现 |
|------|------|
| 保存 | 新建 [A21 POST /exams](../F2-AI-接口规范/08-管理端-媒资与考试中心.md) / 编辑 [A21 PATCH /exams/:id](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)；不改变发布状态，未发布配置显示「待发布」 |
| 试抽 | [A22 POST /exams/:id:preview](../F2-AI-接口规范/08-管理端-媒资与考试中心.md)；服务端返回抽题摘要 + 具体题目预览（不入库 / 不影响学员） |
| 发布 | [A16 POST /exam/:id:publish](../F2-AI-接口规范/07-管理端-发布与举报处理.md)；最近一次试抽必须成功且无阻断 warning 才能发布 |
| 下架 | [A16 POST /exam/:id:unpublish](../F2-AI-接口规范/07-管理端-发布与举报处理.md)；不影响已有 attempt snapshot |

### 条件显示

| 条件 | 行为 |
|------|------|
| 可抽题不足 | 试抽区域显示黄色 warning，保存可继续，发布 disabled |
| 题目被单独下架 | 后续试抽 / 发卷自动排除；满分 100 按当前有效题数动态均分 |
| 阶段 / 章未发布 | 配置可编辑，学员端不展示该考试 |
| HSK 主题 | 额外显示 HSK 模考配置入口；其它主题隐藏 |
| 已有学员 attempt | 编辑后提示「不影响已有答卷快照」 |
