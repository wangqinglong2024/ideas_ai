### 元素规格

| 元素 | 组件 | 说明 |
|------|------|------|
| 阶段折叠 | `Collapse` | 默认全部折叠；点击展开后显示该阶段的章列表 |
| 阶段操作 | 「编辑」「发布 / 下架」「调序」「删」 | 发布 / 下架为行内按钮；保存阶段信息不改变发布状态 |
| 章列表表格 | `Table` | 列：序号 / 章名称（含 i18n.zh）/ 节数（已发布/总）/ KP 数 / 状态 / 操作 |
| 行操作 | `ButtonGroup` | 编辑章基本信息 / 查看节列表 / 发布或下架 / 删除；不使用状态下拉代替发布动作 |
| 章内节列表 | `Table` | 编辑章时直接展示本章节列表，列：序号 / code / 节名 / KP 数 / 题目数 / 状态 / 操作 |
| 状态徽标 | `Badge` | 已发布 / 待发布 |
| 批量操作 | `Dropdown` | 批量发布 / 批量下架 / 批量删除（仅勾选行）|

### 条件显示

| 条件 | 说明 |
|------|------|
| `stage.is_published = false` | 阶段在学员端不可见；子级发布状态不自动改变 |
| `chapter.is_published = false` | 章及章下节在学员端不可见；章下架默认同步下架全部节 |
| `chapter.lesson_count < 6` | 章名称右侧黄点 + Tooltip「未达 6 节基线」 |
| 共享预备阶段 | 唯一允许 4 章；不显示「未达 6 节」警告 |
| 学员有进度的章 | 删除按钮 disabled + Tooltip「有学员进度，不可删除」 |

## 全局：i18n 编辑器组件

| 字段 | 校验 |
|------|------|
| zh | 必填（学习目标显示用 + UI 主语言） |
| en | 必填（回退） |
| vi / th / id | 可空字符串，但 key 必须存在 |

> i18n 编辑器统一封装在 `packages/ui-kit/i18n-editor`，所有管理端表单引用。
