F3 规范

课程学习引擎 · 静态原型

依据 F3 页面交互规范 与 F1 数据模型、 F2 接口 生成。 视觉风格参考 system/apps/web-app/src/pages/china/（小红书红 + 毛玻璃 + 黑白）。

每个原型页面顶部均带 桌面 / 移动 设备切换 与 浅色 / 暗色 主题切换；按钮交互（新建 / 删除 / 编辑 / 提交 等）均带演示弹窗，跳转链接以 toast 提示，便于走查 IA 与每一处操作反馈。
