## A20 · 媒资库

### A20.1 列表

**方法**：GET
**路径**：`/admin/v1/course/media`
**权限**：`readonly+`

**请求参数**：

| 参数 | 说明 |
|------|------|
| `kind` | `audio/image/video/svg/lottie` |
| `source` | `tts/generated_image/human_recorded/uploaded/stock_library` |
| `voice_profile` | 仅 audio |
| `q` | 模糊匹配 url 或关联 KP code |
| `unused_only` | bool，只看未被引用 |
| `page / page_size / sort` | — |

**响应 200**：

```json
{
  "code": 0,
  "data": {
    "items": [
      {
        "id": "<uuid>",
        "kind": "audio",
        "url": "https://cdn.example.com/audio/abc.mp3",
        "hash": "3c5e...",
        "source": "tts",
        "voice_profile": "xiaoxiao_zh",
        "meta": { "duration_ms": 1820, "format": "mp3" },
        "ref_count": 5,
        "created_at": "..."
      }
    ],
    "pagination": {...}
  }
}
```

### A20.2 上传（去重）

**方法**：POST
**路径**：`/admin/v1/course/media`
**权限**：`content_admin+`
**Content-Type**：`multipart/form-data`

**form 字段**：

| 字段 | 必填 | 说明 |
|------|------|------|
| `file` | ✅ | 文件二进制 |
| `kind` | ✅ | `audio/image/...` |
| `source` | ✅ | `uploaded/stock_library`（其他来源由系统内部写） |
| `voice_profile` | ❌ | 仅 audio |

**业务校验**：
- 后端计算 sha256 → 命中 `course_media_assets.hash` 则直接返回旧 id（不重复存储）
- 文件大小：image ≤ 5MB，audio ≤ 10MB，video ≤ 50MB
- MIME 校验

**响应 200**：

```json
{
  "code": 0,
  "data": {
    "id": "<uuid>",
    "url": "https://cdn.example.com/...",
    "hash": "3c5e...",
    "duplicate": false   // true=命中已有
  }
}
```

### A20.3 删除

**方法**：DELETE
**路径**：`/admin/v1/course/media/:id`
**权限**：`super`

**业务校验**：仍被引用 → `409 COURSE_MEDIA_STILL_REFERENCED`，附带引用清单（前端展示）。

**响应 200**：

```json
{ "code": 0, "data": { "deleted": true } }
```

## A21 · 考试中心配置 CRUD

### A21.1 列表

**方法**：GET
**路径**：`/admin/v1/course/exams`
**权限**：`readonly+`

**请求参数**：`scope_type / track / is_published / q / page / sort`

**响应 200**：

```json
{
  "code": 0,
  "data": {
    "items": [
      {
        "id": "<uuid>",
        "title_zh": "EC Stage 1 阶段考",
        "scope_type": "stage_exam",
        "scope_ref_id": "<uuid>",
        "scope_ref_code": "ec-stage-1",
        "question_count": 30,
        "time_limit_sec": 1800,
        "pass_score": 80,
        "max_attempts": 1,
        "is_published": true
      }
    ],
    "pagination": {...}
  }
}
```

### A21.2 新建

**方法**：POST
**路径**：`/admin/v1/course/exams`
**权限**：`content_admin+`

**请求体**：

```json
{
  "title_zh": "EC Stage 1 阶段考",
  "title_i18n": { "vi":"...","th":"...","id":"...","en":"EC Stage 1 Exam" },
  "scope_type": "stage_exam",
  "scope_ref_id": "<uuid>",
  "blueprint": {
    "kp_distribution": [
      { "kp_type": "word",     "weight": 0.5 },
      { "kp_type": "phrase",   "weight": 0.2 },
      { "kp_type": "grammar",  "weight": 0.2 },
      { "kp_type": "sentence", "weight": 0.1 }
    ],
    "q_type_distribution": [
      { "q_type": "mcq_meaning",      "weight": 0.3 },
      { "q_type": "listen_pick",      "weight": 0.2 },
      { "q_type": "fill_blank_choice","weight": 0.2 },
      { "q_type": "type_zh_ime",      "weight": 0.15 },
      { "q_type": "match_pairs",      "weight": 0.15 }
    ],
    "kp_pool_filter": {
      "stage_no_max": 1,
      "must_be_published": true
    },
    "shuffle_questions": true,
    "shuffle_options": true
  },
  "question_count": 30,
  "time_limit_sec": 1800,
  "pass_score": 80,
  "max_attempts": 1
}
```

**业务校验**：
- `kp_distribution` 权重和 = 1.0（误差 ±0.01）
- `q_type_distribution` 权重和 = 1.0
- `scope_ref_id` 必须存在且 `scope_type` 一致
- `kp_pool_filter` 必须能筛出 ≥ `question_count` × 5 的候选（防抽不到）

**响应 201**：返回新建对象，`is_published=false`，管理端显示「待发布」。

### A21.3 编辑

**方法**：PATCH
**路径**：`/admin/v1/course/exams/:id`
**权限**：`content_admin+`

**业务校验**：编辑直接 `version+=1`；已发布的 exam 改 blueprint 不影响**已存在的 attempts**（`snapshot.blueprint_used` 为副本）。

### A21.4 软删

**方法**：DELETE
**路径**：`/admin/v1/course/exams/:id`
**权限**：`super`

**业务校验**：有 in_progress attempts → `409 COURSE_EXAM_HAS_ACTIVE_ATTEMPTS`

### A22 · 试抽题（管理端预览）

**方法**：POST
**路径**：`/admin/v1/course/exams/:id:preview`
**权限**：`content_admin+`

**响应 200**：按 blueprint 抽一次（不写 attempt），返回抽题摘要和题目卡片，便于运营检查配置是否满足。

## A23 · 全局搜索

**方法**：GET
**路径**：`/admin/v1/course/search`
**权限**：`readonly+`

**请求参数**：

| 参数 | 必填 | 说明 |
|------|------|------|
| `q` | ✅ | 关键词，长度 1–60 |
| `types` | ❌ | csv，默认 `kp,question,lesson`；可裁剪 |
| `track` | ❌ | 限定主题 |
| `limit_per_type` | ❌ | 默认 10，上限 30 |

**响应 200**：

```json
{
  "code": 0,
  "data": {
    "kps": [...],         // {kp_id, kp_code, title_zh, kp_type, is_published}
    "questions": [...],   // {question_id, q_code, stem_zh, q_type, is_published}
    "lessons": [...]      // {lesson_id, code, title_zh, is_published}
  }
}
```

> 实现：PG GIN trigram 索引（`pg_trgm`）+ 多列 ILIKE。
> 限流：用户 30/分。

## A24 · 统计大屏

| 路径 | 维度 |
|------|------|
| `GET /admin/v1/course/stats/overview` | 全平台总览（learners DAU、KP/Q 总数、发布率、举报待处理数） |
| `GET /admin/v1/course/stats/track/:track` | 单主题（学员数、已通过 lesson 分布、错题排行） |
| `GET /admin/v1/course/stats/imports?from=&to=` | 导入批次质量与失败趋势 |

权限：`readonly+`

**示例响应（overview）**：

```json
{
  "code": 0,
  "data": {
    "kp_total": 11248,
    "q_total": 56412,
    "lesson_total": 884,
    "active_learners_today": 1532,
    "answers_today": 42168,
    "accuracy_today": 0.81,
    "reports_open": 12,
    "import_batches_today": { "imported": 4, "failed": 1 }
  }
}
```

数据源：`mv_course_track_stats` + `mv_course_user_daily` + 实时聚合。
