# 横切：内部任务接口

> 仅保留课程运行所需的内部任务接口；内容生产相关任务不在系统内编排。

---

## I1 · 健康检查

**方法**：GET
**路径**：`/api/v1/course/health`
**权限**：公开

**响应 200**：

```json
{ "status":"ok", "schema":"zhiyu", "version":"v1", "ts":"2026-04-15T10:00:00+08:00" }
```

---

## I2 · 考试限时到期任务

**方法**：POST
**路径**：`/internal/v1/course/exam-attempts:expire`
**权限**：`service-role`（仅 cron 调用）

**业务**：
- `UPDATE course_user_exam_attempts SET state='expired', score=<计算>, finished_at=now() WHERE state='in_progress' AND now() > expires_at`。
- 返回处理数量。

**响应 200**：

```json
{ "code":0, "data": { "expired_count": 12 } }
```

调度：每分钟（`scripts/cron/expire-exam-attempts.ts`）。

---

## I3 · 媒资清理任务

**方法**：POST
**路径**：`/internal/v1/course/media:purge-pending`
**权限**：`service-role`（仅 cron 调用）

**业务**：
- 清理已进入待清理队列且超过保留窗口的 CDN 文件。
- 只删除未被内容引用的媒资。

**响应 200**：

```json
{ "code":0, "data": { "purged_count": 8 } }
```
