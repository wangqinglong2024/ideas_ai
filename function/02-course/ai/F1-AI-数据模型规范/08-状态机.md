## 1. 内容点检状态机（适用：KP / Question / Lesson / Exam）

来源：[temp/04 §4.4 / §4.7](../../../temp/04-管理端模块设计.md) + [temp/06 §6.4 流程 C](../../../temp/06-关键业务流程.md)

```
        ┌──────────┐
     │  draft   │ ◄── 入口（批量导入 / 人工新建）
        └────┬─────┘
             │ submit_review
             ▼
        ┌────────────┐
        │ reviewing  │
        └─┬────┬───┬─┘
          │    │   │
   approve│    │   │reject
          │    │   │
          │    │   ▼
          │    │  ┌──────────┐
          │    │  │ rejected │ ──┐ edit
          │    │  └──────────┘   │
          │    │                 │
          │    └─edit─►◄─────────┘ (回到 draft)
          ▼
        ┌──────────┐                ┌──────────┐
        │ approved │ ──archive────► │ archived │
        └────┬─────┘                └──────────┘
             │ publish (set is_published=true)
             ▼
        【对学员可见】
```

| 当前状态 | 允许动作 | 角色 | 目标状态 | 副作用 |
|---------|---------|------|---------|-------|
| `draft` | `submit_review` | content_admin | `reviewing` | — |
| `draft` | `delete` | content_admin/super | （物理：DELETE / 软删 deleted_at=now()） | 写 review_log |
| `reviewing` | `approve` | reviewer/super | `approved` | 写 review_log；自动尝试 publish |
| `reviewing` | `reject` | reviewer/super | `rejected` | 写 review_log（带 reason） |
| `reviewing` | `edit` | reviewer | `draft`（小改）/ `approved`（一键改正） | 写 review_log（带 diff） |
| `rejected` | `edit` | content_admin | `draft` | — |
| `approved` | `unpublish` | super | `approved` + `is_published=false` | 学员立即不可见 |
| `approved` | `publish` | content_admin/super | `approved` + `is_published=true` | 学员可见 |
| `approved` | `archive` | super | `archived` | 通常用于版本切换；学员不可见 |
| `archived` | （无） | — | — | 终态；30 天后由 cron 物理删 |

实现：RPC `fn_content_transition(p_target_type, p_target_id, p_action, p_reason, p_diff)`，事务内统一写入 `course_content_review_log`。

## 2. 导入批次状态机

来源：[temp/06 §6.3 流程 B](../../../temp/06-关键业务流程.md)

```
   ┌────────┐
   │ draft  │
   └───┬────┘
        │ validate
        ▼
┌────────────┐
│ validating │
└────┬───────┘
      │ success
      ▼
 ┌────────┐      import       ┌──────────┐
 │ ready  │ ────────────────► │ imported │
 └───┬────┘                   └──────────┘
      │ fail / cancel
      ▼
┌──────────┐
│ failed   │ / cancelled
└──────────┘
```

约束：
- `draft → validating`：上传后异步校验列名、必填、枚举、重复编码与 JSON schema。
- `validating → ready`：写入 `row_count / success_count / validation_errors=[]`。
- `ready → imported`：事务内插入 KP / 题目 / 媒资，并把 `source_batch_id` 写入内容表。
- `validating / ready → failed`：保留 `validation_errors`，不写入内容主表。
- `draft / ready → cancelled`：仅管理员可取消；取消后不可导入。

## 3. 节级进度状态机（`course_user_progress.status`）

来源：[temp/05 §5.4](../../../temp/05-用户端模块设计.md) + [temp/06 §6.7](../../../temp/06-关键业务流程.md)

```
   ┌────────┐
   │ locked │ ◄── 默认（前置节未通过）
   └───┬────┘
       │ unlock_predecessor_passed
       ▼
   ┌──────────┐
   │ unlocked │
   └────┬─────┘
        │ first_enter
        ▼
   ┌─────────────┐
   │ in_progress │
   └─┬─────────┬─┘
     │         │
quiz_pass     quiz_fail (允许重试，状态保持)
     │         │
     ▼         └────► 仍 in_progress
   ┌────────┐
   │ passed │ ─── (终态，可复习不改状态)
   └────────┘
```

| 触发 | 当前 | 目标 | 写入字段 |
|------|------|------|---------|
| 学员首次进入节 | unlocked | in_progress | `started_at=now()`, `status='in_progress'` |
| 通过节末小测 | in_progress | passed | `passed_at=now()`, `best_score=max(...)`, `status='passed'` |
| 节末小测未过 | in_progress | in_progress | `attempts += 1`, `best_score=max(...)` |
| 上一节通过（触发） | locked | unlocked | `status='unlocked'` |
| 节内退出 | in_progress | in_progress | `last_kp_id`, `last_position`（断点续学） |

实现：RPC `fn_lesson_pass(p_user_id, p_lesson_id, p_score)` + 触发器 `tg_after_lesson_pass_unlock_next`。

> ⚠️ 节末小测是否强制（不强制 → unlocked → 直接进下一节），见 [12-待确认问题清单.md Q4](./12-待确认问题清单.md)。

## 4. 考试答卷状态机（`course_user_exam_attempts.state`）

来源：[temp/06 §6.9 流程 H](../../../temp/06-关键业务流程.md)

```
   ┌─────────────┐
   │ in_progress │ ◄── start_exam (RPC 抽题)
   └─┬────┬────┬─┘
     │    │    │
submit│ exit│  │ time_up (cron)
     │    │    │
     ▼    ▼    ▼
  ┌──────────┐ ┌───────────┐ ┌─────────┐
  │submitted │ │ abandoned │ │ expired │
  └──────────┘ └───────────┘ └─────────┘
   学员主动提交  退出弹窗确认放弃   超时
   计算总分      score=0          按已答计分
```

| 当前 | 触发 | 目标 | 写入 |
|------|------|------|-----|
| in_progress | submit | submitted | score, passed, finished_at=now() |
| in_progress | exit_confirm | abandoned | score=0, passed=false, finished_at=now() |
| in_progress | cron(now > expires_at) | expired | score=按已答计算, passed, finished_at=now() |
| 任意终态 | — | — | 不可改 |

cron 任务：`scripts/cron/expire-exam-attempts.ts`，每分钟扫一次。
