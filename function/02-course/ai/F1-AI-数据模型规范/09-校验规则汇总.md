# 五 · 校验规则汇总

> 前后端共享的字段校验，前端 zod schema 在 [shared-schemas/src/course/](../../../system/packages/shared-schemas/src/course/)（待生成）；
> 后端在 RPC / API 层调用同一 schema 二次校验。
> 错误码统一前缀 `COURSE_*`。

---

## 1. 全局规则

| 规则 | 适用 | 说明 |
|------|------|------|
| 字符串去首尾空白 | 所有 text 字段 | `.trim()` |
| 长度上限 | 所有 text 字段 | 见各表 §字段定义；超出 → `COURSE_FIELD_TOO_LONG` |
| `_i18n` jsonb | 所有多语言内容列 | key 集合必须为 `{zh,en,vi,th,id}`；编辑草稿态可暂缺翻译，发布 / 展示前必须补齐；应用端只把 `zh` 作为兜底防线，不作为缺译发布策略 |
| 软删字段 | `deleted_at` | 一旦非空，所有学员侧查询自动过滤 |
| 时间 | `*_at` | 必须 `timestamptz`；前端用 ISO 8601 |

## 2. 编码字段格式

| 字段 | 正则 | 错误码 |
|------|------|-------|
| `course_tracks.code` | `^(share\|ec\|fc\|hsk\|dl)$` | `COURSE_INVALID_TRACK_CODE` |
| `course_lessons.code` | `^[a-z]+-\d+-\d+-\d+$` | `COURSE_INVALID_LESSON_CODE` |
| `course_knowledge_points.kp_code` | `^kp_(share\|ec\|fc\|hsk\|dl)_(pinyin\|hanzi\|word\|phrase\|grammar\|sentence\|dialog)_\d{6}$` | `COURSE_INVALID_KP_CODE` |
| `course_questions.q_code` | `^q_(share\|ec\|fc\|hsk\|dl)_\d{8}$` | `COURSE_INVALID_Q_CODE` |
| `course_ai_prompts.prompt_code` | `^prompt_[a-z0-9_]+_v\d+$` | `COURSE_INVALID_PROMPT_CODE` |
| `course_media_assets.hash` | `^[a-f0-9]{64}$` | `COURSE_INVALID_MEDIA_HASH` |

## 5. 跨字段一致性

| 规则 | 错误码 |
|------|-------|
| `course_user_progress.status='passed'` ⇒ `passed_at IS NOT NULL` | `COURSE_PASSED_AT_REQUIRED` |
| `course_user_exam_attempts.state ≠ 'in_progress'` ⇒ `finished_at IS NOT NULL` | `COURSE_FINISHED_AT_REQUIRED` |
| `course_ai_generation_jobs.status ∈ {done,failed,partial}` ⇒ `finished_at IS NOT NULL` | `COURSE_AI_FINISHED_AT_REQUIRED` |
| `course_content_review_log.action='report'` ⇔ `report_type IS NOT NULL` | `COURSE_REPORT_TYPE_REQUIRED` |
| `course_questions.payload` 必须符合 `q_type` 对应的 schema | `COURSE_PAYLOAD_SCHEMA_MISMATCH` |
| `course_knowledge_points.content` 必须符合 `kp_type` 对应的 schema | `COURSE_KP_CONTENT_SCHEMA_MISMATCH` |
| `course_lessons.code` 三段数字必须与所属 chapter/stage/track 一致 | `COURSE_LESSON_CODE_MISMATCH` |
| `course_exams.blueprint.kp_ids` 必须与 `scope_ref_id` 范围内的 KP 一致 | `COURSE_EXAM_BLUEPRINT_OUT_OF_SCOPE` |
| `course_user_srs` 仅 `context_type ≠ 'practice'` 才更新 | （应用层） |

## 6. 业务规则（RPC 层）

| 规则 | 触发位置 | 错误码 |
|------|---------|-------|
| 学员未学完前置 stage 不能进入下一 stage | `fn_enter_lesson` | `COURSE_PREREQUISITE_NOT_MET` |
| 学员未通过节末小测不能进下一节（取决于 Q4） | `fn_enter_lesson` | `COURSE_LESSON_QUIZ_REQUIRED` |
| 阶段考过了不能重考 | `fn_start_exam` | `COURSE_STAGE_EXAM_ALREADY_PASSED` |
| 学员一天 SRS 复习上限（默认 50 张） | `fn_get_srs_queue` | `COURSE_SRS_DAILY_LIMIT_EXCEEDED` |
| AI Job 累计 cost ≥ budget 时拒绝继续 | worker | `COURSE_AI_BUDGET_EXCEEDED` |
| AI Job 同 idempotency_key 重复创建 | `fn_create_ai_job` | `COURSE_AI_JOB_DUPLICATE` |
| 删除 KP 但仍被节引用 | `fn_delete_kp` | `COURSE_KP_STILL_REFERENCED` |
| 删除媒资但仍被引用 | `fn_delete_media` | `COURSE_MEDIA_STILL_REFERENCED` |

## 7. RLS 校验（运行时拒绝）

| 操作 | 规则 | 错误码 |
|------|------|-------|
| 学员查询 KP/Q/Lesson | 必须 `status='approved' AND is_published=true AND deleted_at IS NULL` | `COURSE_NOT_VISIBLE` |
| 学员写 user_* 表 | 必须 `user_id = auth.uid()` | `COURSE_FORBIDDEN_OTHER_USER` |
| 学员篡改 user_answers / user_exam_attempts | UPDATE/DELETE 全禁 | `COURSE_IMMUTABLE_RECORD` |
| 管理端写内容表 | 必须 `service_role` 或经管理端鉴权 RPC | `COURSE_ADMIN_ONLY` |

---

## 8. 错误码命名规范

`COURSE_<DOMAIN>_<DETAIL>`：
- `COURSE_FIELD_*`：字段格式/长度/范围
- `COURSE_INVALID_*`：枚举不匹配
- `COURSE_*_REQUIRED`：必填/一致性
- `COURSE_*_NOT_FOUND`：实体不存在
- `COURSE_*_FORBIDDEN`：权限拒绝
- `COURSE_*_DUPLICATE`：唯一约束
- `COURSE_*_LIMIT_EXCEEDED`：业务上限

完整错误码登记见 [F2 接口规范 §错误码](../F2-AI-接口规范/00-index.md)（待生成时补全）。
