## 通用规则

| 规则 | 实现 |
|------|------|
| **必填星标** | 表单字段 label 前加红色 `*`；保存时校验缺失项；具体字段红色边框 + 行下错误提示 |
| **保存即校验** | A4/A5/A11/A12 前端均强制校验所有 5 语言 + 拼音；失败不发请求（PM 答 F2-Q5） |
| **离开提示** | 页面/Drawer 持有 `dirty` 状态 → `useUnsavedChangesGuard` 拦截路由切换 + `beforeunload` |
| **二次确认** | 删除 / 下架 / 发布 / 重排 一律弹 D-5 |
| **多端冲突** | 保存成功后比较 `updated_by + updated_at`，冲突时附加 toast 但不阻断 |
| **接口节流** | 搜索 300ms 防抖；进度上报 1s 防抖 |
| **限流提示** | 应用端 C4 命中 429 → toast「操作过于频繁，请稍后再试」 |
