## 元素规格

| 元素 | 组件类型 | 规格 |
|------|---------|------|
| 顶部类目标题 + 介绍 | `<header>` | 显示当前 UI 语言下的 `name_i18n` / `description_i18n` |
| 搜索框 | `Input.Search` | 占位符按 UI 语言翻译；防抖 300ms；支持中文与当前 UI 语言两个字段（PM §三）|
| 文章卡片 | `Card` | 三行：拼音、中文标题、当前 UI 语言标题；右下显示 `sentence_count` + `published_at` |
| 分页器 | `Pagination` | `page_size=20` |

> 应用端搜索逻辑：调用 C2 时附 `q` 参数；后端 OR 匹配 `title_i18n.zh ILIKE %q%` 与 `title_i18n[currentLang] ILIKE %q%` 与 `sentences.content_zh ILIKE %q%` 与 `sentences.content_<currentLang> ILIKE %q%`。**仅命中"当前可见的两种语言"**（PM §三）。本接口的 `q` 行为请在 F2 C2 接口里增加（标记为「F3 引出 → F2 增补」，详见 [11-待确认问题清单.md](./11-待确认问题清单.md) Q1）。

## 数据来源

- 挂载与翻页：[C2 GET /api/v1/china/categories/:code/articles](../F2-AI-接口规范/01-应用端-类目与文章浏览.md#c2--列出某类目下已发布文章) `?page&page_size=20&sort=-published_at&q=...`
- 顶部类目信息可复用 P-C-1 的 SWR 缓存（找到对应 `code` 的项），无需额外请求。

## 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| C2-01 | 进入页面 | 校验权限（若 code ≥ '04' 且未登录 → 跳 `/login?redirect=...`）；调 C2 拉第 1 页 |
| C2-02 | 输入搜索关键词 | 防抖 300ms 后触发，`page=1` 重置 |
| C2-03 | 点击分页 | 仅更新 `page`，URL query 同步 `?page=2` 便于刷新 |
| C2-04 | 点击文章卡片 | `router.push('/china/articles/' + code)` |
| C2-05 | 接口 401 | 自动跳 `/login?redirect=` 当前 URL |
| C2-06 | 接口 404 | 居中显示「类目不存在」+ 返回按钮 |
