## 元素规格

| 元素 | 组件类型 | 规格 | 可见角色 |
|------|---------|------|---------|
| 顶部标题 | `<h1>` | "发现中国"（按当前 UI 语言翻译） | 全部 |
| 类目卡片 × 12 | `Card` | 宽度 自适应 3 列 / 移动端 2 列；高度 140px | 全部 |
| 卡片标题 | `<h3>` | `name_i18n[currentLang]`，回退 `en` → `zh` | 全部 |
| 卡片说明 | `<p>` | `description_i18n[currentLang]`，2 行省略 | 全部 |
| 锁图标 | `Icon(Lock)` | 卡片右上角，仅 `code >= '04'` 且未登录时显示 | 未登录用户 |
| 语言切换 | `LangPicker` | 全局组件，G2 已定义 | 全部 |

## 数据来源（接口）

- 挂载时调用 [C1 GET /api/v1/china/categories](../F2-AI-接口规范/01-应用端-类目与文章浏览.md#c1--列出-12-类目)。
- 响应可强缓存（`Cache-Control: public, max-age=3600`），SWR 在客户端再缓存 1 小时。

---

## 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| C1-01 | 进入页面 | 调 C1，骨架屏 → 渲染 12 个卡片，按 `sort_order` 排序 |
| C1-02 | 点击类目 `01..03` 卡片 | `router.push('/china/categories/' + code)` |
| C1-03 | 点击类目 `04..12` 卡片（未登录） | 弹 D-8「登录提示 Modal」，确认后 `router.push('/login?redirect=/china/categories/' + code)` |
| C1-04 | 点击类目 `04..12` 卡片（已登录） | `router.push('/china/categories/' + code)` |
| C1-05 | 切换 UI 语言 | 全局事件，重渲染 `name_i18n` / `description_i18n` 对应键，无需重新请求 |

---

## 条件显示规则

| 条件 | 显示 | 隐藏 |
|------|------|------|
| 未登录 + 类目 code ≥ '04' | 锁图标 + 点击弹 D-8 | — |
| 已登录 或 类目 code ≤ '03' | 普通可点卡片 | 锁图标 |

---

## 空态 / 异常

| 场景 | 处理 |
|------|------|
| C1 接口失败 | 居中显示「内容加载失败 + [重试] 按钮」 |
| 数据为空（不应该发生） | 显示「暂无类目」 |
| 网络断开 | G2 全局离线提示条 |
