## 表格列定义

| 列标题 | 数据字段（F1） | 宽度 | 排序 | 渲染 |
|--------|--------------|------|------|------|
| Code | `china_articles.code` | 150 | ❌ | 等宽字体；点击复制 + Toast「已复制」 |
| 中文标题 | `title_i18n.zh` | flex | ❌ | 单行省略；hover tooltip 显示拼音 |
| 状态 | `status` | 100 | ❌ | `draft`=灰色 Tag「草稿」；`published`=绿色 Tag「已发布」 |
| 句数 | `sentence_count` | 80 | ❌ | 数字 |
| 更新时间 | `updated_at` | 160 | ✅ 默认倒序 | `MM-DD HH:mm`，hover 显示完整 ISO |
| 操作 | — | 200 | ❌ | 「编辑」「发布/下架」「删除」三个按钮组 |

## 交互行为

| # | 用户操作 | 技术实现 |
|---|---------|---------|
| A2-01 | 进入页面 | 调 A2 拉第 1 页 |
| A2-02 | 输入搜索 | 300ms 防抖；调 A2 带 `q`，page=1 |
| A2-03 | 切换状态筛选 | 调 A2 带 `status`，page=1 |
| A2-04 | 点击「重置」 | 清空 q + status，调 A2 |
| A2-05 | 点击列表行（标题列） | `router.push('/admin/china/articles/' + id)` 进入编辑页 |
| A2-06 | 点击「+ 新建文章」 | 打开 D-1 新建文章 Modal |
| A2-07 | 点击「编辑」 | 同 A2-05 |
| A2-08 | 点击「发布」（草稿） | 弹 D-5 二次确认「确定发布该文章？发布后用户即可看到。」→ 确认调 A6 → 成功 toast 并刷新当前页；失败若 422 显示具体原因（缺句子 / 翻译不全） |
| A2-09 | 点击「下架」（已发布） | 弹 D-5「确定下架？所有用户对该文章的阅读进度将被清空（不可恢复）」→ 确认调 A7 → 成功 toast 并刷新 |
| A2-10 | 点击「删除」 | 弹 D-5「确定删除？删除后 30 天内不可恢复，到期物理清除。」→ 确认调 A8 → 204 → toast「已删除」并刷新 |
| A2-11 | 接口 401/403 | 重定向 `/admin/login` |
| A2-12 | 接口 404（类目不存在） | 全屏 ErrorState「类目不存在」+ 返回 |

## 条件显示规则（PM §四）

| 条件 | 显示「发布」 | 显示「下架」 |
|------|------------|------------|
| `status='draft'` | ✅ | ❌ |
| `status='published'` | ❌ | ✅ |

「编辑」「删除」始终显示。
