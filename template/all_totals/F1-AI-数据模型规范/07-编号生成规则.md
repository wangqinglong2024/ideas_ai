# 七 · 编号生成规则（模板）

> 凡是业务上需要展示给用户/管理员的"编号、编码、流水号"都集中在本文件。

---

## 编号：`[字段名]`

```
格式：[例：ORD-{YYYYMMDD}-{NNN}] 或 [12 位 A-Z0-9]
序号重置：[每天 / 每月 / 全局递增 / 不递增]
并发安全：[使用 advisory lock / 独立序列 / 重试-碰撞-换号]
```

实现位置：`fn_gen_<对象>_code()`，建议 `SECURITY DEFINER` 并对调用方授权。

校验：在 `[CHECK 约束 / Zod schema]` 中以正则 `^...$` 强约束，防止任何路径绕过。
