## 2. 多语言 locale 取值（如功能涉及 i18n）

```ts
export const [FUNCTION]_LOCALES = ['...'] as const;
export type [Function]Locale = (typeof [FUNCTION]_LOCALES)[number];
```

- 用于 `*_i18n` 的键校验
- 缺省回退顺序：`<用户 locale>` → `en` → `zh`
