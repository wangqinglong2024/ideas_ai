# 四 · 错误码登记（模板）

> 本功能新增的所有 `<FUNCTION>_*` 错误码集中登记；与 F1 §5 校验规则交叉对照。
> 命名规范：`<FUNCTION>_<对象>_<字段或动作>_<规则>`，全大写下划线。

---

## 通用错误（全功能共用，不重复登记）

| code | HTTP | 含义 |
|------|------|------|
| `40001` | 400 | 参数校验失败 |
| `40002` | 400 | 排序字段不允许 |
| `40101` | 401 | 未登录 |
| `40301` | 403 | 无权限 |
| `40400` | 404 | 资源不存在 |
| `40900` | 409 | 状态冲突 |
| `42900` | 429 | 限流 |
| `50000` | 500 | 服务内部错误 |
