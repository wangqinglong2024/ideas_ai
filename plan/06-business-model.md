# 商业模式设计

> 知语 Zhiyu 变现体系与定价策略
> 版本：v8.0（终版） | 日期：2026-04-16

---

## 一、收入结构总览

知语 Zhiyu 采用 **"登录即玩游戏 + 分层内容 + 会员订阅"** 三层漏斗。游戏是留存与社交引擎；课程是核心付费阵地；发现中国是流量入口。推广激励不使用现金返佣——采用**平台虚拟币"知语币"**，合规风险极低，长期绑定用户。

```
收入结构
  ├── 会员订阅（核心收入，占比 75-85%）
  ├── 游戏皮肤 / 特效 / 赛季通行证（占比 10-20%）
  └── 知语币虚拟币体系（驱动增长，不形成现金支出）
```

### 核心策略

| 策略 | 说明 |
|------|------|
| **游戏登录即玩** | 12 款游戏对所有已登录用户完全开放，无课程进度门槛 |
| **内容两层分发** | 未登录：3 个发现类目；登录：12 个发现类目全部内容 + L1-L3 课程 + 全部 12 游戏；会员：+ L4-L12 课程 |
| **皮肤变现** | 游戏皮肤和特效是游戏侧唯一付费项，纯装饰不影响平衡 |
| **虚拟币代替现金返佣** | 推广奖励以知语币发放，闭环消费，规避越南跨境支付合规风险 |

## 二、三级访问权益对比

### 2.1 权益详情（基于用户的"登录 / 会员"二维属性）

| 功能模块 | 未登录访客 | 登录用户（免费） | 会员（$2/月） |
|---------|----------|----------------|------|
| **发现中国** | 前 3 个类目 | ✅ 全部 12 个类目 + 全部内容 | ✅ 全部 12 个类目 + 全部内容 |
| **课程模块** | ❌ 跳转登录 | ✅ L1-L3 全部课程 | ✅ L1-L12 全部课程 |
| **游戏 12 款** | ❌ 跳转登录 | ✅ **全部 12 款** | ✅ **全部 12 款** |

### 2.2 设计逻辑

- **未登录 → 登录**：靠"9 个隐藏类目 + 全部游戏 + L1-L3 课程"的解锁吸引力驱动注册。
- **登录 → 会员**：L4+ 课程付费墙是唯一转化触点。
- **会员续费**：连续学习进度、升级证书、持续解锁新课程。

## 三、会员订阅体系（USD 全球统一）

### 3.1 会员方案

| 方案 | 定价（USD）|
|------|----------|
| **月度会员** | **$2 / 月** |

> 定价策略：$2/月是国际 SaaS 教育类产品的最低档位之一，远低于 Duolingo Super（$7-14/月）、HelloChinese Premium（$9/月）。该定价专为东南亚市场购买力设计，用绝对低价击穿付费门槛。

### 3.2 会员权益

会员的唯一核心权益：**解锁 L4-L12 全部课程内容**。

### 3.3 本地化货币展示

- 越南用户展示 **VND**（按实时汇率，$2 ≈ 50,000₫）
- 会员订单走 Paddle 以 USD 扣费；Paddle 自动按用户所在地展示当地货币等值价
- 价格 Anchor：页面始终并列展示"$2 (₫50,000)"降低本地用户心算摩擦

## 五、推广激励体系：知语币（Zhiyu Coins）

### 5.1 为什么不做现金返佣

| 维度 | 现金返佣 | 知语币（本方案）|
|------|---------|---------------|
| 越南用户提现到银行卡 | ❌ 触发 FCT、SBV AML、FATF 合规风险 | ✅ 无需提现，闭环消费 |
| App Store / Google Play 政策 | ❌ 苹果明确禁止"cash referral rewards" | ✅ 属于 in-app loyalty points，合规 |
| 退款/欺诈风险 | ❌ 推荐人已提现 + 被推荐人退款 → 损失 | ✅ 未消费可撤销，已消费的虚拟品作废成本为 0 |
| 账务处理 | ❌ 跨境汇款 + 个税代扣 | ✅ 平台内账本即可 |
| 用户粘性 | ❌ 提现即流失 | ✅ 必须使用平台才能消耗 |
| 对 $2/月 的低价产品 | ❌ 经济上不可行（$0.4 佣金成本高于 PayPal 手续费） | ✅ 零现金成本 |

结论：**对 $2/月 定价的 SaaS 产品 + 越南首发市场，虚拟币是唯一合理的推广激励形式。**

### 5.2 知语币规则

| 规则 | 定义 |
|------|------|
| **命名** | 知语币（Zhiyu Coins） |
| **比价** | 1 币 = $0.01（固定，平台内统一） |
| **用途** | 兑换会员 / 皮肤 / 特效 / 赛季通行证 / AI 加速包 |
| **不可做** | 不可提现、不可转让、不可交易、不可折现 |
| **有效期** | 获得后 **365 天**（防止账本无限累积，也防囤币炒作） |
| **上限** | 单用户账户余额上限 50,000 币（$500 等值），防大额操作 |
| **法律性质** | 平台内 loyalty points，非证券、非电子货币、非支付工具 |

### 5.3 知语币来源（Credit）

| 来源 | 数量 | 触发条件 |
|------|------|---------|
| **推荐付费** | 被推荐人付费金额的 **20%**（折算为知语币） | 被推荐人完成付费且 30 天内未退款 |
| **每日签到** | 5-20 币（随机）| 每日登录签到（会员双倍）|
| **学习任务** | 10 币 | 完成一节课程 |
| **升级证书** | 100 币 | 通过一次升级考试 |
| **游戏连胜** | 20 币 | PK 5 连胜 |
| **新手引导** | 100 币 | 完成首次入学测试 |
| **节日活动** | 不定 | 春节/中秋/双11 等 |

### 5.4 知语币消耗（Debit）

| 用途 | 扣减 |
|------|------|
| 月度会员兑换 | 200 币（= $2） |
| 基础皮肤 | 99 币 |
| 稀有皮肤 | 199-299 币 |
| 赛季通行证 | 299 币 |

### 5.5 推荐流程（Referral Flow）

```
用户 A 分享唯一推荐链接 → 用户 B 点击并注册（系统自动绑定，无需邀请码）
     ↓
用户 B 首次付费 $2 → 系统记录 referrals(A, B, 该订单)
     ↓
30 天冷却期通过（B 未退款）→ 向 A 账户发放 40 币（= $0.4）
     ↓
用户 B 后续续费 / 购买皮肤 / 通行证 → A 继续获得 20% 知语币（同样 30 天冷却）
     ↓
A 的知语币仅可在平台消费
```

### 5.6 推荐里程碑奖励（纯平台内激励）

| 累计成功推荐人数 | 额外奖励 |
|-----------------|---------|
| 5 人 | 1 个月会员（直接发放） |
| 10 人 | 稀有皮肤礼包 + 500 币 |
| 20 人 | 3 个月会员 |
| 50 人 | 6 个月会员 + 专属“传颂者”头像框 |
| 100 人 | 12 个月会员 + 永久称号 |

> 里程碑奖励以会员/皮肤等虚拟品发放，不涉及现金。

### 5.7 防作弊

- 同 IP / 同设备指纹注册 → 不计推荐
- 被推荐人付费后 30 天内退款 → 反向扣除推荐人相应知语币（若余额不足则扣至 0 并记负债账本）
- 推荐关系一人一链，不可修改
- 知语币流水表完整审计

## 七、支付实现

### 7.1 Paddle MoR（Merchant of Record）

| 维度 | 说明 |
|------|------|
| 角色 | **Merchant of Record**（Paddle 是法律意义上的卖方，知语是供应商） |
| 优势 | 支持中国大陆公司签约、全球 VAT/GST 自动处理、欺诈保护、退款处理全包 |
| 费率 | ~5% + $0.50/笔（含支付处理 + 税务合规 + 欺诈保护） |
| 支持方式 | 信用卡（Visa/Mastercard/JCB）、PayPal、Apple/Google Pay、部分区域本地支付 |
| 订阅管理 | Paddle Billing 内置订阅管理（续费/取消/退款/失败催收） |
| 税务 | **全自动** — Paddle 代缴全球 VAT/GST/Sales Tax，公司无需在各国注册税号 |

> **$2/月定价的 Paddle 经济测算**：单笔 $2 订阅，Paddle 扣费约 $0.60（5% + $0.50），净收入约 $1.40/月。

### 7.2 支付流程

```
用户选择套餐 → 前端调用 Paddle Checkout → Paddle 托管支付页面
                                              ↓
                                    Paddle Webhook 通知后端
                                              ↓
                                    后端更新用户会员状态 + 写 coin_transactions（推荐人返佣）
                                              ↓
                                    前端刷新权限
```

### 7.3 iOS 特殊处理

- iOS App 内数字内容购买**必须走 Apple IAP**（苹果 30% 抽成）
- **策略**：引导用户通过 **Web 端** 完成订阅（Paddle），绕开苹果税
- App 内不展示购买按钮，只展示"已有会员？登录恢复"
- Apple 明确**禁止 App 内展示 Web 购买链接**——采用"在设置中管理订阅"等中性措辞引导
- 若必须上架 iOS IAP：Web 价 $2，iOS IAP 价 $2.99（覆盖苹果税）

### 7.4 中国大陆公司签约说明

- **Paddle 支持中国大陆公司直接签约**（需营业执照 + 法人身份认证）
- Paddle 作为 MoR，向全球买家收款后扣除费用将净收入转给公司
- 公司**无需自行处理全球税务**
- 结算方式：银行电汇（支持 USD 或 CNY）
- 后续如注册新加坡公司，可迁移 Paddle 账户至新主体

### 7.5 越南合规一览（仅会员订阅，不涉及用户提现）

| 风险项 | 现状 | 缓解 |
|-------|------|------|
| 跨境收款 | 越南用户通过 Visa/Mastercard 付给 Paddle | ✅ 无合规问题，Paddle MoR 承接 |
| VAT（越南 8-10%） | Paddle 自动代缴 | ✅ 无需行动 |
| FCT（外国承包商税） | Paddle 承担供应商义务 | ✅ 无需行动 |
| 用户获取现金 | **本产品不向用户发放现金** | ✅ 用知语币替代 |
| App Store 规则 | 数字商品需 IAP | ⚠️ 引导 Web 订阅 |

## 十、总结

1. **会员订阅**：$2/月，USD 统一，Paddle MoR 收款
2. **权益分层**：未登录（3 个发现类目）→ 登录（12 发现全部内容 + L1-L3 课程 + 12 游戏）→ 会员（+ L4-L12 课程）
3. **推广激励**：知语币（1 币 = $0.01）替代现金返佣，闭环消费，合规零风险
4. **游戏商业化**：12 款游戏登录即玩，收入 100% 来自皮肤 / 特效 / 通行证
5. **合规定位**：越南市场不涉及任何跨境现金支付给用户，Paddle 承担所有税务
