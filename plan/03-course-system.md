# PlayLingo — 系统课程设计方案

> 从零到一的严肃中文教学体系
> 版本：v3.0 | 日期：2026-04-15

---

## 一、模块定位

**系统课程** 是 PlayLingo 三大模块中最核心的教学模块，定位为**严肃的结构化中文教学系统**。区别于游戏模块的趣味练习和文化模块的兴趣激发，课程模块承担"真正教会用户中文"的核心使命。

### 设计原则

1. **HSK 对标**：课程内容严格对标 HSK（汉语水平考试）1-3 级大纲，应试与实用并重
2. **循序渐进**：从拼音零基础到中级会话，每一步都有明确的学习目标和评估标准
3. **多感官输入**：图片情境 + 文字讲解 + 音频示范 + 互动练习，全方位刺激记忆
4. **螺旋上升**：每个新知识点都建立在已学内容之上，通过不断复现强化记忆
5. **文化融入**：每课设"文化角"，将语言学习与中国文化认知有机结合

## 二、课程体系总览

### 四级课程结构

```
Level 0: 语音入门 (Phonetics Foundation)
  └─ 20 课 | 目标：掌握全部拼音，能准确拼读任意汉字
  
Level 1: 基础入门 (Elementary / HSK1)
  └─ 30 课 | 150 词 | 目标：基本问候、自我介绍、简单交流

Level 2: 初级会话 (Pre-Intermediate / HSK2)
  └─ 30 课 | 300 词 | 目标：日常对话、简单叙事、基础读写

Level 3: 中级应用 (Intermediate / HSK3)
  └─ 30 课 | 600 词 | 目标：职场社交、观点表达、中等阅读
```

**总计：110 课 | 覆盖 HSK1-3 全部 1050+ 词汇**

## 三、Level 0：语音入门（20课）

### 课程目标
- 掌握 23 个声母、24 个韵母、4 个声调 + 轻声
- 能准确拼读任意拼音组合
- 了解拼音拼写规则和声调标注规则

### 单元划分

#### Unit 1：声母（6课）

| 课号 | 主题 | 教学内容 | 练习重点 |
|------|------|---------|---------|
| L0-01 | 唇音声母 | b, p, m, f | 听辨 b/p（送气/不送气对比）|
| L0-02 | 舌尖声母 | d, t, n, l | 听辨 d/t, n/l 区别 |
| L0-03 | 舌根声母 | g, k, h | 听辨 g/k, 发音位置感知 |
| L0-04 | 舌面声母 | j, q, x | 越南语对比发音法 |
| L0-05 | 翘舌声母 | zh, ch, sh, r | 翘舌 vs 平舌对比 |
| L0-06 | 平舌+特殊 | z, c, s, y, w | 综合辨音练习 |

**教学方法**：
- 发音动画：口腔截面动画展示舌位和气流
- 越南语类比：用越南语相似发音引入（如 b ≈ 越南语 b）
- 听辨训练：播放两个相似音，选择正确的
- 跟读练习：播放标准音频，录音对比（v1.1 加入 AI 评分）

#### Unit 2：韵母（6课）

| 课号 | 主题 | 教学内容 | 练习重点 |
|------|------|---------|---------|
| L0-07 | 单韵母 | a, o, e, i, u, ü | 口型图示 + 长音练习 |
| L0-08 | 复韵母(一) | ai, ei, ao, ou | 滑音过渡练习 |
| L0-09 | 复韵母(二) | ia, ie, iu, ua, uo, üe | 双元音组合 |
| L0-10 | 前鼻韵母 | an, en, in, un, ün | -n 收尾训练 |
| L0-11 | 后鼻韵母 | ang, eng, ing, ong | -ng 收尾训练 |
| L0-12 | 特殊韵母 | er, 整体认读音节 (zhi, chi, shi, ri, zi, ci, si, yi, wu, yu, ye, yue, yuan, yin, yun, ying) | 记忆 + 辨识 |

#### Unit 3：声调（4课）

| 课号 | 主题 | 教学内容 | 练习重点 |
|------|------|---------|---------|
| L0-13 | 四声入门 | 一声(阴平)、二声(阳平)、三声(上声)、四声(去声) | 音高走向图 + 手势法 |
| L0-14 | 声调辨析 | 易混声调对比（二声vs三声，一声vs四声） | 听辨选择 + 跟读 |
| L0-15 | 声调变化 | 三声变调规则、"一""不"变调 | 实际词语中的声调应用 |
| L0-16 | 轻声 | 轻声规则 + 常见轻声词 | 自然语流中的轻声 |

**声调教学特色**：
- 音高曲线可视化：每个声调用动画展示音高变化
- 越南语声调对比：越南语有6声调，找出与中文相似的映射
- 手势辅助：每个声调配合手势动作帮助记忆
- 最小对立组：mā/má/mǎ/mà 四声对比训练

#### Unit 4：拼音综合（4课）

| 课号 | 主题 | 教学内容 | 练习重点 |
|------|------|---------|---------|
| L0-17 | 拼读规则 | 声母+韵母拼合规则、ü 的省略规则 | 快速拼读练习 |
| L0-18 | 拼写规则 | 大小写、隔音符号、标调规则 | 听写 + 改错 |
| L0-19 | 实用拼读 | 30个高频词拼读（你好、谢谢、中国...） | 看拼音认词 |
| L0-20 | 语音综合测试 | 全部声韵调综合 | 期末测试（≥80%通过） |

## 七、每课标准结构（详细规范）

### 教学流程（总时长 15-25 分钟）

```
┌─────────────────────────────────────────┐
│ Step 1: 导入 (1-2分钟)                   │
│  情境图片 + 背景描述（越南语）            │
│  引出本课主题和学习目标                    │
│  🔊 背景音频（轻松的中国风BGM）           │
├─────────────────────────────────────────┤
│ Step 2: 生词学习 (3-5分钟)                │
│  8-12 个新词，逐个展示：                  │
│  汉字 → 拼音 → 越南语释义 → 例句          │
│  每词配 🔊 标准发音 + 🔊 例句朗读          │
│  [加入词汇本] 按钮                        │
├─────────────────────────────────────────┤
│ Step 3: 课文 (3-5分钟)                    │
│  完整对话或短文展示                        │
│  三行式呈现：中文(粗) / 拼音(灰) / 越南语  │
│  🔊 全文朗读 + 逐句可点击重播              │
│  高亮词汇可点击查看详情                    │
├─────────────────────────────────────────┤
│ Step 4: 语法讲解 (2-3分钟)                │
│  1-2 个语法点                             │
│  结构图解 + 3-5 个例句                    │
│  越南语语法对比说明                        │
│  简单的语法判断练习（2-3题）               │
├─────────────────────────────────────────┤
│ Step 5: 互动练习 (5-8分钟)                │
│  10-15 道练习题（见题型详解）              │
│  即时反馈：对/错 + 解析                    │
│  错误自动记录到"错题本"                    │
├─────────────────────────────────────────┤
│ Step 6: 文化角 (1-2分钟)                  │
│  与本课主题相关的文化小知识                │
│  → 链接到「发现中国」相关文章             │
├─────────────────────────────────────────┤
│ Step 7: 课后总结                          │
│  本课所学要点回顾                          │
│  新词数 | 正确率 | 获得 XP/金币            │
│  下一课预告                               │
│  [分享学习成果] 按钮                      │
└─────────────────────────────────────────┘
```

### 练习题型详解

| 题型 | 描述 | 适用级别 | 占比 |
|------|------|---------|------|
| 听力选择 | 播放音频，从3-4选项中选择正确答案 | L0-L3 | 20% |
| 拼音填写 | 看汉字写拼音，或听音频写拼音 | L0-L1 | 15% |
| 汉字选择 | 看拼音/听音频选正确汉字 | L0-L2 | 15% |
| 词义匹配 | 中文词与越南语释义连线配对 | L1-L3 | 10% |

## 八、音频规范

### 音频类型

| 类型 | 格式 | 生成方式 | 数量估算 |
|------|------|---------|---------|
| 生词发音 | 单词+例句 | Azure TTS 中文女声 | ~1200 条（L0-L3） |
| 课文朗读 | 全文+逐句 | Azure TTS 中文女声 | ~220 段 |
| 听力题音频 | 对话/句子 | Azure TTS 中文男女声 | ~800 条 |
| 越南语讲解 | 语法说明 | Azure TTS 越南语 | ~200 段 |
| 背景音乐 | 轻松中国风 | Suno AI | 4-6 首 |

### TTS 参数标准
- **语速**：L0=0.8x（慢速），L1=0.9x，L2=1.0x（正常），L3=1.0-1.1x
- **声音**：女声为主（zh-CN-XiaoxiaoNeural），男声辅助（zh-CN-YunxiNeural）
- **格式**：MP3 128kbps + OGG 备选
- **间隔**：生词发音后停顿 0.5s，例句后停顿 1s

## 九、课程与其他模块联动

### 课程 → 文化模块
- 每课「文化角」链接到「发现中国」对应文章

- 课程中学到的词汇和语法，在游戏关卡中作为测试内容出现

### 课程 → 词汇系统
- 课程中所有生词自动加入词汇本（标记来源课号）
- SRS 复习算法整合课程学习进度
- 课程练习中的错词重点复习
