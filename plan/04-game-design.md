# PlayLingo — 游戏化设计方案

> 把教学变成游戏，把故事变成过场动画，把练习变成战役
> 版本：v3.0 | 日期：2026-04-15
> 注：游戏模块是三大模块之一，与「发现中国」和「系统课程」并列

---

## 一、模块定位

**趣味游戏** 是 PlayLingo 三大模块之一，定位为**游戏化练习场**。它不是主要的教学载体（那是课程模块的职责），而是通过 2D 策略游戏将课程中学到的知识变成实战应用，让枯燥的复习和练习变得令人上瘾。

### 核心设计哲学

1. **练习而非教学**：游戏假设用户已在课程中学过基础知识，重点在巩固和应用
2. **即时反馈**：每次正确操作都有视觉+音效奖励，多巴胺驱动的学习循环
3. **渐进挑战**：难度曲线平滑上升，始终保持在"有挑战但不挫败"的甜蜜区间
4. **社交竞争**：排行榜、PvP 对战（v1.1）满足竞争和社交需求

## 三、四种关卡类型

### Type 1：知识教学关（拼音 + 汉字基础 + 语法）

**流程**：概念讲解动画 → 8-12 道练习题 → 结算

**练习题型**：
- 听音选字：播放音频，从4选项选正确汉字/拼音
- 配对连线：左列中文，右列越南语，拖拽连线
- 拖拽排序：打乱的词拖拽排成正确句子
- 填空选择：根据语境选择正确词语
- 判断对错：句子语法是否正确

### Type 2：词汇战役关（词汇平原专属）

**流程**：60秒限时图词匹配 → 3-5 道排序题 → NPC Boss 对话

**详细机制**：
- **图词匹配**：画面显示4-5张图片+对应中文标签，拖拽配对，60秒限时
- **排序题**：打乱的词排成正确句子
- **Boss 对话**：NPC 说中文，玩家从3个选项中选择正确回复

### Type 3：汉字书写关（汉字谷地专属）

**流程**：笔画动画演示 → 红色描边临摹 → 自由书写

**详细机制**：
- **演示**：播放汉字笔画顺序动画，标注每一笔名称
- **临摹**：红色虚线引导，手指/鼠标沿线描写，实时反馈
- **自由书写**：无引导线书写，AI 识别笔画顺序和结构（容差 ±20%）
- **偏旁认知**：拆解汉字结构（如「好」= 「女」+「子」）

### Type 4：听力挑战关

**流程**：播放音频 → 从3-4选项中选正确答案 → 连续10-15题

**难度递进**：
- 初级：听声选声母/韵母
- 中级：听词选汉字
- 高级：听句子选意思

## 四、三款迷你游戏

### 1. 声调狙击手（Tone Sniper）

**灵感**：Beat Saber × 中文声调

**玩法**：
- 声波从四面八方飞向屏幕中心
- 每个声波标注一个拼音（如 "mā"）
- 玩家按声调方向滑动（一声→, 二声↗, 三声↘↗, 四声↘）
- 方向正确±30°容差 = 命中，粒子爆炸特效
- 连击倍率：5连=金边闪光, 10连=2倍积分, 20连=全屏金色特效

**游戏参数**：
- 时长：60-90秒
- 生命：3条
- 速度递增：0-15s=1.5s/个 → 60s+=0.4s/个
- 解锁条件：拼音群岛第5关

### 2. 偏旁爆破（Radical Blitz）

**灵感**：水果忍者 × 汉字组装

**玩法**：
- 目标汉字的幽影轮廓显示在屏幕上方，标注2-4个偏旁位置
- 偏旁部件+干扰项+炸弹从天空落下
- 划动（H5）或点击（PC）切开正确偏旁 → 飞入正确位置
- 砍错干扰项 = 无事发生，砍到炸弹 = 屏幕红闪 + 扣1命
- 3个连续正确 → 1.5x积分, 5连 → 3x积分

**游戏参数**：
- 模式：闯关（5简单+10中等+15困难）/ 无尽 / 每日挑战
- 生命：3条
- 解锁条件：汉字谷地第3关

### 3. 拼音漂移（Pinyin Drift）

**灵感**：赛车游戏 × 听力选择

**玩法**：
- 鸟瞰视角赛车自动向前行驶
- 前方出现3-4条岔路，每条标注一个答案选项
- 播放音频提示（如"苹果"的发音）
- 玩家左右滑动选择正确车道
- 撞错 = 2秒减速 + 屏幕震动 + 扣1次机会（共3次）
- 道具：🛡️护盾（免疫1次错误）, ⏱️减速（3秒思考时间）, 💰双倍分

**游戏参数**：
- 题型轮换：听音选字 → 看字选义 → 听句补空
- 速度递增
- 解锁条件：词汇平原第3关

## 八、进度与解锁

### 关卡解锁规则
- 顺序解锁：通关前一关（≥1星）解锁下一关
- 区域 Boss 需全部普通关≥1星
- 新区域需通关前一区域 Boss

### 重玩机制
- 任何已通关关卡可无限重玩
- 重玩可刷新星级（取最高）
- 重玩获得奖励 = 首次的 30%

## 九、与其他模块的联动

### 游戏 ← 课程
- 游戏中出现的词汇和语法来源于课程教学内容
- 游戏难度与课程进度松耦合：建议先学课程再玩对应游戏关卡
- 游戏失败时提示："去课程 Level 0 第3课复习一下？"

### 游戏 ← 文化
- 过场动画中融入中国文化元素（如 Boss「汉字封印师」背后的文化故事）
- 游戏完成后推荐相关文化文章

### 游戏 → 词汇
- 游戏中遇到的生词自动加入词汇本（标记来源：游戏第X关）
- 过场动画中点击的高亮词汇也加入词汇本

## 十、PvP 对战（v1.1）

### 基础机制
- 实时 1v1 对战，WebSocket 通信
- 匹配规则：同段位±1级，15秒超时匹配AI对手
- 题目服务器端发放，客户端不持有答案（防作弊）
- 答对得分+计时，答快额外加分

### 段位系统（军衔）
列兵 → 下士 → 中士 → 上士 → 少尉 → 中尉 → 上尉 → 少校 → 中校 → 上校 → 少将 → 中将 → 元帅

### 赛季
- 每月重置段位（保底不低于上赛季-3级）
- 赛季奖励：钻石+专属头像框
- 赛季排行榜
