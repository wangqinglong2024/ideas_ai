## 一、产品定位

**PlayLingo** 是面向东南亚市场的综合中文学习 Web App，以三大核心模块构建完整学习生态：

| 模块 | 定位 | 核心价值 |
|------|------|---------|
| 🏛️ 发现中国 | 内容驱动的文化探索 | 激发兴趣，建立情感连接 |
| 📚 系统课程 | 严肃结构化教学 | 从零到一系统掌握中文 |
| 🎮 趣味游戏 | 游戏化练习场 | 巩固应用，寓教于乐 |

### 一句话定位

> **PlayLingo = 了解中国的窗口 + 学中文的课堂 + 练中文的游乐场**

### 三模块飞轮

```
文化内容激发兴趣 → 用户想学中文
       ↓
课程提供系统知识 → 用户掌握基础
       ↓
游戏巩固应用能力 → 用户获得成就感
       ↓
成就感驱动更深入的文化探索 → 飞轮加速
```

三个模块互相导流：文化阅读中遇到不懂的词汇 → 引导去课程学习；课程中学到的语法 → 在游戏中实战运用；游戏中接触的文化背景 → 回到文化模块深入了解。

## 二、核心理念

### 1. 文化先行，兴趣驱动

传统中文学习 App 上来就教拼音、背单词，用户缺乏动机容易放弃。PlayLingo 反其道而行：**先让用户爱上中国文化**——成语背后的精彩故事、唐诗宋词的意境之美、中国地理的壮丽风光——当用户对中国产生情感连接后，学习中文就从「任务」变成了「渴望」。

### 2. AI 生成内容，零边际成本

AI 一次性生成高质量的文化文章、课程教案、游戏脚本、配音音频，这些内容资产零边际成本无限分发。我们卖的不是 AI 能力，而是精心编排的学习体验。

### 3. 透明 AI 定价

AI 增值功能（对话练习、作文批改）按实际 API 成本 +5% 透传，不做暴利加价。用户为内容体验付费，不为 Token 配额付费。

## 三、三大模块概览

### 模块一：发现中国（Discover China）

目标：**让用户全方位了解中国，爱上中国文化**

| 内容板块 | 内容方向 | 内容形式 |
|---------|---------|---------|
| 中华历史 | 远古神话→秦汉→唐宋→明清→近现代 | 图文专题 + 时间线 + 音频讲解 |
| 地理风光 | 名山大川、世界遗产、古城古镇、省份巡礼 | 图集 + 地图标注 + 音频导览 |
| 文学经典 | 四大名著、唐诗宋词、先秦经典、神话传说 | 双语阅读 + 音频朗诵 + 名句赏析 |
| 成语典故 | 历史成语、寓言成语、日常成语 | 故事图文 + 动画 + 互动填空 |
| 传统文化 | 节日习俗、饮食文化、传统艺术、武术功夫 | 专题图文 + 视频片段 |
| 哲学思想 | 儒道佛、兵法谋略、处世哲学 | 深度文章 + 经典名句 |
| 当代中国 | 科技创新、城市生活、流行文化、社交热词 | 时事图文 + 流行语解读 |
| 中越交流 | 历史渊源、汉字与越南语、文化共通 | 对比专题 + 趣味发现 |

**核心特色**：
- 全部内容中越双语呈现，关键汉字标注拼音
- 每篇提取 8-15 个核心词汇，自动关联词汇本和课程
- AI 生成高质量音频讲解
- 三级难度（入门/进阶/高级），适配不同水平
- 阅读后互动小测验强化记忆
- 内容可社交分享，带来自然流量

### 模块二：系统课程（Courses）

目标：**严肃教学，从零基础到中级水平的完整路径**

| 级别 | 对标 | 课时 | 词汇量 | 核心能力目标 |
|------|------|------|--------|------------|
| Level 0: 语音入门 | 拼音基础 | 20课 | — | 声母韵母声调，准确拼读任意汉字 |
| Level 1: 基础入门 | HSK1 | 30课 | 150词 | 基本问候、自我介绍、数字时间 |
| Level 2: 初级会话 | HSK2 | 30课 | 300词 | 日常对话、简单叙事、基础阅读 |
| Level 3: 中级应用 | HSK3 | 30课 | 600词 | 职场社交、观点表达、中等阅读 |

**每课标准结构**：
1. **导入**（情境图片 + 背景音频，30秒）
2. **生词**（8-12词，每词配音频/拼音/释义/例句）
3. **课文**（对话或短文，逐句配音频，点击可看注释）
4. **语法**（1-2个语法点，配图解和例句）
5. **练习**（10-15题：听力选择、拼音填写、语序排列、情境选择）
6. **文化角**（与本课主题相关的中国文化小知识，链接到「发现中国」）
7. **单元测验**（每5课一次综合测试）

### 模块三：趣味游戏（Games）

目标：**用游戏机制让语言练习上瘾**

| 游戏内容 | 说明 |
|---------|------|
| 冒险闯关 | 4大区域60关（拼音群岛→汉字谷地→词汇平原→语法要塞） |
| 迷你游戏 | 声调狙击手、偏旁爆破、拼音漂移（3款街机风格） |
| Boss 战 | 4场区域终极挑战 |
| 剧情动画 | 8段过场动画，故事驱动学习 |
| PvP 对战 | 实时排位对战（v1.1） |

## 五、目标用户画像

### Segment A：越南工厂青年（18-30岁）— 核心用户
- **动机**：会中文加薪 50%+
- **路径**：课程（快速掌握职场中文）→ 游戏（碎片练习）→ 文化（理解中国老板）
- **付费意愿**：高（投资回报明确）

### Segment B：大学生（18-24岁）— 增长用户
- **动机**：HSK 考试 + 求职竞争力
- **路径**：课程（系统备考）→ 文化（提升素养）→ 游戏（减压学习）
- **付费意愿**：中（价格敏感但需求刚性）

### Segment C：华人华侨后代（15-35岁）— 情感用户
- **动机**：文化认同 + 与长辈沟通
- **路径**：文化（寻根）→ 课程（从零开始）→ 游戏（趣味入门）
- **付费意愿**：高（情感驱动）

### Segment D：文化兴趣者（15-40岁）— 流量用户
- **动机**：喜欢中国影视/动漫/美食/功夫
- **路径**：文化（探索）→ 游戏（轻度参与）→ 课程（深入学习）
- **付费意愿**：低→中（兴趣转化需要时间）

### 核心护城河
1. **文化壁垒**：8大板块的中国文化内容库是独特资产，竞品无法短期复制
2. **三合一飞轮**：文化→课程→游戏的正向循环，单一模式 App 无法复制
3. **价格碾压**：6.5倍价差 + AI 零边际成本
4. **本地化纵深**：越南语深度本地化 + 本地支付 + 在地社群
