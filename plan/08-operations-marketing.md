# PlayLingo — 运营与营销方案

> 文化引流 + 课程转化 + 游戏留存 + 佣金裂变
> 版本：v3.0 | 日期：2026-04-15

---

## 一、运营总纲

### 1.1 核心理念

```
"用中国文化吸引用户，用系统课程留住用户，用趣味游戏黏住用户"

三模块运营关键词：
  文化模块：免费引流、SEO 长尾、社交传播
  课程模块：专业转化、留存核心、付费驱动
  游戏模块：上瘾机制、社交裂变、PvP 拉新

整体关键词：便宜 + 专业 + 好玩 + 赚钱（佣金）
价格锚点：$2.99/月 vs Duolingo $12.99/月 vs 线下课 $50-100/月
```

### 1.2 运营节奏

| 阶段 | 时间 | 重点 |
|------|------|------|
| MVP 内测 | 第 1 月 | 200 种子用户，验证三模块学习链路 |
| 冷启动 | 第 2-3 月 | TikTok 矩阵 + 文化内容 SEO + KOL 合作 + 佣金推广员招募 |
| 增长期 | 第 4-6 月 | 付费投放 + PvP 社交裂变 + B2B 企培 + 文化内容持续产出 |
| 爆发期 | 第 7-12 月 | 品牌化 + 印尼扩张 + 大型活动 + 课程体系完善 |

  · "Vạn Lý Trường Thành" (万里长城) — 月搜 3K+
  · "truyện Tây Du Ký" (西游记) — 月搜 4K+

  playlingo.app/discover/[category]/[article-slug]

## 五、付费投放

### 5.1 启动时机

```
触发条件（满足任一）：
  · 自然增长达到 500 DAU
  · LTV/CAC > 3
  · 月收入 > $500
  · MVP 核心数据验证通过（次日留存 >40%、7 日留存 >20%）
```

### 5.2 投放策略

```
Facebook/Instagram Ads：
  · 按 CPI 优化
  · 目标 CPI < $0.30（越南）
  · A/B 测试素材：文化内容 vs 课程专业性 vs 游戏画面 vs 价格导向
  · 预算：起步 $10/天，逐步放大

TikTok Ads：
  · Spark Ads（加热自然内容）
  · 目标 CPI < $0.20
  · 文化类内容加热效果最佳
  · 预算：起步 $10/天

Google Ads：
  · 搜索广告："学中文 app" "中国文化" 越南语关键词
  · 目标 CPI < $0.50
  · 低优先级，文化 SEO 优先
```

### 5.3 投放漏斗

```
广告曝光 → 落地页 → 注册 → 完成引导 → 7 日留存 → 付费
  100%  →  30%  →  50%  →   70%    →   25%   →  5%

目标 CPI: $0.20-0.30
目标 CPA (付费): $4-6
计划 LTV: $15+ (5 个月 ARPU)
```

## 六、佣金推广运营

### 6.1 推广员管理

```
招募渠道：
  · 现有付费用户中招募活跃用户
  · TikTok/Facebook 发布"赚钱学中文"招募帖
  · 大学校园代理（学生推广员）
  · KOC 自动升级为推广员
  · 汉语教师推广员（推荐学生使用）

推广员类型：
  · 普通用户推广员（自然分享）
  · 学生推广员（校园渠道）
  · 教师推广员（教学场景推荐 — 课程模块为主）
  · KOL/KOC 推广员（内容渠道 — 文化模块引流）
```

### 6.2 推广物料

```
App 内一键生成：
  · 个人专属推广链接（点击即绑定，无需输入邀请码）
  · 带推广链接二维码的分享海报（多模板可选）
  · 文化文章分享卡片（带推广链接）
  · 课程成绩分享卡片（带推广链接）
  · 游戏成就分享卡片（带推广链接）
  · 佣金收益报告（实时更新）
```

### 6.3 防作弊运营

```
监控指标：
  · 单个推广员每日新增上限：50 人
  · 推荐用户 7 日留存必须 > 10%（否则不计佣金）
  · 同 IP/设备注册的推荐不计佣金
  · 推荐人和被推荐人不能同设备

处罚措施：
  · 疑似刷量：暂停佣金，人工审核
  · 确认违规：永久取消推广资格
  · 严重违规：冻结账户余额
```
