# 发现中国（Discover China）— 模块设计文档

> 知语 Zhiyu 免费文化探索模块
> 版本：v7.0 | 日期：2026-04-16

---

## 一、模块定位

**发现中国** 是 知语 Zhiyu 的内容流量入口，以 **12 大文化类目** 覆盖中国文化的方方面面。所有用户均可免费浏览各类目的基础内容，会员解锁深度内容。

核心目的：
1. **获客引流** — 多样化免费内容吸引不同兴趣的用户
2. **兴趣激发** — 从美食、功夫到流行文化，总有一个切入点让用户对中国产生兴趣
3. **付费导流** — 基础内容免费 → 深度内容引导会员转化 → 内容关联课程和游戏

> 产品逻辑：先让用户觉得中国有意思 → 再引导他去学中文 → 而不是一上来就推课程。

## 三、内容分级与访问模型

### 3.1 访问规则

| 用户类型 | 可访问内容 |
|---------|----------|
| 非会员（注册即可） | 12 个类目均可浏览，每个类目的**基础内容**免费（每类目约 5 篇） |
| 会员 | 全部 12 个类目的**全部内容**无限访问（含深度解读、系列专题、独家内容） |

### 3.2 内容分层

| 层级 | 说明 | 访问权限 |
|------|------|---------|
| **基础内容** | 入门介绍、趣味科普、短图文卡片 | 全部用户 |
| **深度内容** | 专题系列、长文解析、视频纪录、互动测验 | 仅会员 |

> 每个类目都是一个潜在的转化触点——用户在任何感兴趣的类目中遇到内容墙，都能被引导成为会员。

## 四、内容格式标准

### 4.1 单篇内容结构

```
标题（多语言：中文 + 拼音 + 用户母语）
封面图 / 视频缩略图
正文（多语言对照，300-800 字）
核心词汇表（中文 + 拼音 + 翻译，3-8 个）
趣味小测（1-3 题）
相关推荐（同类目其他文章 / 关联课程 / 关联游戏）
```

### 4.2 内容形式

| 形式 | 适用场景 |
|------|---------|
| 图文卡片 | 快速浏览、社交分享（主力形式） |
| 短视频（1-3 分钟） | 美食制作、景点展示、功夫演示 |
| 音频解说 | 配合阅读、碎片时间听 |
| 互动测验 | 每篇文末，强化记忆和参与感 |

## 六、内容生产流程

### 6.1 生产工具链

```
选题策划 → AI 初稿（Dify / GPT）→ 人工审核润色 → 多语言翻译 → 配图 / 配视频 → 发布
```

### 6.2 团队分工

| 角色 | 职责 |
|------|------|
| 内容策划 | 选题、热点追踪、排期 |
| AI + 编辑 | AI 生成初稿 → 编辑审核、润色、本地化 |
| 翻译 | 越南语 / 英语翻译校对 |
| 设计 | 封面图、配图、短视频制作 |

### 6.3 更新节奏

| 阶段 | 频率 | 说明 |
|------|------|------|
| MVP 上线前 | 每类目 5+ 篇预备 | 保证上线时内容充足（合计 60+ 篇） |
| 上线后常规 | 每周 3-5 篇（分散到各类目） | 保持新鲜感 |
| 节日 / 热点 | 临时加更 | 蹭热点，提升传播 |

## 八、变现逻辑

```
12 个类目免费基础内容 → 用户大量涌入 → 发现深度内容需要会员
                                            ↓
                                     会员转化（核心变现）
                                            ↓
                              内容引导至课程/游戏 → 更深度的学习参与
```

关键：**发现中国是获客漏斗的顶部**，不直接变现，通过内容价值驱动会员转化和课程导流。
