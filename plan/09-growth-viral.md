## 三、社交裂变机制

### 3.1 文化内容裂变（新增 v3.0）

```
设计逻辑：
  文化内容天然具有社交传播属性
  "你知道越南语里有60%是中文借词吗？" → 转发冲动

具体流程：
  1. 用户在「发现中国」阅读一篇成语故事
  2. 文章底部 "分享到 Facebook/Zalo" 按钮
  3. 生成精美分享卡片：成语 + 插画 + 简短故事 + 推广链接
  4. 朋友点击 → 免费阅读文章 → 注册解锁更多内容
  5. 注册时自动绑定推荐关系

裂变倍数估计：
  · 每篇文章平均被分享 3-5 次（优质内容可达 20+）
  · 文章点击 → 注册转化率 ~15%
  · 文化裂变 K-factor ≈ 0.3-0.5
```

### 3.2 PvP 对战裂变

```
设计逻辑：
  对战 → 赢了想炫耀 → 输了想复仇 → 都想拉好友

具体流程：
  1. 用户 A 发起 PvP 匹配
  2. 对战结束，双方看到结果 + 分享按钮
  3. "邀请好友对战！好友通过链接注册自动绑定推荐关系"
  4. 好友加入 → 对战圈扩大 → 更多人加入

裂变倍数估计：
  · 每个 PvP 玩家平均邀请 2-3 个好友
  · 好友转化率 ~30%
  · PvP 裂变 K-factor ≈ 0.6-0.9
```

## 六、病毒系数优化

### 6.1 K-factor 目标

```
K = 邀请数 × 转化率

三模块分别贡献：
  · 文化分享：K = 0.3（高分享、低转化）
  · 课程推荐：K = 0.1（低分享、高转化）
  · 游戏裂变：K = 0.4（PvP + 成就分享）

总 K-factor 目标：0.8-1.0（含佣金激励）

K > 1.0 时（病毒增长）：
  · 不需要额外投放也能自然增长
  · 触发条件：推广员占 DAU >5% 且人均邀请 >10/月
```

### 6.2 优化手段

```
提高邀请数：
  ✓ 文化文章天然可分享（社交货币）
  ✓ 每次成就都弹出分享入口
  ✓ 佣金实时推送刺激再分享
  ✓ PvP 自带社交邀请属性
  ✓ 课程证书 → 社交分享

提高转化率：
  ✓ 文化内容免费可读（降低注册门槛）
  ✓ 被推荐人首月 7 折
  ✓ 落地页展示推荐人的学习成就
  ✓ 新用户注册后可选择感兴趣的模块（不是教程墙）
```

| 指标 | 定义 | 目标 |
|------|------|------|
| K-factor | 每用户带来的新用户数 | >0.8 |

| 课程完课率 | 完成至少一个单元的比例 | >40% |
| 邀请点击率 | 邀请链接点击/发送 | >40% |
| 注册转化率 | 注册/落地页访问 | >50% |
| 佣金用户占比 | 有推广收入的用户/总用户 | >5% |
| 分享率 | 每日分享次数/DAU | >15% |

### 7.2 A/B 测试计划

```
优先级 1（MVP 后立即测试）：
  · 新用户首选模块推荐（文化优先 vs 课程优先 vs 游戏优先）
  · 分享弹窗时机（文章读完 vs 关卡完成 vs 测验通过）
  · 落地页设计（文化内容 vs 课程专业性 vs 游戏画面 vs 价格）

优先级 2：
  · 首月折扣力度（7折 vs 5折 vs 免费）
  · 文化文章免费量（每板块2篇 vs 3篇）
```
